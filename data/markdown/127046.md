### Description
Dark theme with background picture for Searx metasearch engine. It should work for all Searx public servers. Credits and thanks go to user 255004. Please write a comment if you feel something can be improved, I'd be glad to try any suggestion and to add your own background textures and/or Searx public servers ^^

### Update notes
This style is a mod of the original style 115066 superb work by user 255004.

It will look nice only if used with the <i>Oscar Pointhi</i> Searx theme. If it doesn't look as expected just go to search engine's <i>Preferences</i> > <i>Themes</i>, select <i>oscar</i> and <i>Pointhi</i> in the drop-down menu, then <i>Save</i> at the bottom.

<b>Updated 2016/04/30</b>
When your search terms contain the words "ip" or "user agent", Searx returns your IP address or your browser user-agent in a box on top of search results. This value was difficult to read because it was written in clear gray text on near-white background. This is what the .well div customization fixed, now it looks like the category buttons above.
