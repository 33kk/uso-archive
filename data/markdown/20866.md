### Description
This userstyle is for the theme Strata40. It should only be used if you don't want StrataBuddy to do it for you.

It converts the theme into a tabs on top version very closely resembling the Firefox 4 mockups. It is to be used with v0.5.4 - v0.5.5 of the theme. If the theme updates, check here for an updated userstyle if needed, or get StrataBuddy (highly recommended).
