### Description
MBlue Combine Home-Button XP (Ver. 0.1.0)
--------------------------------------------------------------------
Description:
 1. Integrates with the "Combine Nav-Buttons XP" style to combine the Home-Button with the Back/Forward/Stop/Reload buttons.
 2. Gives the same look as the Back/Forward Buttons (Firefox on Vista) to the Home-Button.
 3. Supports both SMALL and LARGE Icons.
--------------------------------------------------------------------
Notes:
-YOU NEED THE "COMBINE NAV-BUTTONS XP" STYLE TO MAKE THIS WORKS.

-To make it works, place the buttons in this order:

 > (BACK|FORWARD) (STOP) (RELOAD) (HOME) <

-In customizing mode you'll see the button's background GREEN if
 the position is right, or RED if it's wrong.

-When using THEMES/EXTENSIONS/STYLES that changes some margins, you
 may need to change some value manually, if so go to bottom of the
 page to customize.
--------------------------------------------------------------------
CUSTOMIZING:

 1. NAV-BUTTONS MARGINS.
 2. HOME-BUTTON MARGINS.

> GO TO BOTTOM OF THE PAGE TO CUSTOMIZE
--------------------------------------------------------------------
List of compatible Styles:
[1] Combine Nav-Buttons Vista ( http://userstyles.org/styles/8620 )
[2] Combine Nav-Buttons XP ( http://userstyles.org/styles/8731 )
[3] Nav-Buttons Vista on XP ( http://userstyles.org/styles/8761 )
[4] Nav-Buttons XP on Vista ( http://userstyles.org/styles/9018 )
[5] Combine Home-Button Vista
[6] Combine Home-Button XP (THIS)

How to:
[A] You are on Vista and want to combine the Buttons (BLUE):
    > use styles [1]
[B] You are on XP and want to combine the Buttons (GREEN):
    > use styles [2]
[C] You are on XP and want the Vista Buttons (BLUE):
    > use styles [3]
[D] You are on XP and want both the Vista Buttons and combine:
    > use styles [1] [3]
[E] You are on Vista and want the XP Buttons (GREEN):
    > use styles [4]
[F] You are on Vista and want both the XP Buttons and combine:
    > use styles [2] [4]

[!] You can combine the Home-Button too by using style [5] or [6]
