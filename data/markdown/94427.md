### Description
Removes or displaces some of the sticky (position:fixed) Google+ navigation.

1. Hide big black toolbar with a red "Join Google+" button. Assuming you don't want to.
2. Hide the semi-transparent bar with the user name and search info (appears when scrolling down).
3. Move white menu bar (Info/Post/Photo/YouTube) to the top, and don't allow it to be sticky.
4. Move the profile menu button to the top.
