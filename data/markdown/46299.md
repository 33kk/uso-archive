### Description
A cursor indicator.  Hovering over hyperlinks will show a different cursor based on the targets file extension. Re-make of style 1183.

Supported file extensions:
- 7Z
- DOC (for doc and docx)
- EXE
- MSI
- E-mail (mailto:)
- PDF
- RAR
- TXT
- XLS (for xls and xlsx)
- ZIP
