### Description
Lucifer's Blog post always come with image too large to fit in browser's width.

This user style removes everything except the content and comments,
and resize the image make it will not produce a horizontal scroll bar.

Also, this user sytle will not affect the homepage of Lucifer's Blog.
You can use the links on the top to back to Lucifer's Blog.
