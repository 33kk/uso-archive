### Description
<p>Increases the width of Github pull requests from the default of 980px to 1400px. Allows for easier viewing and reviewing without having to scroll horizontally for code bases wider than 80 characters.</p>
<p><strong>Note</strong> that this only affects the /pulls/ and /pull/ pages on GitHub.</p>
