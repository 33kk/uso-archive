### Description
Makes About:NetError display an image of your choice.  NetError occurs when a page fails to load, or when the network connection is disconnected or Firefox is in Offline Mode.  An excellent way to implement a new UI feature for your favorite web browser.

Thanks to user 4479 for Idea from this style - style 3836

Integrates with Style 6488 and if all styles are installed, you can customize both the about:blank look AND the new-tab's background image as well.

Features:
- Appears on Page Load Error
- Appears on Offline Mode
- Appears on Web Timeout
- Appears on websites that are down
- Clean and very appropriate default image, included with style (Firefox logo - shadowed)
- Removed borders from original error page, allowing the background image to show completely
- Fully customizable style, for your own enjoyment.

Additional Information:
This Style has been tested on multiple Firefox Themes, and has worked perfectly on each.
*NOTE: Please Clear your Cache and Restart your browser after installing or changing this style, your custom image will need to be cached in order for it to be displayed.

Instructions:
In the CSS, change the website image I provided to the image you wish to display
You may also change the background HEX color to match your image
The images that work best as the background are .JPG and .PNG
