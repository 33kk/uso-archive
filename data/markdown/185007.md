### Description
Changes the default theme for the blogging/social media website PillowFort. By default, the custom theme is an array of pinks, but it can be customized with user-selected colors. This will customize almost every page on the site, except for user blogs/profiles.

### Update notes
I recommend this site to find some nice custom colors:
https://dudleystorey.github.io/thenewdefaults/

Main Color: Used for the main background, links, post/avatar/comment borders
Sub Color 1: Used for link hovers, the Followers button, pagination, and tab hovers
Sub Color 2: Used for the Following button, submit buttons/sliders, and additional button outlines on the donation page
Sub Color 3: Used for the Mutuals button
Sub Color 4: Used for submit button/slider hovers
