### Description
Black theme for twitch with ffz

Installation:
- install <a href="https://www.frankerfacez.com/">FrankerFaceZ</a> extension
- enable dark theme on twitch
- Go to ffz settings > Appearance > theme > enable "Gray (no Purple)"
- install <a href="https://add0n.com/stylus.html">Stylus</a> extension
- install this style to Stylus
