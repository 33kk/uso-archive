### Description
This style fixes the device list at <a href="download.cyanogenmod.org">download.cyanogenmod.org</a>. It exposes actual device names, minimizes CyanogenMod codenames, and removes the hover behavior.

Users know their devices by the name printed on the case. CyanogenMod's download site lists supported devices by a manufacturer pre-production codename, which is only useful to the hardcore developers that write code for that device. Worse, the actual device name exists on the page, but only shows up when you hover over a codename, preventing you from using your browser's "Find" feature to search. They <a href="http://forum.cyanogenmod.org/topic/46860-device-code-names-of-cm9/#entry308357">like it this way</a>.
