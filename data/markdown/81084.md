### Description
This is an extension to my other plugin, Wikipedia - Readable Font (http://userstyles.org/styles/79021/wikipedia-readable-font) for Hebrew.
It replaces the ugly, grey boring font used in Hebrew Wikipedia with the brand new, marvelous open Hebrew font 'Alef'. 

** Make sure you install the font locally - http://alef.hagilda.com/ , otherwise the plugin won't work. **
