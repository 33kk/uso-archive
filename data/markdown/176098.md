### Description
This theme works on Salesianos Triana Moodle (2019). A educational website for the students of Salesianos San Pedro, Sevilla.

It's on alpha, because I didn't have time to test better colors and couldn't check all of the different type of cards the website has.

It doesn't upgrade the theme because it's certainly impossible due to the website it's poorly HTML designed but I did my best to get this theme ready.

I used part of the code of <a href="https://userstyles.org/styles/175631/moodle-essentials-dark-monochrome">Yhsanave Moodle Theme</a>
