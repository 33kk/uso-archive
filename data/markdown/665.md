### Description
Replacement for the WellRounded extension (https://addons.mozilla.org/firefox/933/ ) by Lee Netherton.

See style 666 for a version compatible with Firefox 2
See style 1079 for one that works in Thunderbird

Improvements:
Find in page box doesn't screw up when search box removed
Rounds all textboxes and menulists (experimental, please report any errors)

Recent fixes:
2006-09-29:
Tidied up selectors to increase efficiency (see http://developer.mozilla.org/en/docs/Writing_Efficient_CSS )
2006-08-15:
Fix Googlebar Lite search box problem
Fix Stylish "Specify Sites" input box centering
2006-07-25:
Fix Google Toolbar search box problem
NoScript "Address of website:" text centering
Added second option for menulist focusing
2006-07-20:
Fixed Add bookmark "Create in:" problem as well as related Web Search "Search in:" one (search dialog from Ctrl+K if no search box on toolbar)
Applied style to about:config


To try the second option for menulist highlighting remove the ending tag from the "Begin Normal Focus" comment and the starting tag from the "End Normal Focus" comment then close the "Begin Image Focus" and "End Image Focus" comments.
