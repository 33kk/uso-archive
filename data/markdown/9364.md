### Description
Makes the Site Identity Popup Transparent.

* Thanks to user 6431 for introducing me into the realm of Transparency with style 6771

The Site Identity Popup appears when you click the Favicon of a website, just to the left of the Awesome Bar.  The popup displays information about security, connection, cookies, and verification info.

Features:
+ Background of the popup list will be invisible (see through - transparent)
+ Bottom two edges will be rounded (bottom left and bottom right corners)
+ Background will return to opaque when the mouse is hovered over the popup (will be transparent until mouse cursor's hover event occurs)
+ Edges will return to squared at the bottom of the popup when the mouse is hovered over it


Applies to:
+ Site Identity
+ Passport information 

Browser Versions:
+ Works only on Firefox 3

Integrates with the following Styles:
+ Style 8661

Changelog:
