### Description
Restyling Panorama:
1. Removing all unnecessary items, such as: the favicons sites, the close button, a new tab button, the group name field, shortcuts AppTabs, the search button and exit from Panorama;
2. Adding transparency to the group;
3. Decrease the font size for headings tabs and shift them to the left.

--------------------------------------------------------------------------------

Рестайлинг Panorama:
1. Удаление всех лишних элементов, таких как: фавиконки сайтов, кнопки закрыти, кнопки создания новой вкладки, поля имени группы, ярлыки прикрепленных вкладок, кнопка поиска и выхода из Panorama;
2. Добавление прозрачности к группам;
3. Уменьшение размера шрифта для заголовков вкладок и смещение их влево.
