### Description
A dark theme for the Austrian newspaper "Die Presse".

Besides the dark style, the ads and social network buttons are hidden and the website is centralized.
As some domains differ heavily from the default one, not everything is supported (e.g. immobilien.diepresse.com, karrierenews.diepresse.com, ...)
