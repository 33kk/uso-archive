### Description
Często korzystam z rozkładu PKP (rozklad.pkp.pl) i denerwował mnie ogólny rozkład strony - szczególnie na szerszych monitorach oraz denerwujące reklamy i nieprzydatne informacje.
Napisałem krótki arkusz CSS dla tej strony. By zmienić u siebie jej wygląd należy:

a) zainstalować rozszerzenie Stylish dla Firefoxa:
https://addons.mozilla.org/pl/firefox/addon/2108

b) załadować mój arkusz CSS (z tej strony) do w.w. rozszerzenia.

Arkusz nie aspiruje do miana idealnego - jest moim pierwszym CSS dla Stylish. Jako, że strona napisana jest w brzydkich tabelkach - nie starałem się ich poprawiać tylko sprawić by wyglądały ładniej.
Mam nadzieję, że się przyda.
