### Description
Once again, this is a very simple script that changes the width of a story on a number of Fan Fiction websites.

This script is for fanficauthors.net, tthfanfic.org, hpfandom.net, and tenhawkpresents.com at present.

Edit - Added Spacebattles.com, YourFanfiction.com, Stardestroyer.net, ArchiveOfOurOwn.com and Ficwad.com.
