### Description
<p style="position:fixed;z-index:999;top:0;right:0;margin-top:0;width:calc(70% - 32px);padding:16px;color:#000;background-color:#ff6;text-align:center;font-size:20px!important;"><b>Firefox 57 or Stylish 3.0 users: Please go to <a href="https://www.jeffersonscher.com/gm/url-bar-tweaks.html">https://www.jeffersonscher.com/gm/url-bar-tweaks.html</a> to install.<br>Stylish 2.x users: Use the "Customize Settings" button before installation.</b></p>
Remove the top line added in Firefox 43+ (site - Visit, text - Search with...) from the address bar autocomplete drop-down, reduce size of Firefox 48+ dropdown or use an older-style two-line layout or Firefox 2 Oldbar layout, keep the dropmarker displayed even when not hovering the address bar. URLbar AKA Location Bar AKA Awesomebar<br><br>To restore a two-line drop-down in Firefox 48+, see "Drop-down Style" choice (2) below. For Oldbar fixed width URL-title layout, see choice (3).

The number of items on the drop-down is determined by a setting you can modify in the about:config preferences editor. Here's how:

(1) In a new tab, type or paste <b>about:config</b> in the address bar and press Enter/Return. Click the button promising to be careful.

(2) In the search box above the list, type or paste <b>URLB</b> and pause while the list is filtered

(3) Double-click the <b>browser.urlbar.maxRichResults</b> preference and enter the desired value, for example, change 10 to 15 (one is lost because of the hiding of the Visit/Search item, so 15 actually shows 14 rows)

(4) To test, launch a new window using Ctrl+n (on Mac, Cmd+n)

To update this style for the new number of rows, reload this page until you get a green Update button, then you can update the number of rows to display without a scroll bar, and Update.
