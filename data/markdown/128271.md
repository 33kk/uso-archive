### Description
Dark theme for GitHub - Switch off the lights on GitHub and browse GitHub in nighttime mode.


For bug reports, open an issue or submit a pull request on <a href="https://github.com/cquanu/github-dark">GitHub</a>.


Alternatively, copy-pasta the styles from <code>github-dark.css</code> into a new userstyle. Be sure to specify that it apply to sites beginning with https://github.com/*.
