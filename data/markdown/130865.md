### Description
pixiv 見る専の人用のスタイルシートです。

イラストを見る以外のものは極力排除しているので、
必要に応じてソースのコメントを参考に調整して下さい。


・広告など不要な要素を消す
・画像一覧の表示領域を広める
・不要な間隔を詰める
・ユーザーアイコンを右にするなどレイアウトを変更
・「無題」とタイトル付けされている画像はタイトルを非表示

※スクリーンショットは
Mac Firefox47、
Autopagerize アドオンを使用しています。

参考のスクリーンショットでは、ヘッダーも消していますが、
一応デフォルトの設定ではコメントアウトして無効化されています。

ヘッダーはたまに使いたいという人は、
下記をコピペして別のStylishファイルに分けると、
必要に応じて部分的にオンオフできるようになります。
また、作品も投稿せず、検索もFirefoxの検索エンジンからやってしまうのであれば、
そのままこのStylishファイルに書き加えてもいいかもしれないです。


  @-moz-document domain("pixiv.net") {
  .layout-wrapper{
  display: none;

}
}
