### Description
Стиль открывает возможность гулять по бутику, при не открытых бонусах. Прежде чем использовать данный стиль, ОБЯЗАТЕЛЬНО прочитайте полное описание. Действует для:  Moyabimbo Ma-bimbo

### Update notes
Во-первых. Самое главное. Вы не сможете купить вещи не активировав 1 из имеющихся бонусов. 
Благодаря этому стилю вы можете только пройтись по бутику и примерить вещи, посмотреть расцветки или узнать цены и даже пополнить корзину.
НО! Для покупки активация бонуса обязательна! 
 
(!) Всплывающие окно со списком бонусов - это тоже самое окно, в котором находятся расцветки вещей. Если полностью его убрать - вы не сможете открывать и выбирать расцветки. 

Что бы временно закрыто окно  - откройте расцветки любого предмета, вы перейдете на всплывающее окно расцветок данного товара. Это окно уже можно закрыть. 
НО! при переключении между разделами или страницами - окно бонусов снова откроется. 

"Высота, на которую нужно опустить всплывающее окно." Для большего удобства, вы можете выбрать наиболее удобную для вас высоту окна исходя из особенностей вашего экрана. 
"Кнопки переключения страниц" Для большего удобства использования стиля у вас есть возможность перенести управляющие кнопки на видимую позицию (рядом с корзиной), для того что бы не приходилось выключать вплывающее окно на каждой странице. 
(!) Для открытия примерочной необходимо закрыть действующие всплывающее окно.

P/S: Оптимальное отображение стиля при масштабе страницы 75%

По настройкам: 
Если вам нужно полностью открыть окно магазина, то в графу:
"Высота, на которую нужно опустить всплывающее окно" 
выставляем значение - 1000
