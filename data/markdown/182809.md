### Description
I recommend using the <a href="https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne">Stylus</a> browser extension.

This is a simple customizable theme for 2b2t.io 
The default configuration is dark but it is fully configurable. 
If you like the look of any of the examples images their configurations are as follows:

Text : FFFFFF
Total: F5A623
Que: 006CFF
Prio: FF0000

(Grey) Background: 414141
(Red) Background: 800000
(Green) Background: 008000
(Blue) Background: 000080
