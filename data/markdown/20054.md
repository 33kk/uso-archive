livetube（http://livetube.cc/）の放送ページのコメント欄を適当に調整
amazon広告が戻ってきたので調整しつ修正
２、３個の映像サイズに対応しただけ

コメント欄のサムネイル画像を変更をしたい場合は、以下のように変更してください。
ソースには以下のように記述されています。
デフォルトでは20x20pxのサムネイル画像を表示させるように設定してます。

/* コメント欄のサムネイル画像を消す
	display: none			!important;
 */
IMG[align="left"] {
	width:20px			!important;
	height:20px			!important;
}

コメント欄のサムネイル画像を消す場合はこのように変更する。

IMG[align="left"] {
	display: none			!important;
}

サムネイルのみのトップページはここで変更
@-moz-document url("http://118.6.134.123/")
