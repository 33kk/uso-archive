### Description
I didn't really like any of the default built-in themes of Vivaldi WebMail so I decided to create a nice easy on the eyes dark theme based on Arc Darker theme, a GTK+ theme for Linux distributions.

It's not 100% complete theme rework, so some things will look a bit off. Most of what's completed is in the main mail inbox section.
