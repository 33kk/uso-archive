### Description
RARBG style that is EXTREMELY Minimalistic, many website features are hidden.
    This style was made for personal use, so anything I didn't use was removed.

Custom color picker for the main color. Recommended Colors: {
        White:    #fff
        Orange: #ff6600
        Blue:      #3860BD
                      #64b8e6
}

Removes all unnecessary, or unused (by me) elements on the website.
    Hides all search categories other than "Movies", and "TV Shows".
    Removes login / user / comments fields (keeps the comments sections, but doesn't allow posting).
    /index.php page isn't stylized because I use a bookmark that takes me to the TV Shows sorting
    Removes "Recommended torrents", and all extra website tabs other than "Torrents".

Pages styled:
    /torrents.php - Main search page for torrents
    /torrent/         - Individual torrent page
    /tv/                 - Complete TV Show listing with Season / Episode list.
