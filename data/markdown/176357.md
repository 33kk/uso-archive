### Description
注意：腾讯文档最近经历过一次更新，只有更新后创建的文件才能适配黑暗模式。老文件在开启黑色模式之后会出现 CSS 错乱。

默认情况下文章本身仍然会是白色背景，仅把 UI 元素变为暗色。如果需要修改，请选择性启用代码中的前三段代码中的一段，第一段负责将整个文章反色；第二段负责降低背景白色的亮度；第三段为【实验性】仅改变编辑器背景和黑色文字，保持其它颜色不变。启用其中一个即可。
