### Description
Moves reload/stop buttons to the left of the address bar and optionally makes them appear only when hovered.

### Update notes
Don't forget that you can still review security and site permissions by right clicking on the page and selecting "View page info", this style just removes the possibility of clicking the identity box itself (if you so choose).
