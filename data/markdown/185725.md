### Description
A dark theme for New Blogger.
It does include stuff for their old theme, but that serves more like an anti flash bang in case you accidently opened the old theme.

Specials:
- HTML Editor after Notepad++ Deep Black XML style
- Semi-Easy adjustment for your personal taste since it's only using 6 colors
- Some smaller spacing improvements that I thought were useful
