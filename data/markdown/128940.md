### Description
A dark version of <a href="https://userstyles.org/styles/121229/better-modern-square-tabs-for-firefox-australis">Better Modern Square Tabs for Firefox Australis</a> by <a href="https://userstyles.org/users/303580">JoelDroid</a>.

### Update notes
The Firefox theme I'm using is <a href="https://addons.mozilla.org/en-US/firefox/addon/dark-fox-18066/">Dark Fox</a> by <a href="https://addons.mozilla.org/en-US/firefox/user/randomaster/">randomaster</a>.
I've got my reload button to the left thanks to the <a href="https://userstyles.org/styles/122664/firefox-42-left-reload-button">Firefox 42+ Left Reload Button</a> style by <a href="https://userstyles.org/users/305216">Trixlight</a>.

<b>2016.06.15</b> - Made the active tab and navbar's background colour slightly lighter and made the text cyan to improve the distinguishability among other tabs.
I've also added some options that let you choose the colours of the active tab. So if instead you want the text colour to be red instead of cyan, then be my guest.
