### Description
Written to assist me to read postings at the many Web interfaces for gmane.org.
The Stylish style Uses the Anonymous Pro monospace font as main font for reading the posted articles.
The style attempts to preserves whitespace structure and contributor intention for the mail list article posting archived at gmane.org, while changing some presentational attributes of non-crucial elements. Soft reverse colors are dominate in the content areas, to ease eyestrain.
