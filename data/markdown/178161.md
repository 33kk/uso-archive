### Description
Enjoy the best Dueling Nexus theme. Designed with love.

Apart from the instant visual enhancements. The custom vector game field scales to fit any screen size. The layout have been enhanced specially the card description/chat area. The card selection window will not interfere with the card description anymore. And various other minor changes and game-play animations to enhance your user experience.  

♦ ♦ ♦ You can select any background you prefer besides the included images  (by clicking on Customize Settings) from this page.
You have two background options now:
-The duel background.
-The background for the rest of the website.

♦ ♦ ♦ Make sure you turn on auto-updates for styles on Stylish/Stylus extension settings.

This skin only affects the duelingnexus game/duel & deck editor window so far, more sections will be designed in time.

Amazing Yu-Gi-Oh! backgrounds are created by: maxiuchiha22
https://www.deviantart.com/maxiuchiha22
