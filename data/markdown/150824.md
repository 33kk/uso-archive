### Description
Modifies the look of the Stream Layout for Countdown. Defaults to a dark them but will be customizable to whatever you want in the future.
<br/>
<br/>
List of changes other than coloring:
<ul>
<li>Size of the Numbered Preset buttons. (this fixes the text not being centered)</li>
<li>Margin on top of the "Ready" button to be the same as all the other buttons.</li>
<li>Centered Icons for item/equipment/quest/mask menus.</li>
</ul>
<br/>
Use the "Advanced Style Settings" button below to set custom colors.

### Update notes
