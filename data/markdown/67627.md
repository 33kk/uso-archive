### Description
Unlike Firefox and IE, Google Chrome does not automatically look for characters in Mingliu_HKSCS fonts if characters are not found in the default fonts.  This can be solved by this style.
I have defaulted the fonts to Microsoft JhengHei which I find in a much clearer font in Windows 7.
