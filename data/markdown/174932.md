### Description
A more compact layout to reduce the amount of scrolling up and down in project workflows on <a href='https://www.zooniverse.org/'>Zooniverse</a>. Primarily aimed at the larger species option lists in the <a href='https://www.zooniverse.org/organizations/meredithspalmer/snapshot-safari'>Snapshot Safari</a> projects and the longer more detailed question tasks like <a href='https://www.zooniverse.org/projects/sassydumbledore/chimp-and-see'>Chimp&See</a>.

Task features:
- Task & Tutorial bar removed, tutorial button now a side tab
- Fixed size exemplar images, less position 'bounce'
- Descriptive text hidden, reveal with mouse click
- Less padding, less scrolling!
- Footer bar hidden, most links available elsewhere

List features:
- Smaller margins, more space!
- Single line options, neater lists!

Also has an experimental portrait layout for tablet screens with larger fonts and button sizes.

All feedback welcome.
