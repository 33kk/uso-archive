### Description
A very basic filter (for the forum only) that eliminates the bulky header which is common to the main Runes of Magic website.

NOTE:
You will lose the main web site links: Register, To the main website, Search 
as well as the links to UK, US, EU and DE forums.

You do not lose any forum search links.

Suitable for RoM forums at any of the above 4 regions.
