### Description
A redesign of the mastodon ui based on the pico-8 pallette (with a few extra shades for contrast).

Features:
  * Flexible columns
  * Arbitrary number of columns
  * More colorblind friendly
  * Warmer and more varied colors
  * Clearer visual hierarchy

(note: uses a lot of transparency and drop/box shadows, so performance may possibly be affected on less powerful computers.)
