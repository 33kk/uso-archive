### Description
A steampunk style for RPOL.net with gears, cogs and swirly decorations. Steampunk'd works best over RPOL's existing white theme, though it can be used over any.

I've updated this theme to work with the recent site upgrades on RPOL.net (RuBB v.1.9.54).
