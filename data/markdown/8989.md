### Description
Makes Style 8977 work even better by placing the Close Button on the Left.
** Only install this style if you have the above style installed!!! **

Features:
Close Button is on the Left
Text of Website (or Label) is moved to the Right

Instructions:
1. Install Style 8977
2. Install This Style
3. Move your mouse over to the right-hand side of screen
4. Slowly move your mouse over towards the Sidebar until the RESIZE icon appears as your mouse icon
5. Resize the sidebar to be as small as you can make it.
6. Enjoy the smallest sidebar when browsing, and when you hover over it, it will become un-compressed!

Install this style along with Style 8977

Browser Versions:
- Works only on Firefox 3

Integrates With Style 8977
