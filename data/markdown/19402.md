/* 
 * ******************
 * * USER GUIDE 4.0 *
 * ******************
 *
 * with this style you can add a style to a line in the league 
 * list in challenge page.
 * you can use different style according to different flag cheasing level
 * 
 * The levels are 4
 * 1- BOTH MISSING (green)
 * 2- HOME MISSING (yellow)
 * 3- AWAY MISSING (red)
 * 4- completed (line through, and opacified)
 *  
 * In the style there is a list of code called ID-LIST, 
 * in that list each row contains the correct national ID 
 * and the name of the nation (commented).
 * When you play a friendly match you have to pick (select and copy) 
 * the corresponding line, go to the end of the page and paste 
 * the line just between the comment "begin paste here" and 
 * "end  paste here" .
 * the code looks like that : 
 * 
 * div[id*="CPMain_pnlPoolDDLs"]  select[id*="CPMain_ddlPoolLeagues"] 
 * option[value="1"] /*	Sverige			*/
 * 
 * you have to copy the line corresponding to the nation you played with 
 * and paste it in the corresponding region (1 or 2 or 3 or 4)
 * save and close ^_^
 * 
 * enjoy
 */
