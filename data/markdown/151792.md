### Description
Makes the new Imgur home page layout to be linear instead of the new non-linear layout.
<br><br>
Option: Trending Tags - Can display or hide the Trending Tag banner on the Home page
<br>
Option: Favorites Page - Can apply the linear style of the Home page to the Favorites page
<br>
Option: Comment Enhancements - A healthy mix of the new and old style comments
