### Description
Animates the background and alert of the findbar when you type in a search that cannot be found.

Works only for Firefox 3

** Although this is an ORIGINAL style, I would like to thank user 6587 for style 5928, which introduced me to the find-bar customizations.

Features:
+ Animated background of the findbar text when the following occurs:
+ Animated Alert icon when any of the following occurs as well:
 1. The phrase is not found
 2. Reached the end page, continued from the top
+ Text of this alert will be Bold
+ Text Background will be white, so that you can still read the text
+ More flashy to the eyes, notifying you that you entered some incorrect information
+ Animation and background color may be customized


Applies to:
+ Quick Find
+ Find Bar

Performs Like:
+ FAYT (Find as you Type) Addon
+ Find Toolbar Tweaks Extension


The FindBar can be accessed by pressing the Key Combination (Ctrl+F   or   Command+F on MACs)  It can also be activated by typing on a page if you have the right setting (Tools / Options / Advanced Tab / ENABLE "Search When I Start Typing")
