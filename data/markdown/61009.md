### Description
For Firefox 4+ and latest Firefox Beta, Aurora, and Nightly builds. 

This style makes toolbar buttons positioned in the tab bar look like Fx Linux OS rounded tabs on Windows OS.

<b>!Important!</b> After applying this userstyle you may have to toggle the Window state from (Minimize to Maximize) or restart the browser, so the style will appear correctly.

Works well with Userstyles:

style 44825

style 45584

Tested on WinXp with Firefox 4+ and up to the latest Firefox Beta, Aurora, Nightly builds.
(<b>Style not tested on Win7/Vista, but may work without problems...</b>)
