### Description
By default works with: http://pastebin.fossnet.info/

* Make use of screen width by setting a few elements width to 90~100%. 
* Add darker colors which aren't as vanilla as the defaults.

You will have to edit the script (line 1) to work with other pastebins running the same software.
Example: http://pb.gshellz.org/
