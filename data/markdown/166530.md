### Description
<b>Description</b>

<a href="https://Instagram.com/">Instagram</a> recently blocked users from selecting text on PCs. That's unfair!
With this style, you can select and copy any text in the description of a post.

<b>How to contribute</b>  

Go to <a href="https://github.com/Kenya-West/Instagram-selectable">GitHub</a>, clone this project and start your changes. If you want to use styles from source code, then create new style in Stylish extension and put source code in opened window.

<b>How to use this style?</b>

You need <a href="https://github.com/openstyles/stylus#releases">Stylus</a> extension installed on your browser to use this style.
