Automatically Hide Firefox navigation bar(which contains stuff like home button, backforward button, addressbar & searchbar).
 
Show it up Floatingly above page content WITHOUT squeezing when mouse hover browser top area.

Recovered & bugfixed from <a href="http://userstyles.org/styles/45652/firefox-4-autohide-navigation-bar-alternative">Firefox 4 - Autohide Navigation Bar Alternative</a>

Now it's fully support small Icon as well as Large Icon Mode.
