### Description
This style aims at switching the position of Appinn's content block and sidebar with additional features:
# wider main content and comment containers;
# move the google ad aside;
# optional slide-open sidebar**;
# the avatars no longer cover the comment text!

**If you have a smaller browsing width like below 1280px, you might feel like a sidebar sliding effect. If so, uncomment the following lines:
/*div#sidebar:hover
 {right: -20px !important;
  -moz-box-shadow: 1px 0px 23px #333;
  -webkit-box-shadow: 1px 0px 23px #333;}*/

--------------------
Appinn 最近的改版让主体内容放在右，而这个样式则是为了把侧边栏放回右边而主体内容和评论放回左边，并添加以下特性：
# 加宽的内容和评论；
# 给广告一个“好一点”的位置；
# 可选的滑动打开侧边栏**；
# 评论者头像不再挡住评论文字了！

**如果你的浏览宽度小于 1280 像素，你可能会喜欢滑动侧边栏，那样的话你只需要把代码中如下几行的注释去掉：
/*div#sidebar:hover
 {right: -20px !important;
  -moz-box-shadow: 1px 0px 23px #333;
  -webkit-box-shadow: 1px 0px 23px #333;}*/
