### Description
Um es richtig ausführen zu können, müssen Sie ein paar IP-Adressen des Stromzählers im CSS Code ändern.

Am Anfang des CSS-Codes:
@-moz-document url("http://IP-ADRESSE des Stromzählers/index.html"), url("http://IP-ADRESSE des Stromzählers/status.html")

Am Ende des CSS-Codes:
a[href="https://IP-ADRESSE des Stromzählers/index.html"] {
display: none !important;
}
a[href="https://IP-ADRESSE des Stromzählers/installateur/index.html"] {
display: none !important;
}
