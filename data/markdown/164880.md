Customization Options Notes: To get colors, use a color picker, for example https://pixlr.com/editor/ is online software with a color picker. For the RGBA, it will say customcolor#transparent#, for example: "customcolor1transparent", which will use an RGBA version of the first color. These use RGBA values, and userstyles does not have color picker support for RGB or RGBA, use this link: https://www.rapidtables.com/convert/color/hex-to-rgb.html to convert a hex code to RGB, and leave THE LAST NUMBER (will be 0.5 OR 0.65) alone
