### Description
Override the mystery meat navigation in the new Gmail UI. Adds text labels back in alongside the Gmail icons.

Recommended:
http://userstyles.org/styles/56063/gmail-easy-access-colored-buttons Gmail Easy Access Colored Buttons

EDIT:
Gmail now has an option to switch the standalone icons to text (Hurrah!), but this userstyle is still useful if you want both - keep the icons enabled on Gmail, and this userstyle will add the labels alongside the icons.
