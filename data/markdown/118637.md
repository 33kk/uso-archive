### Description
It changes the default cover photo of Last.fm.

The cover photo size should be:
- your screen width resolution x 200px (e.g. 1600 x 200px)
- preferabily in darker colors for a better contrast with the header text

Instructions:
- upload the cover photo to a filehost (e.g. Imgur, Imgbox, Imageshack, Photobucket, etc)
- copy the direct link of the picture to the input box below (it must be the original/fullsize picture, not a thumbnail)
- install the script
