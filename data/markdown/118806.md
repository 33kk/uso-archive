### Description
Features:
- White text-shadow behind black text (Ash theme). 
- Smaller font, monospace.
- Darker colors, overall.
- Better spacing between nicknames and the message.
- Brighter time.
- Small drop-shadow behind content in the chat window.
- Works with all 8 IRCCloud default colored themes.

I update this theme often so if you notice a bug or want something added, post below.
