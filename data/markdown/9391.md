### Description
MBlue Nav-Style Buttons Vista
--------------------------------------------------------------------
Description:
 1. Transforms the TOOLBARBUTTONS and make it like the NAV-BUTTONS: same blue background and dropmarker.
 2. Can use Separator/Spacer to customize the buttons (read below).
 3. Supports SMALL and BIG ICONS.
 4. Integrates with "Combine Nav-Buttons Vista".
 5. Integrates with the other Nav-Style styles.
--------------------------------------------------------------------
Notes:
-Using other Styles/Extensions/Themes may change some margin/border settings, making the buttons not aligned. If so, try not using other styles that changes margins in the Nav-bar, and use the default firefox theme. 

-Some Ext. Icons (like Download Helper) will not work correctly.

-Works in the Nav-Bar only.

-Even if this style supports big icons, it works better with small ones, because the default big icons are too big to fit inside the buttons, so this style resizes them.

-You can optionally change the OPACITY of the buttons.
--------------------------------------------------------------------
How to Customize the Buttons:
 Follow the Instructions included. Also see the screenshot below.
