# Python documentation 字体美化。

Python documentation正文默认使用sans-serif，阅读起来感觉有点吃力，所以对字体稍微修改了一下，外观等其他方面未做任何改动。

===============================================================
使用的字体有，可以自由组合：
  * 正文组：Calibri, "Lucida Grande", Verdana, Arial
  * 标题组："Bookman Old Style", "Gerogia", "Times New Roman"
  * 等宽组：Anonymous, consolas, monaco, "Courier New"
===============================================================
