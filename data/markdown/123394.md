### Description
- Non-destructive
- Reduces the paragraph width to book format.
- Makes links only visible when hovering the paragraph/list containing the link
- Highlights links when focused
- Uses (default) serif font with increased font size
- Increases content font size.

Use the browser settings to change the font to the one you like.
