### Description
Сверху закреплена панелька cо ссылками (Новости, Галерея, Форум, Уведомления, Трекер, Wiki, Поиск, профиль и кнопка выхода).
<a href=http://img10.imageshack.us/img10/7161/4m5g.png>Теги перемещены вправо</a> в списке тем для повышения читабельности. Убраны границы ячеек таблицы, благодаря чему также повысилось читабельность.
<a href=http://img824.imageshack.us/img824/5808/3ujv.png>Убраны квадратные скобки у кнопок</a> (<a href=http://img571.imageshack.us/img571/773/c88e.png>ещё пример</a>).

Топик стиля: http://www.linux.org.ru/forum/linux-org-ru/8690347

Стиль работает только с темой Black.
