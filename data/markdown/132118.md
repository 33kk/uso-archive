### Update notes
Tested on Firefox and Chrom(ium).
Uses css variables, requiring a recent browser version. http://caniuse.com/#feat=css-variables

The flags are from <a href="https://github.com/lipis/flag-icon-css">lipis/flag-icon-css</a> and can be found <a href="https://gist.githubusercontent.com/xiyzzc/83f6629cc987ad4256be174c3798bab2/raw/5257775b8d8ee2cfc5e127c024285a2112f34574/tftv-svg-flags.css">here</a>.

To install the flags in Firefox:<ul><li>Create a new blank style</li><li>Paste all the text</li></ul>
To install the flags in Chrom(ium):<ul><li>Create a new style</li><li>Click <b>Import</b> under <b>Mozilla Format</b></li><li>Paste all the text</li><li>Click <b>Overwrite Style</b></li></ul>
Make sure to give the style a name and click <b>Save</b> in both cases.

I also use <a href="https://gist.github.com/xiyzzc/34cc143b689ca630a442f572e43f3218/raw/a030733e895be72aa61ce2f9b2755bc538049749/image-relinker.user.js">this userscript</a> to embed links to images as an image.
