### Description
This does not need DarkBox to work! This can be the only theme on Google Inbox if you wish! It has no coloring alterations.<br><br>This theme will widen the navigation panel so you can read more text on your 'labels' while decreasing the line height to include more labels with out the need to scroll as often.<br>
