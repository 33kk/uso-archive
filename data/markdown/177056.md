### Description
This style lets you change the appearance of scrollbars among all major browsers (Firefox, Google Chrome, Opera, Brave, any chromium-based browser...).

You can choose the colors you want, or use the default dark with blue thumb scrollbar.

On Firefox you can only do a basic recolor of the scrollbars (unless you use the method explained in <a href="https://github.com/Izheil/Quantum-Nox-Firefox-Dark-Full-Theme/tree/master/Full%20dark%20theme/Alternative%20scrollbars%20%26%20tooltips">repository on github</a>.

Alternatively, you can install the <a href="https://github.com/Izheil/Dark-userstyles#dark-scrollbars">userCSS version</a>, which will allow you to customize the scrollbar at any moment from the context menu of Stylus addon.
On Chrome you can change the colors right away, and hide the scrollbar buttons. By default the scrollbar has flat colors, but you can set it to use gradients (changing the 2nd colors).
