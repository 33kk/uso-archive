### Description
Customize the Body of Feed Previews when you open a Feed using Firefox's built-in subscribe page.

See the before and after screenshots to see what preview page I mean!

** Code is based on user 3059's similar style 3862.

Features:
+ Header image will be a faded Firefox logo with shadowing
+ Supports a huge list of Fonts (discussed below)
+ Image supplied in default code is hard-coded - allowing you to view the background, even when offline!
+ Background image is static and is centered on the window.
= Optional - Font size of the popup will be larger (must uncomment)
= Optional - Bold text for easier readability (must un-comment)
= Optional - Remove the Horizontal Rule just below the feed description (must uncomment)

Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font

Applies to:
+ RSS Feed View Preview Page
+ Live Bookmarks Subscribe screen
+ Works on RSS, Atom, XML, NetVibes, ATOM, JSON, CSV, YAML, and XSL feed pages

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2

Additional Information:
+ Feel free to edit the CSS.  You can customize the fonts, enable bolding, and change the background image, even change the font size!  No Re-posts without asking though!

Integrates with the following Styles:
+ Style 8971
