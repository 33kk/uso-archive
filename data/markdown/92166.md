### Description
There are plenty of extensions and userstyles out there that add a restart option to Firefox via a toolbar button or menu option, but none that simply add a restart button to the Add-ons Manager. I prefer this configuration because it is simple, unobtrusive, and is accessible from the Add-ons Manager where it is most commonly needed.
