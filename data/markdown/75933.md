### Description
For those of you who hate FB chat as much as I do...

   For best results, set yourself to be offline to all but one person:
   - Temporarily disable this style
   - Go to 'Advanced Settings'
   - Select 'Turn on chat for only some friends...'
   - Enter the name of somebody you know doesn't use chat
   - Then enable this style
