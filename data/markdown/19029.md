### Description
Created by PKF_647, this script changes the default Bungie.net theme to a brand spanking new one.
Userscript file: http://macro-halo.110mb.com/Bungie%20Themes/Blue%20Terminal/Blue_terminal_Bungie_site_Theme.CSS

To create the Userstyle:
1. Go to the script file and copy all (CTRL+A , then, CTRL+C)
2. Right click on the Stylish Icon in the Firefox Statusbar.
3. Go to menu Write Style
4. Then go to menu Blank Style..
5. With the blank page open, add the title "Blue Terminal - Bungie Theme"
6. Click in the body of the blank page (the large white space) and paste the contents of the script file.
7. Once everything has been pasted into the blank style, click save.



DO NOT LOAD THIS INTO STYLISH VIA THIS PAGE,  IT MUST BE DONE MANUALLY.
