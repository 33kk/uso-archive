### Description
<p>This is a style which changes the default VNDB interface and adds some functions like the ability to navigate between screenshots with arrows, clicking outside the screenshot box to close, better tag readability and so on.
The style uses code from "Better VNDB.org" and "VNDB Dark and Flat", so all credit for these parts belong to the original creators.</p>

<p>Here is a list of the functionalities:</p>
<p>[From "Better VNDB.org"]</p>
<ul>
<li>Original names in parentheses next to links</li>
<li>Font resize</li>
<li>VNDB title restyled and top of page reduced in height</li>
</ul>
<p>[From "VNDB Dark and Flat"]</p>
<ul>
<li>Tags aligned in columns</li>
<li>Smooth animations on images</li>
<li>New font</li>
</ul>
<p>[NEW with VNDB Complete]</p>
<ul>
<li>Rounded corners for boxes and tabs</li>
<li>Titles under thumbnails on homepage</li>
<li>Screenshot navigation with arrows</li>
<li>Close screenshots by clicking outside the box</li>
</ul>
