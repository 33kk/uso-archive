### Description
This Firefox style is an imitation of the GTK 3.14/16/18/20 theme <a href="https://snwh.org/paper">Paper</a>.  The screenshot at right shows its similarity to the file browser.

This theme formerly enabled the findbar to automatically expand to fill the available width.  That functionality is now contained in the separate style <a href="https://userstyles.org/styles/131934">Auto-Expand Ctrl-F Findbar</a>.
