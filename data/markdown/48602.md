### Description
DudleFox, or maybe 'Yet Another Firefox Glass-Style But With Some Nice Optional Tweaks' ;D
It's my first style so there are probably a few bugs (please contact me if you found one or more ;D)

Features:
- Firefox Glass-Style (for Firefox 4+)
- Transparent Tabbar, Navbar, Bookmarkbar, Addonbar, Searchbar
- Transparent about:blank
- Transparent Webdeveloper-Toolbar (if you have it)
- Transparent Download-Statusbar (if you have it)
- Tabs-not-on-top-Support
- New Appmenu-Button/Firefox-Button (optional; thanks to user 62569 for the graphics)
- Show Bookmark-Item-Text only if hovering (optional)

-----------------------------------------------------
PLEASE READ:
Restart Firefox or just resize the window for best result!
