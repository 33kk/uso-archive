### Description
Microsoft Vista Aero Glass Transparency effects can now be enabled by the Glasser extension: http://forums.mozillazine.org/viewtopic.php?t=649449

Inspired on style 3874 by user 241 and style 5189 by user 1203 this style mimics IE7 behavior with Vistaglazz applied.
Vistaglazz available from http://www.codegazer.com/vistaglazz keeps maximized Window borders transparent.

Tested on firefox 3.0 and default theme only.
For clearer picture see http://forums.mozillazine.org/viewtopic.php?t=633274

Tip: great with style 7589
note: Lost your bookmarks toolbar? Hover your mouse over the navigator toolbar ;-)
