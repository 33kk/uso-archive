### Description
An enhanced theme for Discord!
<b>Changes:</b>
Translucent menus, user profile pop-ups, other pop-ups;
Chat box is round and has a shadow when hovering over it & typing;
Small transitions for buttons, icons and channels;
A slightly smaller topbar for more room;
Shadows for user profiles, mute/settings buttons, normal buttons, linked accounts on user profile, bot embeds, images, server icons,search bar, dropdown boxes.
Buttons are round..
Social Media links in settings are invisible.
Space between messages is slightly smaller.
