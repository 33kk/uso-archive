### Description
Specify the amount of Live Bookmark Items to show when displaying Live Bookmarks in the bookmarks menu.
This allows you to bring up your favorite bookmarks, while conserving more screen space, and faster loading times.  This style works on both Firefox 2 and Firefox 3.  Compresses the Live bookmarks menu. 


Features:
- Normally, the Live Bookmarks Items will ALL display.  This style filters out the newest 6 (by default), but can be easily customized.

Browser support:
+ Works on Firefox2
+ Works on Firefox3

Extension Support:
+ This style works Perfectly with the Brief RSS Reader Firefox extension.

Additional Information:
- Simply edit the CSS and Add or Delete more lines.  (Notice how each line expands by + menuitem)

Integrates with Style 7784
