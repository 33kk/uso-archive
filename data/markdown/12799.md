### Description
Allows you to add a Custom Background to the context menu "Set As Desktop Background" popup.
This popup appears when you right click on the main menu, from the context menu.

Features:
+ Background image will be a faded Firefox logo with shadowing
+ Image supplied in default code is hard-coded - allowing you to view the background, even when offline!
+ Background image is static and is centered on the window.

Applies to:
+ Set as wallpaper prompt
+ Center, Tile, or Stretch

Browser Versions:
+ Works on Firefox 3
+ Works on Firefox 2

Additional Information:
+ Feel free to edit the CSS. You can customize the fonts, enable bolding, and change the background image, even change the font size!

Integrates with the following Styles:
+ Style 9446
+ Style 8306
