### Description
Displays keyboard shortcuts for Gmail.  For more detailed shortcut tips, type "?" for display (Gmail default feature).  Full description is here: http://mail.google.com/support/bin/answer.py?hl=en&ctx=mail&answer=6594

NOTE:
You need to ENABLE keyboard shortcut feature in settings.
Certain Lab feature may destroy the style.  If you are using Tasks, see discussions below for more.
