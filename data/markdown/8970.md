### Description
Always shows Firefox 3's new StatusBar feature, the Download Monitor!

Features:
+ Always shows the download icon (if there were NO downloads since last browser restart)
+ Always shows the download text (if there WAS a download since last browser restart)
+ Clicking on the icon will open up the download manager

Applies to:
+ Downloads statusbar icon
+ Download Monitor

Browser Versions:
+ Works ONLY on Firefox 3

Integrates with the following Styles:
+ Style 8809

** Warning, with this style enabled, if you download a file, the download monitor status bar will become huge. **

Future Releases:
+ I am currently working on a way to fix the Above starred bug where the monitor shows text after downloading a file.
