### Description
Removes the "Split" menu from the Firefox's main menu toolbar.  

This is the menu drop down list that gets displayed when you install the Firefox Extension, Split Browser.
Gives you more room for extension toolbars, items, or buttons.   Now works for Firefox2 and Firefox3.


Additional Information:
Works on BOTH Firefox 2 and Firefox 3 !!!
Gives you the ability to only enable the menus you use the most!

SplitBrowser is an amazing Add-on, unfortunately, it is impossible to remove the SPLIT menu on some occurrences (BUG with the extension's options).  Luckily, this style fixes the bug, and returns your menus back to normal
