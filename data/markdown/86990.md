### Description
Derived from TODOIST Plus. Fixes some of the things wrong with it, and does a whole lot more.
<ul>
  <li>Adjusts width to the window. Moving the sidebar to the top when the window isn't wide enough.</li>
  <li>Pads and styles each item more so each item is more prominent.</li>
  <li>Properly aligns the handle onto the left border. Aligns the show/hide arrow with it's child.</li>
  <li>Hide the premium ads, premium features, and the karma menu. Remove the #completed_items { display: none; } if you want the completed hover menu back.</li>
  <li>Use a different icon for the todo_item list button. The 'config wheel' icon is overused.</li>
</ul>
