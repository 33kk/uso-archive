### Description
I followed the twinkling stars background tutorial. You can find it <a href = "https://www.script-tutorials.com/demos/360/index.html"></a>

http://www.script-tutorials.com/night-sky-with-twinkling-stars/

Only applies to threads

The twinkling image opacity has been set to 75% opaque so stars don't completely disappear - if you want to change that, get the original background and set the opacity of the layer in gimp. 
Animation lasts 10 seconds to move 100px,100px.
