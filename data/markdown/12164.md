### Description
Allows you to Customize the text and background image of Save File popup prompts.

Features:
+ Background image will be a faded Firefox logo with shadowing
+ Image supplied in default code is hard-coded - allowing you to view the background without loading an image from a website
+ Background image is static and is centered on the window.
= Optional - Text size of the update details will be larger (must uncomment)
= Optional - Bold text for easier readability (must un-comment)
+ Supports a huge list of Fonts (discussed below)



Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font

Applies to:
+ Downloading Files popup Opening file prompt
+ Choose where to save
+ Save File or Open With
+ Torrent

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2

Additional Information:
+ Feel free to edit the CSS.  You can customize the fonts, enable bolding, and change the background image, even change the font size!  No Reposting without asking please!


Integrates with the following Styles:
