### Description
Mantis Bug Tracker custom CSS attempt to match MantisBT 2.0 Modern UI and some extra adjustments. Note this is intended for MantisBT < 2.0. Prior to implementation of Modern UI. I use it on 1.2.15. I have Priority and Category columns disabled so the coloured cells are in the Status column, not the the Summary column.

Style includes:
* Higher contrast ratio (less eye fatigue)
* Grey table grid
* Larger checkboxes
* Larger text areas
* Right aligned category text
