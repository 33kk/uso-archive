### Description
Don't you just hate when your dialog boxes are cut off because the custom font you're using doesn't fit.  This style will fix that problem for those of us who use fonts other then default.  And if you have Downloads in Tab extension it will also remove the empty space in options and adjust the properties dialog height.

Features:
- Resizes the Fx Options window as well as some other parent windows
- Supported extension options
-- Download in Tab
-- Organize Statusbar (OSB)
