### Description
A dark theme for NewsBlur.

No structural changes made to website, it's a pure color theme. A couple of images were remade for dark backgrounds and embedded on the CSS as Base64 code. I also removed a few instances of shadow and gradient effects when I found them to not work as well, or just when I thought it would work better without.

The CSS code is a bit messy and could use some cleanup and reorganization since I wrote it directly on the Stylish editor. I'll work on cleaning it up once I have a final version. Consider this a beta version, and please report any problems.
