### Description
For Firefox 6+ and latest Firefox Beta, Aurora, and Nightly builds.

This style Autohides the Urlbar Identity-box label, so you can reclaim some address bar space until it's site favicon is mouse hovered. This style is a modified version of (user 17622) style mentioned below to provide better compatibility with Third Party Themes and Urlbar styling userstyles on Fx6 and newer.

YouTube video found here:
http://www.youtube.com/watch?v=jbxn8FEECDo

Works well with styles:
style 46915
style 49786

Tested on Firefox 6+ and up to the latest Firefox Beta, Aurora, Nightly builds.
