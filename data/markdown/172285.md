### Description
Increases the width of the browser-based app. 

Customization options:
<b>Total width</b>
Changes the width of the entire app. Affects the content area only
Choose between 60%, 70% or 80% of the browser window width. The default is 80%.

<b>Left Column</b> 
Changes the width of the left column
Choose between "Default" & "Wider"
