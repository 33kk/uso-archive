### Description

### Update notes
Noto Sans Light Fonts is here: http://pan.baidu.com/s/1eQAiIf4 .

<b>!important</b>
This version is for <b>fonts have bold weight</b>. If this style is not working, use https://userstyles.org/styles/115326/system-cjk-fonts-replacement instead.

Generic family-name(sans-serif, serif, monospace) is not supported, but you can set them in browser's font settings.
