### Description
Replaces with the default close.png icon of current theme the default toolbar button icon of "Back IS Close 1.4" add-on by TP Diffenbach (http://www.diffenbach.org/back-is-close.html).

For users who do want to use small icons on their toolbars...
Tested with Firefox 2 and Mostly Crystal theme only.
