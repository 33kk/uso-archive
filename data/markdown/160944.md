### Description
The best layout to workflowy for me. A mix of three other styles listed below:

WorkFlowy Total Minimalist: https://userstyles.org/styles/132659/workflowy-total-minimalist

Clears up Workflowy UI in several ways:
- makes it full-width / fluid, not page-like
- hides the top bar (mouseover to see it)
- hides a few of the promotional boxes that sometimes appear
- changes the font (customizable)
- changes the footer text (customizable)

Workflowy Dark: https://userstyles.org/styles/143680/workflowy-dark

Dark Theme for the website Workflowy.

WorkFlowy : Zen Colour Labels: https://userstyles.org/styles/120974/workflowy-zen-colour-labels

Add ( up to 140 ) coloured labels for visual enhancement of your WorkFlowy lists.

Thanks for all the authors out there!
