### Description
Erster kleiner Versuch das neue Layout des Forums von runnersworld.de ergonomisch anzupassen.
Der Inhalt des Forums wird um 160 pixel höher gesetzt, so dass die Kopfzeile verschwindet. Zusätzlich wird die Hintergrundfarbe wieder auf ein neutrales weiss geändert.
Im Rahmen des "Redesigns" des Forum von Runnersworld hat Fori Jublu den Code erweitern um die Fusszeile von Runnersworld im Forum zusätzlich verschwinden zu lassen.
Die Seite nimmt zunehmend wieder ein ansprechendes Layout an ;-)

siehe auch Anpassung von User steilkueste
