### Description
This just sets the default body background to the toolbar background image, which looks surprisingly quite good on almost all the pages I tried so far. ________________ NOTE:this very simple rule works only with themes that us a toolbar background image named toolbar-bg.png, like the Red(purple...) Cats and Curacao themes.
