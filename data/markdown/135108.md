### Description
Dark gray theme for the GMX Webmail interface.
Compatible with the default color themes.
However, the link color for each theme is too dark: I've replaced it with an orange color.
Feel free to modify it from here.

This style is used on a french GMX account (navigator-bs.gmx.fr/calendar.gmx.fr/cloud.gmx.net).

But...it should work natively with other languages: I've defined regex for url containing "-bs.gmx.", "calendar.gmx." & "cloud.gmx.".

***Apparently not working on the gmx.net domain (german/austrian/swiss websites)***
