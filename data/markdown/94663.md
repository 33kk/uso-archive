### Description
Replace the ugly Tango "triple-A" icons in Tb3 and later with the classic BIU icons from Tb2 -- for the Bold, Italic, and Underline buttons in the compose window formatting toolbar.

How to install this style in Thunderbird: click green Install button below, Switch to Edit, select all/copy code to a New Style window for Tb Stylish, and Save.
