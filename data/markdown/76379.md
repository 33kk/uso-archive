### Description
Стиль адаптирован для SPACES.RU
Изменения:
1. Абсолютная прозрачность стиля.
2. Изменен размер и цвет шрифта.
3. Есть возможность замены фонового изображения.
Для установки: 
1. Откроем оперу и зайдем на свою страницу. 
2. Кликнем правой кнопкой по странице сайта и выберем: 
"Настройки для сайта">"Вид">"Моя таблица стилей", нажимаем "Обзор", и ставим нужный нам стиль: "Blackspaces.CSS" - нажимаем 'Ок'. 
3. Теперь обновим страницу.. )) 
P.S. Скрипт нужно скопировать и сохранить с помощью блокнота выбрав расширение ( *.CSS )
