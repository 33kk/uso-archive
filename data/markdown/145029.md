### Description
Chrome拡張のStylishの最近のバージョンのアイコンをクリックすると現れるポップアップメニューが当方の環境では正しく表示されない(最小フォントサイズ指定のためかもしれません)のでそれを修正するためのスタイルを作りました。
　修正内容はポップアップ中の主な表示要素幅指定の解除と、「︙」メニューのエントリーを矩形表示にして左寄せにしただけです。
※いつからは不明ですが､本ページのインストールボタンやアップデートボタンが効かなくなっています｡スタイルを見るボタンを押してスタイルをコピーしてから「スタイルの管理」からエディターを起動して貼り付けてください｡

This style that corrects the disorder of the popup menu of recent version of Chrome extension STYLISH.
The correction content is to release the width designation of some display elements in the popup and to make the entry of the menu rectangular and left-justified.

* Installation buttons and update buttons on this page are no longer effective. Please copy the style by pressing the Show Style button, and activate the editor from "Manage Style" and paste it.
