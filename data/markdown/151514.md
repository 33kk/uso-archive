### Description
Normalizes all manga images to a certain width (can be chosen in Advanced Style Settings).
Additional features: 
* Margin between manga pages can be chosen.
* Optional zoom when hovering the mouse over an image (to read small text etc.).

### Update notes
Version 1.0.3:

Version 1.0.2:
* Options for margin between manga pages added.

Version 1.0.1:
* More width options.
* 100% width is now really 100% (removed margin).
* Now working with single page view as well.
