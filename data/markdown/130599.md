### Description
A Dark Theme for the Firefox <a href="https://addons.mozilla.org/en-US/firefox/addon/keefox/">KeeFox addon</a>. This works especially well for Firefox dark themes like <a href="https://addons.mozilla.org/en-us/firefox/addon/ft-deepdark/?src=cb-dl-users">FT DeepDark</a>. 

Note that this theme is not actively developed. I am posting this here so that other KeeFox users can have awareness and easy access to this dark theme.
