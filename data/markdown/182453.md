### Description
Apply a theme that look like the real Azul boardgame to the online Rojo game.

You can customize the font, I recommand Lora. You can <a href="https://www.fontsquirrel.com/fonts/download/lora" title="Download Lora font from fontsquirrel.com/">download it here</a> or add this code to the begining of the code : 
<pre>@import url('https://fonts.googleapis.com/css2?family=Lora:wght@500&display=swap');</pre>
