### Description
Replaces the plain, white tab icons with colored SVG icons.
"The better to see them with, my dear." 

Unless you specifically want SVG icons, you'll probably be happier with style 108170.

<em>Firefox only</em> - Chrome users, install <a href="http://userstyles.org/styles/80453/colorful-tumblr-tab-icons">this style</a> instead

### Update notes
This version sets the color with a SVG stylesheet instead of copying the color directly into the SVG image. Smoother, perhaps, but less versatile than <a href="http://userstyles.org/styles/80453/colorful-tumblr-tab-icons">the other approach</a>, and this approach <b>only works in Firefox</b>.
