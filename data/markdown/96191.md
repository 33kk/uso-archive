### Description
Tweaks fimfiction's header so as to make it scrollable.

On mobile firefox, overflow:hidden is scrollable; on dektop browsers, you'll need to set Display Scrollbar to Yes

### Update notes
Note that this changes the positioning rules for the site logo. Where it was before set to display 500PX left of center, it now displays 40px right of viewport edge
