### Description
Sets the opacity of the search bar go button to 50% when not typing into the search textbox.

Users of Searchbar Autosizer add-on by xeen (http://searchbarautosizer.mozdev.org/) who want the search bar to match the behavior of the location bar and prevent the name of the search engine to overrun the space of the go button (unlike on the contrary my style 13539 does), have to set opacity of searchbar to zero when unfocused, i.e.:

#searchbar .searchbar-box:not(focused), #searchbar .searchbar-textbox:not(focused) .search-go-button
{
  opacity: 0 !important; 
}
