### Description
The progress bar in Tab Mix Plus become animated when loading (like Mac/Ubuntu)
Firefox 3.x only

for Fission, please go style 12586 

-------------------------------------------------------
version 0.1.5:
can change the color of progress bar FROM the Options of Tab Mix Plus (by default)

-------------------------------------------------------
Tab Mix Plus setting:

Display > Tab > 
Select "Progress meter on tabs"

click "Customize Styles"
on "Progress Meter", "Background Color" MUST be selected (as default setting)
