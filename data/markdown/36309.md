### Description
This simple style shrinks the Desire2Learn (D2L) version 9 "Saved Successfully" alert to a smaller size and repositions it to be below the top row of Navigation links.

Be sure to edit the style to replace "Your Domain" with the URL of your D2L instance (for example, college.mystate.edu).
