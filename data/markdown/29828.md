### Description
For those who don't use the Chat feature of Facebook, and don't need the button and other reminders cluttering up the page, this style will simply hide anything related to Chat.


UPDATE HISTORY

2010-12-07: Facebook renamed the container for the left-hand column notice. Matched.
2010-10-17: Added hiding of the Chat login-reminder notice in the left-hand column ("You are currently offline...").
2010-05-09: The style has been updated for the most recent CSS changes, and is working as of 2010-05-09.
