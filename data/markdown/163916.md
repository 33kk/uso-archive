### Description
This aims to fix various aspects of the popular theme Carbon for Reddit (https://userstyles.org/styles/161093/carbon-for-reddit)

### Update notes
Carbon for Reddit is a great theme however it lacks support for various RES classes as well as completely removing the new Reddit chat and leaving it unthemed once put back. This theme aims to fix these shortcomings.
