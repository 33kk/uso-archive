### Description
<p>Стиль для режима на широком экране. Настраивает просмотр видео в таком размере, чтобы ролик занимал всю высоту или ширину окна. Панель поиска будет перемещена вниз.</p><p>В параметрах нужно задавать высоту, ширина будет рассчитана пропорционально.</p><p>Параметры для установки стиля под ваш размер окна будут вычислены <a target="_blank" class="stl" href="http://static.scriptomatika.ru/stylish/window-size/">на этой странице</a>. Нажмите кнопку «Установить» в табличке «Ролик <abbr title="YouTube">YT</abbr> на всё окно». Скопируйте оттуда параметры и вставьте значения в соответствующие поля под кнопкой «Advanced Style Settings».</p>
