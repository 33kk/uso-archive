### Description
This script cleans up extra space in the Halo.Bungie.Org forums by
- Hiding extra elements at the top of the page
- Widening the text body (optional)
- Forcing images to fit (optional)

While it is NO LONGER a prerequisite, It plays nice with the HBO Forum Enhancer Greasemonkey script ( http://userscripts.org/scripts/show/62016 ).
