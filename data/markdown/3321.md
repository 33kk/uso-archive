but for the toolbar which comes with the del.icio.us Bookmarks firefox add-on (https://addons.mozilla.org/en-US/firefox/addon/3615).

Mac OS X users: favicons are not displayed in the default theme so you'll need to use an alternative default theme Mac Favicon (https://addons.mozilla.org/en-US/firefox/addon/3692) which is exactly the same, but has favicons in the toolbar enabled.
