### Description
For use with Firefox theme FXChrome https://addons.mozilla.org/en-US/firefox/addon/fxchrome/ this style makes the Firefox app button look like the Google Chrome settings wrench.

Requires Firefox extension Moveable Firefox Button https://addons.mozilla.org/en-US/firefox/addon/movable-firefox-button/ to move the button to the tool bar.
