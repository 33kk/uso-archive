### Description
Cleans up articles on Wikia sites by removing the sidebar, increasing the font size, and adding some padding to articles.

Note that by removing the sidebar, it also removes the search box. If you want to re-enable the search box, try this user script (requires Greasemonkey or equivalent): http://userscripts.org/scripts/show/88663
