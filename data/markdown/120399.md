<mark>--Options--</mark>

<b>Change background (default ON)</b>
Swaps the default room background with a custom one. Chose an image from the list below.

<b>Background Image (default option #1)</b>
Replaces the background with the selected image.
You can also link or upload one. Recommended size: 1920x1080

<b>Player Options (default option #2)</b>
Reduce Opacity: lowers the opacity, shows player on mouse hover.
Reduce Opacity and Resize: same as above, player box is also smaller.
Don't Install: use default player

<b>Change background (default Ms Sans Serif)</b>
Changes the font used in the chat to the one selected. <a href="http://i.imgur.com/IiT6x4C.png">Example</a>
If the selected font isn't available, it will use the next one in the list (order: ms sans serif > segoe ui > trebuchet ms > tahoma > arial).
