### Description
A few tweaks on the #newnewtwitter twitter.com experience:

- reduced emphasis on actions (fav, rt, reply) so you don't accidentally click on them and you can read the tweets
- emphasis on username instead of full name
- removed gutters for modules
- swapped left and right columns so tweets/main column is on the left
- removed rounded corners for most things
- when loading new tweets, highlights the last new tweet with a black bottom border
- removed suggested / similar users module
- full height DM windows
