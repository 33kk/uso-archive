### Description
This is snippet of CSS styles which adds some really great blur effects to YouTube website. (Currently this theme only works with dark youtube theme, You can enable it from youtube setting)
Features: 
1. Animated drop down for search.
2. Blur effect on app bar, sidebar, drop down menu, setting menu, notification, search.
3. Animated loading progress bar on top of page.
4. Hover transitions.
5. Gradient Background.
6. Curved Icon borders.
