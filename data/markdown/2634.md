### Description
This is a user stylesheet for livedoor clip (http://clip.livedoor.com/).
You can set user comment block invisible.
If you want to hide private clipped comment block, remove comment-outed CSS.
This style apply for "clip.livedoor.com" domain, but actually useful for url like "http://clip.livedoor.com/page/[ANY_URL]".
