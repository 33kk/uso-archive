### Description
Zoom tool for the Abandonia Daily Screenshot Quiz. (Tested for W7/FF4.0.1)
Allows game screenshots to be zoomed when hovered. Also shrinks and reorganizes individual game screenshots to reduce scrolling.
NOTE: Some items will be hidden from view when applying this style. Recommended (not required) for higher screen resolutions. (1024x768 or greater)
