### Description
Utopia.lv style for smaller screen.

Features:
1. Navigation bar is narrower in both width and height.
2. No icons, no mouseovers.
3. Page width is also optimized for 800 px width.
4. Always hide chatbox.

Note: If you need chatbox, remove the last section. Some texts might not be displayed properly.

You might want to use my other style for MyMiniCity, style 5507.
