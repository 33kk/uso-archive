### Description
Recover hidden date publicate articles in downloadsource (<a href="http://downloadsource.net">downloadsource.net</a>, <a href="http://downloadsource.fr">downloadsource.fr</a>, <a href="http://downloadsource.es">downloadsource.es</a>, <a href="http://downloadsource.com.br">downloadsource.com.br</a>) / <a href="http://download.net.pl">download.net.pl</a> if possible in CSS.

And show texts: author, date of publication, "comments"* in heading acrticle.

### Update notes
<b>New option:</b>

<b>Show text</b>: Author, Publication date, "comments"* in heading acrticle.

* - comments text in heading article show only Polish version now, because this element is not translated with other template websites (You would see the text "Ilość komentarzy" instead of the language proper for the page - <i>"Number of comments" / "Nombre de commentaires"  / "Numero de comentarios / "Número de comentários"</i>).

<b>Limitations</b>:

- <b>Usercript no works in suspend tabs</b>  - limitation generated script / addons?
- comments text in heading article NO show in other languages than Polish.
