### Description
A theme for the browser version of WhatsApp, on the same design style as the Nocturnal theme from <a href="https://userstyles.org/users/268662">Mokocchi</a>, specifically a sort of 'port' from the <a href="https://userstyles.org/styles/114076/messenger-facebook-theme">Messenger theme</a>. 
Use with <a href="http://decagonal.deviantart.com/art/Nocturnal-W10-582106490">Nocturnal for Windows 10</a> and <a href="https://userstyles.org/styles/105352/moko-firefox-theme">Moko Firefox Theme</a>. Also install <a href="https://www.google.com/fonts/specimen/Open+Sans">Open Sans</a> or change it. The theme is quite easy to edit, so feel free to make changes.
