### Description
A dark theme for the Pirate Bay that allows the user to customize a link and accent color for a slightly more personalized theme. Enjoy!

Please <a href="https://forum.userstyles.org/post/discussion?Discussion/StyleID=149135">leave me feedback here</a>. It helps me know what you want and what needs to be fixed so I can make my theme better, so please give descriptive feedback. If you like my theme, I would appreciate it if you would comment on and rate it.
