### Description
Allows you to set a Specific Fixed Width for the Search Bar.  The search bar will stay at the width you specify, and will not grow, shrink, or be able to be moved physically.  Works for both Firefox 3 and Firefox 2.
Allows you to use the Most Possible area for your Location Bar or the Awesome Bar in Fx3

Instructions:
- Specify the Min-Width in Pixels -  (Smallest the SearchBar can become) - Customizable
- Specify the Max-Width in Pixels - (Largest the SearchBar can become)
- Make the two lengths above the same and the search bar will be Fixed!

Integrates with Style 7173.
Integrates with Style 8930.


Works on:
-Firefox2 (LocationBar is lengthened)
-Firefox3 (AwesomeBar is lengthened)
