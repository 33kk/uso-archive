### Description
using this style and the "Showcase" addon >
https://addons.mozilla.org/en-US/firefox/addon/showcase/?src=ss you can -
move the Sidebar horizzontally to the Bottom - 
add the gradient colors background -
add the dock -
show the tabs tumbnails with a ZoomIn effect on hovering the mini blue sidebar at center bottom and ZoomOut on focus lost -

If you don't care about the search box as I did open about:config and set the integer value
extensions.showcase.sidebarFindBarMode to 2
