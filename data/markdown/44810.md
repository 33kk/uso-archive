### Description
Users are requested to use this restartless addon instead >> http://addons.mozilla.org/en-US/firefox/addon/noverflow/

Firefox Bug 597564 (https://bugzilla.mozilla.org/show_bug.cgi?id=597564) aims to reduce min-width but may never be fixed.

Get more useful tweaks from user 43231

### Update notes
27/9/11 - Use a slimmer new tab button
16/11/11 - Added option for slimmer pinned tabs
29/11/11 - If min-width is set to 24px, hide all tab close buttons or use a wider active tab
3/1/12 - Added settings for various features, removed code to hide the gap from title bar (use style 44801 instead)
21/1/12 - Updated minwidth values in dropdown menu

Set browser.tabs.animate to false for smoother transition with too many tabs. Set browser.tabs.tabClipWidth to a low value (~60) and restart to see tab close buttons on lower widths.
