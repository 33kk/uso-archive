MBlue Nav-Buttons XP on Vista
--------------------------------------------------------------------
Description:
 Makes the Back-Forward Buttons like XP buttons, on Vista.
 Supports both small and large icons.
--------------------------------------------------------------------
Notes:
-Using this on Vista is completely useless.

-Can be used to replace the Back-Forward buttons with the XP buttons when using different Themes.

-When using THEMES/EXTENSIONS/STYLES that changes some margins, you may need to change some value manually, if so go to bottom of the page to customize.
