### Description
Простой темный стиль для сайта только на css. (для себя)
Установить в браузере firefox дополнение Stylish (скачать для firefox https://addons.mozilla.org/firefox/downloads/latest/stylish/addon-2108-latest.xpi?src=cb-dl-mostpopular ) (скачать для Opera https://addons.opera.com/extensions/download/stylish/ ) , перезагрузить браузер, открыть сайт http://rumedia.ws  клик на иконку Stylish, клик "найти стили для текущего сайта",клик ссылка "css стиль для rumedia.ws", клик кнопка, клик установить.
В Opera также.
