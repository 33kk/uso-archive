I've decided to rebuild the style 36271 style from the ground up.  Notable differences:

- Removed the sidebar completely (submit button still available!).
- Fluid width.
- Works with custom logos and some custom subreddit styles.
- A lot less clutter.
- The reddit enhancement suite's features work very well (no need for style tweaks or comment boxes though, as they're included in this style).
