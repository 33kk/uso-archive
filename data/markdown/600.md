### Description
Firefox Extension Manager - Disabled Exts Crossed Out (normal)

This style makes disabled extensions easier to identify by crossing out their icons in the extension manager.
For use with normal extension view (for Slim Mode see here: style 4261)

Note: FF2.x doesn't work for some reason unless. every time. you go into Stylish, open this style, and then "Preview" or "Save" it.
