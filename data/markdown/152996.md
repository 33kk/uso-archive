### Description
Gray color theme with improved accessibility.
<ul>
<li>Light gray background</li>
<li>Configurable story font and text width.</li>
<li>Most author formatting disabled.</li>
<li>All unreadable tiny light gray text on white background is bigger and much darker.</li>
<li>Soft borders, shadows and hover effects.</li>
<li>Background image removed.</li>
</ul>
Recommended to install <a href="https://userstyles.org/styles/153002/">SoFurry.com - Big and Fluid</a> style too, but can be used alone.<br><br>
