### Description
Animates the Statusbar when you disable the AdBlock Plus ad-filter.

Works only for Firefox 3

Features:
+ Animated background of the Status Bar when the Adblock Plus Extension's status is Disabled
+ The animation will cycle through Red, then Yellow, then Orange.  This is done in a slow manner, not to be too distracting to the user.
+ More flashy to the eyes, notifying you that your browser is at a security risk
+ Animation and background color may be customized


Applies to:
+ Adblock Plus Status Bar Icon (ABP)

Clarification:
**This style does not flash the statusbar when you disable the AdblockPlus Extension, it flashes when you disable Ad Block Plus**

Integrates with Styles:
+ Style 11505
+ Style 8297
+ Style 10319
+ Style 8770

Works on:
- Firefox3 Only.

Changelog:
1.0.0.1:
+ Added code to compress (shrink) the ABP statusbar icon when Adblocking is enabled.
+ When Adblock is disabled, the statusbar will expand and be more noticeable.
