### Description
This style, to release the fixed width in several pages of site USTREAM.TV. The browser window width that can display without a horizontal scroll bar is 800 pixels at the minimum.This is also valid in large window width.

USTREAM.TV の横幅固定を解除して伸縮表示します。当方ナローバンダーなのであまり利用しないサイトですが、たまに見るときにレイアウトが気になってイライラするのでユーザーCSSで場当たり的に調整してみました。アカウントも持っていないし僅かなページでしかためしていません。

* I'm using this style on Opera12.15+Stylish and Firefox22+Stylish
* Also use ABP and register EasyList+EasyPrivacy and ABPJapanese Filters. 
* I have not tested in not Japanese page.
* sorry, this is computer aided translation.

* Opera12.15S+tylish 及び Firefox22+Stylish で確認しました
* 同様の環境に加えて ABP を使ってそれに EasyList+EasyPrivacy 及び ABPJapanese フィルタを適用しています 
* 日本語ページ以外はテストしていません
