【特色介绍】
　　custom.css 系列是适用于中文用户的轻量级美化样式表，可用于支持 Stylish / Stylus 扩展的浏览器。本样式表使用了 Microsoft 的 Microsoft YaHei、Microsoft JhengHei、Consolas 字体家族，另有“<a href="https://userstyles.org/styles/123597" target="_blank">全局思源黑体</a>”可供选择，请根据个人喜好任选其一，不要同时使用。

　　1. 使用前请先<a href="https://n2o.io/go?page=custom.css" target="_blank">下载相关字体</a>。
　　2. 因无法及时查看本网站留言消息，问题反馈请邮件联系 francis@n2o.io，本人会第一时间回复和处理。
　　3. CSS 只能单纯隐藏广告元素，浏览器还是会在后台加载内容，如搭配广告过滤扩展 Adblock Plus 一起使用可彻底屏蔽，拥有最纯净的浏览体验。请通过浏览器的“应用商店”或“附加组件”等页面搜索并下载，安装成功后欢迎订阅本人编写的 <a href="https://n2o.io/go?page=quarklist" target="_blank">QuarkList 过滤规则</a>。
