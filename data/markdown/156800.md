### Description
<b>No Ads</b>
Удалена реклама везде, где нашёл.

<b>Playlist Fix</b>
Вариативный выбор размера всплывающего окошка по количеству плейлистов.

<i>tags: music.yandex.ru, music.ya.ru, я.музыка, яндекс.музыка, ads, реклама</i>

### Update notes
Параметр "Скрывать рекламу сдвигом" скрывает шапку и предложение купить подписку на прогресс баре размещая контент поверх них. Если часть сайта съезжает, попробуйте открыть страницу ЯМ на новой вкладке или выключите функцию.

Параметр "Скрывать рекомендацию плейлистов" убирает тизеры плейлистов справа над списком треков.

Параметр "Количество плейлистов" активирует Playlist Fix, который увеличивает количество строк во всплывающем списке для добавления нового трека при нажатии кнопки "+".

Если плейлистов >10, перенесите наиболее актуальные повыше, либо отредактируйте стиль под нужное количество: селектор div.d-addition__content, его параметр height должен быть равен (n+1)*40, где n — количество плейлистов.
