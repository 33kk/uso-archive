### Description
This style minimizes the browser UI if tabs are set on bottom. 

--

TIP: If you want to re-enable tabs on bottom (As Mozilla removed the option to set them on bottom in Firefox 15) you can do this:

1. Type about:config in the adressbar.
2. Click "I'll be careful, I promise!"
3. At the top, search for "onTop" (without the ").
4. Set the result browser.tabs.onTop to false by double-clicking it.
5. The tabs will be instantly set to bottom.
