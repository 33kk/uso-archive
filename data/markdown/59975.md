### Update notes
<ul>
<li><b>Firefox</b> (Iceweasel): усталёўваем дадатак <a href="https://addons.mozilla.org/en-US/firefox/addon/stylish/">Stylish</a>, усталёўваем нашае аздабленьне.</li>
<li><b>Chrome</b>:</li>
 <ul><li>або усталёўваем дадатак <a href="https://chrome.google.com/webstore/detail/fjnbnpbmkenffdnngjfgmeleoegfcffe">Stylish</a>, усталёўваем нашае аздабленьне,</li>
 <li>або ўсталёўваем скрыпт наўпрост: пстрыкаем па спасылцы «<i>install this style as user script</i>» (уверсе), але так ён абнаўляцца ня будзе.
 </ul>
<li><b>Opera</b>:
 <ul>
 <li>захоўваем файл па спасылцы «<i>install as plain CSS</i>» сабе, у Опэры адчыняем <a href="http://vk.com">vk.com</a>, пстрыкаем па бачыне праваю кнопкаю, абіраем «Зьмяніць перавагі пляцу…», укладку «Экран», у полі «Мая табліца стыляў» пазначаем шлях да спампаванага файла, націскаем «Добра», абнаўляем бачыну,</li>
 <li>або захоўваем сабе файл па спасылцы «<i>install as user JS</i>», у Опэры адчыняем <a href="http://vk.com">vk.com</a>, пстрыкаем па бачыне праваю кнопкаю, абіраем «Зьмяніць перавагі пляцу…», укладку «Скрыптаваньне», у полі «Карыстальніцкая тэчка JavaScript» пазначаем каталёґ спампаванага файла, націскаем «Добра», абнаўляем бачыну.</li>
 </ul>
</li>
</ul>
