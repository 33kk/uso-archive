### Description
트위터 홈페이지를 간단하고 큰 화면에 맞는 구조로 바꿉니다.
* 트위터 로고를 파비콘으로 교환하고 화면 좌측 상단에 놓음
* 상단 메뉴를 작게 바꾸고 화면에 항상 표시되도록 변경
* 트윗 버튼을 화면 상단에 항시 표시되도록 변경
* 입력 필드를 화면 상단에 항시 표시되도록 변경. 남은 글자수는 필드의 오른쪽 하단에 표시되도록 변경. (마우스 오버시 선택이 되지 않게끔 사라지게 함)
* 새 트윗 공지를 입력 필드 하단에 표시되도록 변경.
* 우측 메뉴 전체를 기본적으로 숨김. 좌측 상단의 트위터 로고에 마우스 오버할 경우 나타나도록 수정
* 남은 공간 전체를 트윗들이 쓰도록 변경.


Make Twitter use more screen asset, and simplify everything.

* Hide twitter logo & replace with favicon
* Login and its associated menu, input field, character count, and input button are all fixed at the top of the screen
* Remaining character count is moved to lower right hand corner of input field. Disappears upon mouse hover so as to get it out of the way.
* New tweet received notice shows up directly under the input field.
* Remove right menu from view altogether. It only apepars when mouse is hovered over the twitter favicon
