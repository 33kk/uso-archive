### Description
Using "-moz-box-ordinal-group" you can change the order of icons or the like. The higher the number, the more to the right. By default everything in Firefox is set to 1, I think. 

Have a look at MDC: https://developer.mozilla.org/en/CSS/-moz-box-ordinal-group
