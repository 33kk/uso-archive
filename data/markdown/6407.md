### Description
eSports.lv is the biggest internet sports media in Latvia with lots of user generated content and multimedia stuff.
This script removes all annoying flash banner spaces, preserving only Nikon "Picture of the day" spot.

Removes bright orange background from "LMT" sponsored interview pages. My eyes!!1

Makes this site browsable in multiple tabs under Linux (has slow Flash performance).

Updated for latest changes on January 2009.
