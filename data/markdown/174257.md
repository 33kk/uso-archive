### Description
A Material version of Gmail. Based on Jetbrains Material theme.
3 themes available : Oceanic, Darker, Palenight

If your eyes like it, I did the same on some other well-known websites with the same theme:
<a href="https://userstyles.org/users/763420">https://userstyles.org/users/763420</a>

### Update notes
IMPORTANT: For a better experience, you have to select "Dark theme" in Gmail Settings > Themes > Dark

Most of the code is from the great work of EvHaus (https://github.com/EvHaus/google-redesigned/)
