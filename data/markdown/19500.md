### Description
This style simulates the Stop, Reload, and Loading bar for Safari 4 which is put into location bar.

REQUIRES:
1. userChromeJS extention @ http://userchromejs.mozdev.org/ + Sub-Script/XUL Loader (for easier installation)
2. Stop-Reload-Safari4.uc.js script @ http://dl.getdropbox.com/u/428065/Stop-Reload-Safari4.uc.js
3. Fission extention @ https://addons.mozilla.org/en-US/firefox/addon/1951
4. GrApple Yummy theme @ https://addons.mozilla.org/en-US/firefox/addon/6704
5. style 18000

NOTE:
In case Stop/Reload button disappears, try to ADD THEM AGAIN in customize menu and restart Firefox.

For EVEN MORE Safari 4 look use style 18681.

This style is inspired/taken from style 2131 by user 875, style 8249 by user 4639, and style 518 by user 12.  Thanks!
