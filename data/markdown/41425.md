### Description
For Firefox 4+ and latest Firefox Beta, Aurora, and Nightly builds.

This overrides Firefox "In-Content UI"(browser UI is presented in Tab) from hiding of the Navigation Toolbar, so it can be seen when viewing the Add-ons Manager.

To override Firefox "In-Content UI" from hiding the Bookmarks Toolbar use either style:
style 59867
style 41338

Tested on Firefox 4+ and up to the latest Firefox Beta, Aurora, Nightly builds.

To customize or fix Firefox Beta, Aurora, and Nightly builds, you can also have a look at my other styles: user 61655
