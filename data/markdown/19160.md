### Description
Well, the effect is just the same as style 19134 by user 25431.
The only advantage is the code is a little bit simpler. :-)

I recommend using this with setting browser.tabs.closeWindowWithLastTab false via about:config, to avoid unexpected shutdown of Firefox.

Firefox 3.5 でタブが一つの時でも「閉じる」ボタンを表示するスタイルです。
コンセプトは上記した他のスタイルと同様ですが、コードが若干シンプルになってます。
about:config で上記設定を false にして、Firefox 本体が閉じないようにしておくことをお勧めします。
