### Description
Similar (but not identical) to the old MobyDark skin that used to be selectable from the MobyGames user preferences. Try this if you miss it, or if you just prefer light-on-dark. As of Feb 2014, the site design is in a state of change, so I hope to maintain updates accordingly.
