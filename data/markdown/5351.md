### Description
This causes that links is semi-opaque and returns to be opaque with mouse-over on these.

Compatible: Firefox 1,2 and 3.

For images only :

@namespace url(http://www.w3.org/1999/xhtml);
a[href]:visited img{ opacity:0.3 !important;}
a[href]:visited:hover img{ opacity:1.0 !important;}
