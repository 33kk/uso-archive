<p>Various improvements:</p>
<ul>
<li>Story UI: disabled pagination for text submissions. Instead all text is displayed without any scrollbars.</li>
<li>Story readability: text of text submissions is displayed on light grey background and with larger font size (customizable).</li>
<li>Caption readability: darker grey color for light grey text.</li>
<li>Compact header: small height, clean text links.</li>
<li>Compact footer: hide the useless part.</li>
<li>Pagination usability: larger buttons.</li>
</ul>
