### Description
A fix for the recent update on google docs that broke every single other dark theme here. Makes everything on google docs black and sepia.

To remove the sepia paper, edit the CSS by scrolling all the way down and changing the "BEGIN Sepia Background" section's #F3EAD9 hex code to whatever color/hex you want. White hex is #ffffff

Feel free to message me if this ever breaks again.
