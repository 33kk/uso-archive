### Update notes
New: <b>Version 1.3.1</b>

<b>Changelog:</b>
We are working to make this Style even better!

<b>New: Version 1.3.1</b>
- Improvements in Search Filters design
- Improvement in design of the "New" badge on the search page
- Bug fixes and stability improvements.

Version 1.3
- Several visual refinements
- Completely restylized playlist module
- Significant refinements in the navigation bar
- Visual improvements in 'Creator Studio'
- Visual improvements in the 'History' and 'Watch Later' menus
- Bug fixes and stability improvements.

Version 1.2.3
- New design for 'Live Now' badge in the Subscriptions tab.
- Now the video length indicator is also shown in Recommendations list.
- Bug fixes and stability improvements.

Version 1.2.2
- New design for verified commenter.
- Bug fixes and stability improvements.

Version 1.2.1
- Fixed the display colors of the notification indicator.
- Bug fixes and stability improvements.

Version 1.2.0
- Correction of a serious bug that caused, in certain regions, the disappearance of the side menu icons and counters when hover and select.
- YouTube region indicator (next to the logo) is now displayed in white color.
- Bug fixes and stability improvements.


<b>Known Issues:</b>
- The search bar looks misaligned when the YouTube display language is not English (United States). We are investigating this issue. To work around, try changing the display language to English (United States).
- If you find something wrong, leave your feedback! We'll be happy to answer you.


<b>This Style was tested in:</b> Google Chrome 50+. 
<i>It can also work in Opera.</i>

<b>Observation:</b> Works better in English/United States.

<b>Important:</b> This Style does not work properly when the "Ink for Google" extension is installed. If you are experiencing some display issues with this style, try disabling other extensions that change the appearance of YouTube.


<b>This style is based on:</b> <a href="https://userstyles.org/styles/121329/clean-youtube-2016-by-underdev">Clean YouTube 2016 by UNDERDEV</a>
