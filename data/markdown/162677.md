### Description
Clears out the cruft on thesaurus.com, light on dark. Edit the values in the `filter` rule on `html` to change the color and feel.

Not set up to work on the homepage, only individual word pages.

Note that this may stop working at any time, because the site uses programmatically-generated class names that may change.
