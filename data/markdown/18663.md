### Description
Forces all default text selections to all conform to the default background and foreground colors for selections. This intentionally does not override explicitly defined selections.
If you wish to override explicitly defined selections, replace "background: Highlight;" with "background: Highlight !important;" and "color: HighlightText;" with "color: HighlightText !important;".
