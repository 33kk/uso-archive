### Description
A niche desire, but if you'd rather use the spacebar to play/pause and arrow keys to move around, this will hide all the navigation controls from constantly showing up every 5 seconds.
Space - pause, play
F - Toggle Full Screen
left/right arrow keys to jump forward backward
