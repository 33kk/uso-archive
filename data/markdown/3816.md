### Description
Dieses CSS biete eine einfache Möglichkeit, eine Karo-Sidebar für den Firefox herzustellen. Dafür muss man einfach dieses CSS aktivieren, ein Lesezeichen auf http://www.karopapier.de/showgames.php?dranbin=1 erstellen und bei den Eigenschaften des Lesezeichens den Haken bei "Dieses Lesezeichen in der Sidebar laden" setzen. Schon hat man eine praktische, platzsparende Übersicht aller Spiele, bei denen man an der Reihe ist, in der Sidebar.

Wer die Liste lieber ohne Streckenbilder haben will, braucht nur die 8. Zeile mit einem # auszukommentieren.
