### Description
A <a href="http://ethanschoonover.com/solarized">Solarized</a> theme for the new JIRA

<p>Containing both Solarized Light and Dark, you choose which theme you would like to use. </p>

<p><strong>For best support use the Stylus plugin for <a href="https://addons.mozilla.org/en-US/firefox/addon/styl-us/">Firefox</a> or <a href="https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne?hl=en">Chrome</a>

and install straight from the Github version. This will give you the ability to change the theme on the fly
</strong></p>

<p>File issues on the <a href="http://github.com/snowe2010/jira-solarized">GitHub page</a>.</p>
