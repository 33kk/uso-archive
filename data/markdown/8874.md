### Description
Throbber displays a Spinning Animated Firefox Logo when loading a page in Firefox.
Throbber displays a static Firefox logo when static (not loading a page)

** Thanks to user 1 for similar code, idea, and release - style 469

Features:
+ Spinning Firefox Logo when throbber is active (loading)
+ Firefox Logo when throber is disabled (not active)
+ Firefox logo will spin when you hover your mouse over the throbber as well
+ Works in Firefox 3 and 2
+ Clean and very appropriate animated default image, included with style (Firefox logo Spinning)
+ Makes your browser look great, and adds that "wow" factor.

Change the throbber to your own favorite icon!
This style is fully customizable.

Integrates with the following styles:
+ Style 9729
+ Style 8635
+ Style 8770

Works on Firefox Versions:
+ Firefox3
+ Firefox2
