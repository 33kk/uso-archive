### Description
Sets https://play.rust-lang.org style to dark tones to reduce eye strain.

<strong>Optional Settings:</strong>
Page Background Color: Sets page background color
Page Color: Sets page color
Editor Background Color: Sets editor background color
Editor Color: Sets editor color
Editor Font Size: Sets font size for the editor
Editor Font Family: Sets font family for  the editor
Editor Flex Grow: Sets editor width in respect to output width.
Output Flex Grow: Sets output width in respect to editor width

Editor Flex Grow and Output Flex Grow sizes are important only when you keep the output window open all the time, otherwise just a convenience.

It overwrites monokai theme for pitch perfect background color for the editor. You can use config button on play.rust-lang.org page to change editor theme to monokai.

It wraps console output text in order to get rid of ugly scrollbars.
