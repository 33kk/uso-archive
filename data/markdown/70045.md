### Description
In Firefox, if an outer element is styled with -moz-user-select: none, its sub-elements cannot be selected, regardless of the CSS set. Using the -moz-none attribute for the -moz-user-select property, the sub-elements can be restyled to be selectable.

This particular user style will enable Firefox users to select the results displayed on Google's calculator.
