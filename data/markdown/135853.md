### Description
Hides ads, major performance improvements, responsive to fit any screen size, and much more readable. <b>[BETA]</b>

<ul>
<li>No ads, sidebars, or wiki controls</li>
<li>No rounded corners (there are so many it strains browsers)</li>
<li>No Project banners, Bulbapedia "announcements", or other garbage they like putting up</li>
<li>All tables set to "table-layout: fixed" which lets browsers lay out the page much faster</li>
</ul>
