### Description
An experimental style for hiding the address bar by default, showing only the status icon. Hovering the icon or focusing address bar will show it in full width.

### Update notes
I call this style experimental, because if you can't see the URL at all times, you are not protected against phishing. You've been warned, use it if you know what you're doing.

To be used with <a href="https://addons.mozilla.org/en-US/firefox/addon/classicthemerestorer/">CTR</a>/<a href="https://addons.mozilla.org/en-US/firefox/addon/prospector-oneliner/">OneLiner</a>/<a href="https://addons.mozilla.org/en-US/firefox/addon/line-1/">Line</a> so that you can put the address bar next to tabs.

Heavily modified after style 123572.
