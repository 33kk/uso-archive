### Description
When you subscribe to a new Ubiquity Command, the page will display a Green background.

Integrates with Style 10224

Features:
+ Appears on any Subscription page (must be a successful subscription)
+ Fully customizable style, for your own enjoyment.
+ Passport officer logo as background image top left
+ White text
+ Green Background


Text of the Command (Success):
Subscription Successful
You are now subscribing to the command feed and can use its functionality.
If you ever want to unsubscribe, you can use the Ubiquity "help" command at any time to get to the command management page, where you can see what commands you are subscribed to, and cancel any that you no longer want.
You may now close this window.

Instructions:
You may also change the background HEX color.

Browser Versions Supported:
Firefox 3+ only

** Must install the Mozilla Labs Ubiquity from their website:
http://labs.mozilla.com/2008/08/introducing-ubiquity/
