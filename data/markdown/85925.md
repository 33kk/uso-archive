### Description
<b>Developed for Chromium/Chrome only.</b>

Removes unnecessary and rarely used elements, fixes formatting quirks, shrinks text, and provides a host of other functionality to improve the School Loop UI.

Font: Enter a font name (installed on your computer) to use as the main font for all elements.
Single Column: A single column view with the sidebar moved down below the portal Calendar.
Static Header: Header (school name, "Logged in as", etc) stays at the top of the page.
