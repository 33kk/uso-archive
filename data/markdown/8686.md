### Description
This style is for extending Glasser (http://forums.mozillazine.org/viewtopic.php?t=649449) into the personal (bookmarks) toolbar. A cool custom blue semitransparent background for the links enables them to be read against dark or light backgrounds behind the transparency. For the style to work you must make one change to a Glasser file:

%appdata%\mozilla\firefox\profiles\(click yours)\extensions\glasser@sixxgate.com\chrome\content\dwmoverlay.xul

... whew!

(Note: You can get to %appdata% by pressing windows+R or going to Start:Run and typing in %appdata% and pressing return)

Change
<toolbar id="nav-bar" glassToMe="true"/>
to
<toolbar id="PersonalToolbar" glassToMe="true"/>

You have to restart Firefox to see the effect.

Note that if you are using Google Toolbar - Transparent (http://userstyles.org/styles/8685), you want to keep the change from there instead as it supersedes this one.

Thanks to 6XGate for the excellent Glasser add-on.
