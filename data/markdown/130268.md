### Description
This allows you to set Sender colors.  Both Unread & Read, with foreground/background pairs, can be set.  The inbox and thread listing will use the coloring.

### Update notes
Important disclaimer: This extension is NOT supported by IBM. Any issue must be sent to myself only (do not open a PMR).

This extension is provided AS IS.
