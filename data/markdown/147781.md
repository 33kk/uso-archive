### Description
This style is for JIRA's new theme change. Some of the pages are not styled yet. This is a work in progress. You do need a JIRA account and be able to login to use this style.

***This is a chrome style. All styles are not guaranteed to apply to any other browser. Need to at least be on chrome 33 or firefox 31 otherwise you may need to update your browser***

YOU CAN CHANGE THE ACCENT COLOR!!!. The default accent color is BLUE, but can be changed by clicking on the ADVANCE STYLE SETTINGS button.

Enjoy.

Please feel free to donate. Anything helps :)
