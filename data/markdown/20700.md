### Description
Notes textarea is resized up in edit mode.
NEW: Notes in view mode are displayed as pre-formatted text clips.
Useful for those who comment their bookmarks, use Delicious instead of Clipmarks or as a todo tracker, etc.
Does not work for Delicious Bookmarks plugin dialog windows, however.
