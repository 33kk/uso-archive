### Description
The Gmail message list when using vertical split mode is not very readable, and it takes up more space than needed.  This style change does several things to remedy that:

<ul>
<li> Compress some extra whitespace out, both vertically and horizontally. </li>
<li> Move the star to the upper left.</li>
<li> Unflatten the checkbox.</li>
<li> Embolden the title, and redden the new message authors and time.</li>
<li> Gray out the previously viewed messages even more.</li>
</ul>
