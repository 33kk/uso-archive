### Description
From Firefox 21, this style is not available.
This style displays plugin names, when you use click-to-play.
Names are displayed bottom of plugin area.

このスタイルはclick-to-playがブロックしている要素名(flash,wmpなど)を表示します。
自分がclick-to-playを使ってる時にブロックしている要素が何かわからないと困る気がしたので作りました。
いざ冷静に考えてみると別に困りません。
