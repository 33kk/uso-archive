### Description
Tired of Youtube UI lagging even on your modern PC, due to an overhaul of unnecessary eye candy?
This style will make you feel born again!

1. Removes advertisements
2. Changes several Youtube site backgrounds to clear white
- Youtube general background
- video player background (while in medium size mode)
- video description box gradient-background
3. Removes background hover effects in video suggestions list (a real performance hog those were)(view screenshot)
4. On big resolutions, video player is no more being stretched to an almost full screen size while in medium size mode
5. Removes various alert messages
6. Removes sponsored banners below video player, e.g. in Vevo-sponsored videos (view screenshot)
7. Various site-wide performance hogs are now obliterated:
- rounded corners on all site elements - disabled
- shadow effects on all site elements - disabled
- text shadow effects on all site elements - disabled
- transparency effects on all site elements - disabled
