### Description
Allows you to specify a background image that will display whenever you View the Source of a webpage.

Integrates with Style 8934.

Features:
+ Appears on all Source Viewing pages (Show Page Source) Code
+ Fully customizable style, for your own enjoyment.


Firefox Browser Versions:
+ This style works on Firefox 3

Instructions:
In the CSS, change the image I provided to the image you wish to display

Changelog:
1.0.0.1:
+ The image now stays stationary, and will appear in the middle of the screen regardless of the scrolling position.
