### Description
Classic forum style for Stylish plugin users or for browsers supporting user script or user JS.
-> Stylish plugin available for firefox/chrome only
-> Browsers supporting user script: chrome, internet exploder with IE7Pro, Safari with greasekit
-> Browsers supporting user JS: opera.
-> Style code may work also with other browsers/plugins, you can copy paste it from "show code"-button.

Install manually or using the link below. More information and help from <a href="https://forums.eveonline.com/default.aspx?g=posts&m=24709#post24709">official thread</a> at eve online forums.
