### Description
<a href="https://mstdn.kemono-friends.info/">ますとどんちほー</a>のために作ったテーマです．<br>
mstdn.jpなどでも使えるはずです．<br>
<b>基本機能</b>
<ul>
<li>ウィンドウサイズに合わせてカラムの大きさが変わるようにします．</li>
<li>画像を常に広く表示します．</li>
</ul>
<b>追加機能</b>
<ul>
<li>CW投稿を常に開きます．</li>
<li>スクロールバーを隠します(Win10のFirefox, GoogleChromeで確認)．</li>
<li>非収載・非公開投稿を見やすくします．</li>
<li>ログアウトボタンを隠します．</li>
<li>トゥートウィンドウを自動的に隠します．</li>
<li>トゥートウィンドウを右側へ移動します．</li>
<li>ウィンドウサイズが960px以下の場合コンパクトなデザインになります．</li>
<li>リアクションを自動的に隠して表示領域を広げます．</li>
</ul>
