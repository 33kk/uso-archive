### Description
DE:
Da ComputerBild.de eine deutsche Seite kann die Beschreibung ja ruhig deutsch sein ;D
Naja, zum Userstyle:
Die Artikel haben abgerundete Ecken und einen grauen Hintergrund,
durch die Farbdurchstufung erkennt man gut, was zusammengehört, und was nicht.
Die Seite nimmt nun die ganze Bildschirmbreite ein, anstelle eines "schmalen" 800px breiten Streifens
Es sieht allgemein "schöner", abgerundeter und übersichtlicher aus.
Dadurch, das die Texte mehr in die Breite als in die Höhe dargestellt werden, muss auch weniger gescrollt werden ;)

Hinweis: dieser Style hat keine Auswürkung auf das Forum (forum.computerbild.de)!

EN:
A new Design for computerbild.de
I let the articles look like boxes, which I rounded up and put them gray.
Because of the color-leveling you get a better overview of which things meant to be together, and which are different.
Because the hole width of the screen is now used, you also have to scoll less, because the text uses more width than height ;)

Notice: this Style don't effect the forum (forum.computerbild.de) in any way!
