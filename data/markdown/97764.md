### Description
A nicer color palette for nick colouring that fits darker themes. Based on http://clrs.cc/
Can be used with style 97736
Includes option to colourize memberlist.

### Update notes
<strong>Colour memberlist:</strong>
no: default behaviour, only colours nicknames in memberlist on hover.
yes: instantly colour all nicknames in memberlist regardless of hover.
