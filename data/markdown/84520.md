### Description
Стиль решает проблему отображения полос прокрутки при большом количестве открытых вкладок.

### Update notes
<b>v. 1.0 (15.03.2012)</b>
Первая версия. Работает на всех текущих релизах Firefox (4-22a1).

<b>v. 1.1 (18.03.2012)</b>
По просьбе сообщества убрана опция «<i>max-width</i>».

<b>v. 1.2b4 (29.03.2012)</b>
Сейчас в разработке: расширение вкладки при наведении.
Фикс: при наведении курсора на вкладку не отображалась кнопка закрытия вкладки.
Фикс: использован код пользователя @freecyber. Спасибо!
Таскать вкладки стало удобнее.
