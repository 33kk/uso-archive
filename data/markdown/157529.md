### Description
Flat theme for Godvillegame with smooth animations (light).

* Smooth popup animation is available for the third eye, lab, last voices, ark and battle history.
* All main pages of the site are supported except encyclopedias, including the main, forum, etc.
* Don't conflict with Erinome Godville UI+.

Work in progress, message me if you find errors in the display.


Плоская тема для Годвилля с плавными анимациями (светлая).

* Плавная анимация появления попапов доступна для третьего глаза, лаборатории, последних гласов, ковчега и истории сражений.
* Поддерживаются все основные страницы сайта кроме энциклобогии, в том числе главная, форум и пр.
* Не конфликтует с Erinome Godville UI+.

Работа над темой продолжается, пишите, если найдёте ошибки в отображении.
