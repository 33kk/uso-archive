### Description
Similar to other userstyles, this outlines rel="nofollow"'d links, however this also outlines all immediate children of the links. This is for the case when the a tag wraps a block level element and the outline is not shown, for example:

&lt;a href="http://example.com" rel="nofollow">
  &lt;h4>My header&lt;/h4>
&lt;/a>
