### Description
A Firefox userstyle that makes minor theme changes throughout the UI using the Mozilla's Australis design guidelines for consistency. This is a fork of Blinky's "<i>Firefox Australis Windows 8 Appearance</i>" (https://userstyles.org/styles/96014/firefox-australis-windows-8-appearance). All credit goes to him first!

This theme is only optimized for Windows, specifically the Developer Edition build (DARK ONLY). 
Restart the browser to see complete changes.
