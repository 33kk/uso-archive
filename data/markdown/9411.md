### Description
Animates the Notification Bar when it appears (asking you to save a password, or telling you to block pop-ups)

Works only for Firefox 3

* Similar to style 5283 by user 6147

Features:
+ Animated background of the Notification Bar
+ The animation will cycle through Red, then Yellow, then Orange.  This is done in a slow manner, not to be too distracting to the user.
+ More flashy to the eyes, notifying you that your browser is asking you for something, or letting you know something important
+ Animation and background color may be customized


Applies to:
+ Notification Bar
+ Remember Passwords Prompt at top
+ Popups blocked (blocker)
+ Extension requiring restart
+ New Add-ons message
+ Addons compatibility warning popup
