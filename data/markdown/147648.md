### Description
Use with addon <a href="https://addons.mozilla.org/de/firefox/addon/new-tab-override/">new tab override</a> to choose which site to open on every new tab (like www.google.com).
My own customizable theme.
Allows you to change colors and background easily.
Don't hesitate to send a FeedBack.

### Update notes
If you want to use a locally-stored image as background :
1. Use "Browse..."
OR
1. Select your image
2. Drag-and-drop the image to the browser
3. Copy its URL from the address bar
4. Paste that URL in the custom background URL
