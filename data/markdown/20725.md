### Description
What this script does:

General changes:
- makes the font size of the main text larger
- gives visited links a different colour (the same as the hover colour for the main page, a light grey for the drop-down menus)

Profile page changes:
- removes the coloured bar under the username
- removes the dotted lines below the section headers
- adds extra left- and righthand margins to each section body
- removes the unnecessary text from the tool/action bar
- moves the default usericon to the right side of the text
- makes the font size of the title and subtitle a bit larger
- makes the font size of any adult warning the same as the stats and removes the bolding
- makes the font size of the stats a bit larger and the font colour black
