### Description
<b>Description</b>

A css file for Stylus to make Ykt.ru services dark and friendly for your eyes. This project is a dark version of <a href="https://ykt.ru/">YKT</a> website.

<b>How to contribute</b>  

Clone this project on <a href="https://github.com/Kenya-West/YKT-dark">GitHub</a> and start your changes. If you want to use styles from source code, then create new style in Stylus extension and put source code in opened window.

<b>How to use this style?</b>

You need <a href="https://github.com/openstyles/stylus#releases">Stylus</a> extension installed on your browser to use this style.
