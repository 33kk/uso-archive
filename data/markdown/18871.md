### Description
1. Set text columns' maximum width to 30em.
2. Set font family in <code> and <pre> (usually code samples) from Courier to Consolas or Courier New.
3. Set <pre> background from gray to transparent. Sun marks up code samples in different ways, so it doesn't work in all pages. 

Tested 24-June-2009.
