### Description
Make workflowy tasks into an active Calendar. 

this includes the following custom tags:
 (copy and paste this list into workflowy to get started ) 

#ThisWeek
&nbsp; #Monday 
&nbsp; #Tuesday
&nbsp; #Wednesday
&nbsp; #Thursday
&nbsp;  #Friday
&nbsp;  #Saturday
&nbsp;  #Sunday

#NextWeek 
#Now
#Next

Each day is styled to be headings of lists: Bold, italic, and dark grey. so you can create a list underneath for each of the days tasks. (screenshot1) 
you can use this to create a list for each day that is searchable by day and... 
Since they are searchable, you can also use the tags outside of your calendar list, to quickly move project tasks to your calendar. (screenshot 2)


*note this install also removes underline of tags
