### Description
Mandarin Popup is a great little light-weight Chinese popup translator that uses the newer CC-CEDICT as compared to others which use Google Translate or CEDICT.
https://addons.mozilla.org/en-US/firefox/addon/9931


The purpose of this userstyle is to replace the ginormous, statusbar-hogging "Mandarin Popup: Enabled/Disabled" text, and replace it with a simple on/off icon, to better compliment this great little Firefox extension.
