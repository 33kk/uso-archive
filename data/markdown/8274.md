### Description
Allows you to Customize the text and background image of the Extensions List's Newly Updated extensions.  This page appears when there is a new update to one of your extensions.  It displays the Change log of the extension, along with details of what is new.

To get to the changelog, open up your Extensions list, then click the Updates Tab, then at the bottom click the Show Information Button.
	

Features:
+ Background image will be a faded Firefox logo with shadowing
+ Text size of the update details will be larger (14pt instead of 12)
+ Optional Bold text for easier readability
+ Supports a huge list of Fonts (discussed below)
+ Background image is static and is centered on the window.

Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font


Applies to:
+ Extensions List
+ Addons List
+ Themes Page
+ New Add-ons available window

Browser Versions:
- Works only on Firefox 3

Additional Information:
+ Feel free to edit the CSS.  You can customize the fonts, enable bolding, and change the background image

Integrates with Style 8028
