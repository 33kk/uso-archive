### Description
el gran problema con muchas paginas web, es el excesivo brillo que tienen. los cuales son demasiado agotadores para la vista.

estilos como este solo buscan reducir la carga de brillo en la vista, para tener una mejor experiencia de usuario con el sitio.

esperemos que los desarrolladores del sitio se den cuenta de estas falencias y generen alguna version "Dark-Mode" para sus usuarios. a estas alturas ya debiese ser un estandar el desarrollo y publicacion de estos estilos dark, a modo de respeto para la vista de sus usuarios.
