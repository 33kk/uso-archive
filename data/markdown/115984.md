### Description
This style is a darker version of my "Editing Global Black for AMOLED - Red theme" style which is a fork of "<a href=https://userstyles.org/styles/112107/global-black-for-amoled-displays>Global Black for AMOLED displays</a>" created by user 288422. It is applied to all webpages and it's purpose is the same as the original style: save battery on AMOLED displays and/or reduce eye strain in low light.
