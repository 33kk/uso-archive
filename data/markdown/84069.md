### Description
An add-on Icon-package for MuroBBS (2013) Web 2.0 Theme by Lifeless. 

Brings back the nice Icons from earlier MuroBBS (2013) Web 2.0 Theme versions. I have decide to split these in separate style sheet for easier management. Also, this allows alternative Icon sets to be used with the MuroBBS (2013) Web 2.0 Theme. 

The MuroBBS (2013) Web 2.0 Theme can be installed from:
http://userstyles.org/styles/18371/

(This add-on Icon-package can be used with other theme's also but cannot provide support for them.)
