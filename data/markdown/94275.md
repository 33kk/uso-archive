### Description
DuckDuckGo, version épurée, aux couleurs d'Elementary OS.
Basé sur le travail de DanRabbit : http://danrabbit.deviantart.com/art/DuckDuckGo-385296599

Pour une expérience optimale, vous pouvez paramétrer DuckDuckGo comme ceci :
Results :
-0click box : off
-Site icons : off
Color :
-all to "default"
Look &_feel :
-width : wide
Layout :
-Ads : off (if you want)
