### Description
Description:
 Transforms "some" Toolbar Buttons into Tabs-like Buttons if
 moved inside of the Menubar.
 Integrates better with the Simple-Gray Theme.
--------------------------------------------------------------------
What it does:
 - Create tabbed-like buttons for some specific extension icons
 - Create a tabbed-like look for the Firefox menubar
 - Create a tabbed-like look for the "Foxclocks" estension
 - Create a tabbed-like look for the "ForecastFox" estension
 - Fix the "Video Download Helper" icon that moves when active
 - Create a button for "Piclens" that change colors when active
 - Create a button for "ABP" that change color when state changes
 - Create a button for "stylish"
 - Create a button for the Throbber
 - Nav-buttons aligned to other buttons when inside the menubar*
--------------------------------------------------------------------
Supported Extensions:
 Foxclocks, Video Download Helper, Piclens, ADBlockPlus, Stylish.

 Also supports standard icons (Bookmarks, History, Addons,
 Downloads) and some Ext. (Tab Sidebar, TagSifter, QuickNote), but
 without any special effect.
