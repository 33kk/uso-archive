### Description
<ul>
<li>Arrange photos in Instagram feed in a grid; </li>
<li>Fit more photos in a row on user profile page (and make profile page wider); </li>
<li><i>(Optionally)</i> Hide #tags and/or comments in a feed; </li>
<li><i>(Optionally)</i> Remove spacing between photos on a profile page. </li>
</ul>
