### Description
A style that let's you completely control the look and feel of the appmenu button in Firefox 4+.

### Update notes
-Don't like the colors given? Simply replace the xxxxxx on the line "background-color: #xxxxxx !important;" with your color (note: this line only shows up if you pick an option OTHER than transparent)
-To use Terminus as the font, you need to download it from http://fractal.csie.org/~eric/wiki/Terminus_font
-To use the Ubuntu font you need to download the font from http://font.ubuntu.com/

Tested on Windows 7/Vista with Firefox 4.0+
