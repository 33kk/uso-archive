### Description
twitaddons.com에서 사용성을 높였습니다. 기존의 레이아웃에서 다음의 사항을 변경합니다.
(나눔고딕 폰트 기준으로 제작되었습니다 - 다른 폰트를 쓰시는 분들은 스타일을 살짝 조정하셔야 할지도 모릅니다)

* 출력영역의 너비를 더 넓힘
* Twitaddons 로고 숨김
* 공지사항 숨김 (Update 4/30 - 볼 수 있는 옵션 추가)
* 광고 숨김
* 메뉴와 로그인/로그아웃 버튼을 화면 상단에 항시 표시되도록 변경 (스크롤과 상관없이 위치유지)
* 입력필드를 메뉴 바로 아래에 항시 표시되도록 변경 (스크롤과 상관없이 위치유지)
* 트윗 업데이트 버튼을 입력 필드 바로 위에 위치하도록 변경 (스크롤과 상관없이 위치유지)
* 가입 당 리스트를 화면 우측 하단에 위치하도록 변경 (스크롤과 상관없이 위치유지)
* 새 트윗 경고필드를 입력필드 바로 상단에 위치하도록 변경 (입력 버튼을 가리게 됨)

하단의 "@-moz-document url-prefix("http://twitaddons.com/group_follow/multitag.php") {" 부터는 

javascript:document.write('<HTML><HEAD><TITLE>멀티 트위터</TITLE></HEAD><FRAMESET%20COLS=\'50%,50%\'><FRAME%20SRC=[URL1]><FRAME%20SRC=[URL2]></FRAMESET></HTML>');%20document.close()

이 북마크릿 사용이 전제됩니다 주1). 화면을 2개의 프레임으로 나눠서 별개의 Twitaddons.com 그룹을 띄워 놓을 시 필요없는 공간을 차지하는 우측 메뉴들과 상단 메뉴의 대부분을 화면에서 없애는 스타일이며, 필요치 않는 경우, 삭제해 주시면 됩니다.

주1) [URL1], [URL2] 을 원하는 URL로 바꿔서 사용할것.

Improves user experience at Korean twitter group site: twitaddons.com. Changes the following:

* Widens the total display area
* Hides logo
* Hides notification area (Update: 4/30 - Added option to view them)
* Hides advertisement
* Moves navigation menu and login area to the top of the screen (persistent)
* Moves text input area to directly under the navigation (persistent)
* Moves input button to directly above the text input area (persistent)
* Moves the joined group list to the bottom right of the screen (persistent)
* Moves the new twit notification to just above the text input area (covers input button)
Update: 2010.04.29
* Added support for group menu function that was introduced. Menu positioned next to group listing
* Group menu has 80% transparency to allow users to read tweets on the bottom. It turns to 0% upon mouse hover.
* Moves group menu pulldown menu to top, just to the left of the navigation menu.
* Slightly modified the size and location of the group list and group menu list to better fit into the layout.
* Changed the color scheme to grayscale.
