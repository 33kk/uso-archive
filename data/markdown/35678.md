### Description
Aggressively hides Gmail's UI and chrome, leaving a simplified view of your email, labels, and search box. Inspired by Matt Constantine's <a href="http://mattconstantine.com/mg">Minimalist Gmail browser extension</a>.

This style is compatible with (but doesn't hide) Gmail's <a href="http://gmailblog.blogspot.com/2013/05/a-new-inbox-that-puts-you-back-in.html">new tabbed inbox</a>.

Tips:

- Use <a href="http://support.google.com/mail/bin/answer.py?hl=en&answer=6594">keyboard shortcuts</a>! This userstyle expects that you do; it's pretty much unusable otherwise.

- You can hide individual labels in the <a href="https://mail.google.com/mail/u/0/#settings/labels">Gmail settings Labels tab</a>.
