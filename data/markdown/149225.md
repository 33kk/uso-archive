### Description
The new photon style has had mixed reactions, unfortunately Mozilla decided to end XUL-extensions so we don't have CTR anymore.
This skin aims to enable the most important classic theme features in the new photon era of Firefox.

IMPORTANT!
- Click on Advanced Style Settings Button to customize your theme
-Copy the code into the userChrome file.
