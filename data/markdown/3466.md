### Description
Pink Style for Google's web search page.

******************************************************************************************
!!! IMPORTANT !!!: If you use Google in another country other than the United States, which uses "google.com", then you'll need to make a small change to the code in order for this style to work for your country's Google page.

1. Copy and Paste the code into a simple text editor (like Notepad on Windows or BBedit on Mac)
2. Use the "Search and Replace" feature of your text editor to find "google.com" and replace it with "google.co.uk" for example if you live in the UK. Most countries should work.
3. Paste the code back into Stylish and save it.
