### Description
<p>This will disable the nearly daily distracting logo changes. Style will replace doodles with the perfect standard logo.</p>

<strong style="font-weight:600">Chrome Notice:</strong>

Chrome 61 no longer allows extensions to effect the new tab page.

However, you may use the an extension to regain styling control over the new tab page. Simply install new tab replacement extension and set the replacement url to:
<a href="https://www.google.com/_/chrome/newtab?ie=UTF-8">https://www.google.com/_/chrome/newtab?ie=UTF-8</a>

A few possible extensions:
<a href="https://chrome.google.com/webstore/detail/new-tab-redirect/icpgjfneehieebagbmdbhnlpiopdcmna">New Tab Redirect</a>
<a href="https://chrome.google.com/webstore/detail/replace-new-tab-page/cnkhddihkmmiiclaipbaaelfojkmlkja">Replace New Tab Page</a>
