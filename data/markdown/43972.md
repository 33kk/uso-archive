### Description
Compacts Brief's default feedview further with Headlines View on.  This style does not work without the Brief extension. 

See Brief's website for more about Brief: http://brief.mozdev.org/

NOTE >> With the release of Brief 1.6 final and it's improved Headlines View style, I no longer intend to update Compact Brief for the considerable future.
