### Description
For Firefox 4+ and latest Firefox Beta, Aurora, and Nightly builds.

This style removes the "(Untitled)" or "(New Tab)" text and Tab favicon on Blank tabs or New Tabs.

Works well with userstyle:
style 43992

If you use a non-English version of Firefox, you must replace "(Untitled)"/"New Tab" (without " ") in the code with whatever reads in your blank tab (Sans titre, Nova aba, etc).

Tested on Firefox 4+ and up to latest Firefox Beta, Aurora, Nightly builds.

Modified code from Userstyles:

style 13323
