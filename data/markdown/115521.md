### Description
Tries to undo the Beta changes to Slashdot.

With the failure of Slashdot Beta, Dice stated they would leave Slashdot alone, only making incremental changes "where they make sense".  What this really means is that they're slowly rolling up Beta changes into the main site.  This style tries to undo or at least make these changes palatable.
