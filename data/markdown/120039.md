### Description
Alternate color themes for KickassTorrents mirrors.
Customizable settings. Default options as follows:
"Beige" for background, search box and comments.
"Bisque" for odd rows and "Wheat" for mouse over.

### Update notes
Default theme: Warm background.
Use "OldLace" as background for a subtler effect.
