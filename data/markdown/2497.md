### Description
Turn the forums blue. Default font set to Verdana, but you can easily change that if you so wish. Just copy the code, paste it into Notepad, then press Ctrl+H. Type "verdana" (without the " s) in the Find field and the name of your chosen font in the Replace field. Then click Replace All.

Right-click and View Image to see full size screen shot.
