### Description
This is a style for blogger.com/blogspot.com blogs in Fedora night colors, intended for faster reading:
It hides the (slow!) sidebar widgets (except mini-chat) and displays about 95% of the pages correctly.
Don't use it however if you like the bloggers sidebar widgets.
