### Description
This userstyle makes the Personas Extension look even better by making all of the Navigation Toolbar items transparent, so you can see the Personas background image even better.

* Thanks to user 6431 for introducing me into the realm of Transparency with style 6771


Your personalized Personas background will stand out even better.



Features:
+ Background of all toolbars will be invisible (see through - transparent)
+ Background will return to opaque when the mouse is hovered over any of the buttons (will be transparent until mouse cursor's hover event occurs)
+ You can customize the level of transparency in the code


Applies to:
+ Navigation Toolbar
+ Bookmarks Toolbar
+ ... Any other toolbars!!


Browser Versions:
+ Works on Firefox 3+
+ Make sure to install the Mozilla Personas Extension here: 
  http://www.getpersonas.com

Integrates with the following Styles:
+ Style 18337
+ Style 8941
+ Style 9657
