### Description
Same as Trello Slim except with labels that have text.
Trello Slim features: Much more compact style, for the Product Manager that can never have enough pixels.

* Reduce the spacing all around
* Tools on each card are smaller
* Labels as dots, instead of stripes
* Smaller owner icons.
* Uses a small Tahoma font size.
* More readable card descriptions.
