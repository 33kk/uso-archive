### Description
Тёмное оформление для форума 4pda.ru.
По умолчанию цвет основных элементов — белый, для изменения цвета под свой вкус вам надо дополнительно поставить <a href="/styles/165009/doc-darker-4pda-color-overrides">DoC: Darker 4PDA - Color overrides</a>.
Вы, конечно, можете изменить цвета в этом файле стилей, однако в данном случае будут проблемы с обновлениями, когда/если они выйдут, потому и было решено разделить стили, чтобы каждый пользователь мог настроить цветовую гамму под себя.
