### Description
You can choose which color telegram web chat will look like and now the webpage adapts to the entire width of the screen available.

### Update notes
To change the color type the following values for:

Cyan:   0 or 360 (default look)
Purple: 90
Red:    180
Green:  270

Any value between will be a different color eg. between red and green values may result in YELLOW!
