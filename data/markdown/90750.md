### Description
Turn off the light! This is a dark theme for Duolingo.com. This is a customizable version of <a href="http://userstyles.org/styles/90721/duolingo-dark-2">Duolingo Dark 2</a>.

You can use the text boxes below to customize the background and text color.

### Update notes
<strong>Using Style Settings</strong>

Some versions of Stylish can now update userstyles that have style settings. If the stylish you have doesn't support updating style settings, you will need to uninstall the original style from Stylish and reinstall. If you do not mind the default color setting and would prefer the ability to update conveniently, then use <a href="http://userstyles.org/styles/90721/duolingo-dark-2">Regular Duolingo Dark 2</a>.
