### Description
This style adds numbers beside Google search results with Web search, Book search, and Scholar. This style is inspired by the great style "Google result number" http://userstyles.org/styles/3825 . Numbers added beside search results are linked when using "Google result number," but not when using this style "Google Result Number (Multi-service)." Also see the style "Search Engine Result Number" http://userstyles.org/styles/29085 . 

Google のウェブ検索、ブック検索、スカラーの検索結果の横に番号を振ります。「Google result number」 http://userstyles.org/styles/3825 というすばらしいスタイルを基にしました。「Google result number」では検索結果の横の番号にもリンクが張られますが、この「Google Result Number (Multi-service)」では張られません。「Search Engine Result Number」 http://userstyles.org/styles/29085 もご覧ください。
