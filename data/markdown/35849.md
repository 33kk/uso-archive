### Description
When a Tab is selected it displays the favicon of the website along with the website title. The Inactive tabs are turned into favicon only tabs, just like apptabs Should Work .

This Style works with the Minimal Theme for firefox( http://boneyardbrew.deviantart.com/art/Minimal-Alpha-1-Firefox-4-172481919 )
This Style has been tried on Firefox 4 betas with success on windows 7 and Vista.

Fixed The Close buttons: Now they appear only on the active tab and whenever it is hovered(updated-23-August, 2010)
