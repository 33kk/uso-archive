### Description
When print Google+ Article, there are too wide margins.
This stylesheet will reduce such wide margins.

You can adjust article width and left margin when install this stylesheet or edit it after install. It might be help you with zoom factor on printing.

NOTE:
This stylesheet was designed for NEW Google+ UI only.
