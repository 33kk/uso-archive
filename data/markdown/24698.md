### Description
This theme cleans up the glass look and extends it to all the tool bars native to FF. It also skins the Stumbleupon toolbar as well. It repositions elements from the AiOS sidebar as well as removes the gray borders that appear for a cleaner look.

Note the before and after images show the appearance with the AiOS Sidebar both open and closed.

To be used with 'NEW Glasser by SzymekPL' 
tested in WIN 7 64-Bit and 32-Bit

Installation note: Sometimes a restart after install is needed
