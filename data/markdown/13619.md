### Description
Стиль ориентирован на незарегистрированных пользователей, у которых основной акцент идет на контент и комментарии.

Специально оставлял комментарии в коде CSS для того, чтобы вы смогли переработать стиль по вашему вкусу.
Смотрим скриншоты для сравнения. Основные модификации:
- боковая панель теперь внизу
- контент растянут на 100% по ширине
- убраны некоторые элементы, которые безразличны для незарегистрированных пользователей

P.S.: Бог храни хабру с его грамотными разработчиками (layout страниц просто прекрасен)!
