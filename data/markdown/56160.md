### Description
Twitterからトレンド、広告、おすすめユーザー、プロモートツイートを非表示にするためのスクリプト。
日本語のハッシュタグきもすぎ＼(^o^)／

===================================

This style remove User Recomendations, Trends and PROMOTED TWEET on Twitter.
And also can remove Ads in any languages.
