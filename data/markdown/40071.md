### Description
Removes elements which I find unnecessary on Facebook webpages.

Elements removed include:
* Friend finder and friend suggestions (shown in right column)
* Friend photo albums (right column)
* Friends left column
* Right column of suggestions on friend list page.
* Advertisements (right column)
* Friend updates ticker (right column)
