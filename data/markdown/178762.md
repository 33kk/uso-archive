### Description
This style is designed to work on version 3.6.1 of the EcoleDirecte site, the style may no longer work correctly following updates of it.

The source code is available on  <a href="https://github.com/CustomStyles/Dark-EcoleDirecte">Github</a>. You are free to contribute or report any problems with the style.
