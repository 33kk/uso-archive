### Description
I like to keep stackoverflow etc in a smaller window on my screen but AskUbuntu.com (one of the latest StackExchange spinoffs) eats up far too much vertical space.  So this style attempts to restore some vertical space by reducing the top header, removing the system message and reducing the height of each question.
