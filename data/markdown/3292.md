### Description
Removes the Search and Go buttons while keeping the new Favorites icon intact.

NOTE: If you're running Cutemenus - Crystal SVG you have to open the Add-ons right click on Cutemenus - Crystal SVG > Options > Misc Settings (tab) and uncheck "Remove the 'Go' Button on the Address Bar" and "Remove the 'Go' Button on the Search Bar" otherwise this style will not work since the extension overwrites this style.
