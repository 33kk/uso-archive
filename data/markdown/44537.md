### Description

Descriptions
============

This style combines the tab close button into favicon; the close button only shows when *hovering the favicon*
To me it seems like the proper way (against another approach where close buttons are shown upon *hovering the tab*)

Tested on Mac latest Nightly, not sure if the metrics fit Windows theme. Enjoy ;)
