### Description
Makes the Search Bar Transparent until you hover your mouse over it. 

* Thanks to user 6431 for introducing me into the realm of Transparency with style 6771

Features:
+ Search Bar text is Transparent until it is hovered over with the mouse - or focused with key combination
+ SerarchBar icon will remain visible
+ Previous Search Text will remain, so will the search provider name.

Works on Firefox 3 / FF3 AND Firefox2
Gives more Focus to Firefox 3's new Awesome Bar.

Integrates with the following styles:
+ Style 9362
- Style 7173
- Style 7723
- Style 7956


Works on:
- Firefox3 (Search Bar is Transparent)
- Firefox2 (Search Bar is Transparent)
