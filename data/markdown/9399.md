### Description
Makes the Tab Scope popup preview Transparent until you hover your mouse over it. 

* Thanks to user 6431 for introducing me into the realm of Transparency with style 6771

Features:
+ Tab Scope previews are Transparent until hovered over with the mouse
+ Transparency can be customized to allow more or less opacity through.

Works on Firefox 3 only

Integrates with the following styles:
Style 9366

Works on:
- Firefox3 (Tab Preview is see-thru)

Changelog:
1.0.0.1
+ Updated code to work on Firefox 3.1 (Shiretoko) - No longer uses -moz-opacity, it is now just opacity.
