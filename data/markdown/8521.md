### Description
Allows you to Customize the selector background image of the Extensions Window.
What this means is you can change what the extensions list selection highligher looks like when you click on the different themes, plugins, or extensions.


* Also applies to the Currently Selected Item on Firefox 3's AwesomeBar

Features:
+ Background image will be a faded Black Firefox logo
+ Bold text for easier readability (Of selected item only)
+ White Text
+ Supports a huge list of Fonts (discussed below)
+ Background image is static and is centered on the window.
= Optional - Text size of the update details will be larger

Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font


Themed:
+ This Style has been defaulted to integrate with dark themes (NASA Night Launch for example).  This can be easily changed.

Applies to:
+ Extensions List
+ Addons List
+ Themes Page
+ Plugins Tab
+ New extensions update tab

Browser Versions:
+ Works on Firefox 3
+ Works on Firefox 2

Additional Information:
+ Feel free to edit the CSS.  You can customize the fonts, change font sizes, and change the background image.  *No Re-Posting without my approval please!
