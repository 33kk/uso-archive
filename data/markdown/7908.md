### Description
Removes the Status Bar from the "Organize Bookmarks" page.  This is a useless/statusbar, hiding it gives you a full-screen kind of view.
Increases editable screen size, allowing you to view and organize more bookmarks.
This style supports Only Firefox2.  (There is not a statusbar on Fx3, by default)

Applies to:
- Organize Bookmarks Window (Main Menu / Bookmarks / Organize Bookmarks)
- No more statusbar, giving you more screen real-estate for your bookmarks


Browser Versions:
- Works only on Firefox 2

Integrates with Style 7901
