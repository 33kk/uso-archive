### Description
Здарова я Рыж. Данная тема работает только по адресу игровой, т.е. только в игровой. Дизайн специально для пододеяльщиков.

Изменения:
- Общий тон игровой - чёрный;
- Убраны некоторые интуитивно понятные заголовки;
- Большинство полей и кнопок заимели скруглённые углы;
- Эффект нажатия на кнопку сопровождается высветлением кнопки;
- Выравнено положение температуры;
- Игровая больше не будет темнеть во имя изменения времени суток и всегда будет светлой;
- ЛС|Чат|Домашняя страница и ТБ выравнены по центру.
