### Description
Moves the side-bar navigation menu into the header and expands the stream to full width. Designed to work with Diaspora multi-column ( https://userstyles.org/styles/165257/diaspora-multi-column ) and similar styles.

Responsive support for both a horizontal menu and dropdown depending on available screen width.
