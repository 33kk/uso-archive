### Description
------------------------------- 2019/09/23 -------------------------------
Windows 10 + Chrome
1. 網址列輸入：「chrome://flags/」
2. 找到「Match @font-face { src: local(<name>) } names by PostScript and full font name.」
3. 改為「Disabled」

Firefox
1. 網址列輸入：「about:config」
2. 找到「security.fileuri.strict_origin_policy」
3. 值改為「False」
------------------------------- 2019/09/18 -------------------------------
Windows 10下Chrome使用@font-face的方式，使用非系統內建的字型，
不知為何已被Google封鎖，已無法使用，請自行將字型改為"微軟正黑體"等內建字型即可
已無法使用@font-face的方式使用自行新增的字型...@@
-------------------------------------------------------------------------------------------------
下載Advanced Font Settings後，將字型設成與Stylish裡面設定一致的字型。
詳細說明可參考 https://goo.gl/XjefX2
使用Advanced Font Settings這個外掛大部分的網站仍會使用自己的字型，
所以還是需要加入以下CSS，目前測試結果，幾乎都可以完美把字型改掉。

58版後的Chrome套用字型名稱若有空白，部份字型會無法取代，如：Noto Sans CJK TC Medium
可自行改為"微軟正黑體"，或其他無空白名稱的字型，或下載下面的字型。
https://goo.gl/9hxkJh，移除Noto Sans CJK TC Medium名稱空白的部份。
