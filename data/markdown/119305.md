### Description
I created this out of curiosity, one of the main reason is I use Chrome a lot, but the browser color scheme does not change even i changed the Windows 7 color scheme to High contrast mode.
And high contrast scheme is pretty ugly (classic theme) so i decided to create my own version of CSS.

### Update notes
Suggestion and correction are welcomed.
Note, to update to latest theme, simply delete the existing shuu theme and re-download again.

rev.0.9 added option to disable Theme effects, altering color scheme, and more.

Utilized Web Open Font Formats, Oxygen, Lato, Titillium Web from Google: fonts.google.com
Adobe Typekit fonts require Javascript to works https://greasyfork.org/en/scripts/24371-adobe-typekit-custom-fonts-for-shuu/code

background Image Opacity CSS copied from http://stackoverflow.com/questions/4183948/css-set-background-image-with-opacity

Option marked with ** have no practical usage on shuu, at the moment.
