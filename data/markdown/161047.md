### Description
<b>Description</b>

This project is a dark version of <a href="https://tproger.ru/">TProger</a> website.

<b>How to contribute</b>  

Clone this project on <a href="https://github.com/Kenya-West/TProger-dark">GitHub</a> and start your changes. If you want to use styles from source code, then create new style in Stylish extension and put source code in opened window.

<b>How to use this style?</b>

You need <a href="https://github.com/openstyles/stylus#releases">Stylus</a> extension installed on your browser to use this style.
