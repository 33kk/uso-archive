### Description
DS - YouTube Fixed Position Player

<ul><li>Keeps YouTube player <strong>fixed</strong> at the top, allowing you to scroll to see suggested videos, comments, sharing and embed forms, etc while still seeing the video</li>
<li>You can select on which modes <strong>(default, theater or both)</strong> you want  (see notes below)</li></ul>

Suggestions? <a href="http://fb.com/messages/renatofrota">Send me a message</a>.

### Update notes
Tip: with "all modes" enabled during installation you can easily disable the script on Userstyle with 2 clicks (click Stylish icon on your browser, then click "DS - YouTube Fixed Position Player"). 

If you choose to fix the player only in a specific mode you won't be able to enable it so easily (will need to come back here and re-install). So test with "all modes" first and then re-install only if you are sure you want it enabled for only one mode.
