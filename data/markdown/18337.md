### Description
Applies the Personas Background to the browser's Status Bar

Features:
+ Makes the Personas Style Image apply to the Statusbar
+ Makes the Statusbar background Transparent
+ Removes the border from the status bar - compacting the icons and making it look real awesome

Integrates with the following Styles:
Style 8941

Additional Information:
This style works only on the new Firefox 3 browser.

Requires:
Mozilla Labs Personas Firefox Extension
