### Description
A userstyle to make YouTube a little more like how it was in the past!

Now available as a userscript: http://userscripts.org/scripts/show/161726

Features:
- The YouTube logo has been changed to the classic logo.
- Gray background!
- Like and dislike buttons are more like the old ones!
- Username looks more like it did before. (Credit to <a href="http://userstyles.org/users/94353">Hedja</a>)
- Guide is removed. (As it was useless)
- Hides comment avatars (Credit to <a href="http://userstyles.org/users/169178">KleinerDosenpfand</a>)
- No G+ menu (Credit <a href="http://userstyles.org/users/247766">LabyStudio</a>)
