### Description
Reformats the Watch the Show page (http://www.cbc.ca/thenational/watch/) so that it can be opened in a smaller popup window, moved to one side and kept playing while you browse other sites.

Works best with this bookmarklet that opens the page in a popup:

javascript:(function(){window.open('http://www.cbc.ca/thenational/watch/','','width=600,height=400');})()
