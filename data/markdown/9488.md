### Description
MBlue Nav-Style Tabs Vista
----------------------------------------------------------------------------------------------------------------------------------------
Description:
 1. Gives the TABS the Nav-Buttons style.
 2. Skins "New Tab Button Position" (NTBP).
 3. Align the "FAVICONIZED ICON" to the center of the tab.
 4. Transforms the Tabs THROBBER icon.
 5. Skins the Notification Box.
 6. Highly Customizable.
 7. Integrates with the other NAV-STYLE Styles.
----------------------------------------------------------------------------------------------------------------------------------------
Notes:
-If something looks wrong, try to not use other styles that changes the tabs settings, and use the default Firefox theme.
-Use the "Nav-Style Tabs Vista - Inverted" if you prefer the selected tab to be blue and the others white ( http://userstyles.org/styles/14466 ).
----------------------------------------------------------------------------------------------------------------------------------------
CUSTOMIZING:

 1. TABBROWSER BACKGROUND COLOR.
 2. TABS OPACITY.
 3. ICONS OPACITY.
 4. TEXT COLOR/SIZE/WEIGHT/OPACITY.
 5. NOTIFICATION BOX BG/TEXT.
