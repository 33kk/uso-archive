### Description
This style:
- displays only the favicon for Bookmarks Toolbar items
- displays only the folder name for folders on the toolbar (Icon for folders can be shown by deleting the last item in the CSS code) 
- shows the favicon name on mouseover

Tested on Mac OSX Firefox 3

This is a mashup of various other styles. Shoutouts to:
- Mac Favicon XL, Firefox 3 By Bevan
- Mac bookmark toolbar favicons By Daniel Einspanjer
- Bookmarks toolbar - icons only + text on hover By pyhedgehog

PS: Screenshot is using Firefox Personas Lino White
