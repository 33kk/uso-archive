### Description
Make OverAPI cheatsheet print friendly.<br>
All text black, remove unnecessary blocks.<br>
Use the print dialog to see the difference.<br>
(This stylesheet is applied for printing only)<br>
<br><br>
tip: If the last column is cutoff in print preview, change the 'zoom' level of the page to make the columns readjust.
'Zoom' can be changed by Ctrl+'+' or Ctrl+'-' , Ctrl + scroll wheel or from the browser menu.
