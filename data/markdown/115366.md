Reduce size of Back button and reduce height of Navigation Bar.

Firefox 34, <a href="https://wiki.mozilla.org/Releases/Old/2014">Dec 2014</a>, recreated grotesque enlarged back button, this style attempts to reduce size of Back button to match Forward button.
