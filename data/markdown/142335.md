### Description
Easy reading with bigger font on <a href="https://touch.facebook.com"><u>Facebook mobile site</u></a>.<br>
<br>
<a href="/styles/142336"><u>Click here if you prefer large font size</u></a><br>
<br>
This style is designed to increase font size on Facebook mobile site. Please note that changing font size is complicated because each element is affecting one another. Therefore only the followings are supported:<br>
<ul>
<li>Newsfeed (including text statuses, links, photo and video posts, events, sponsored posts, etc)</li>
<li>Liked / Commented / Shared By Friend (including posts and pages)</li>
<li>Timeline (including friends, groups and pages)</li>
<li>Suggestion (including groups, posts and pages)</li>
<li>Single Post (including comments, replies to comments, reactions and shares listing)</li>
<li>Hashtag Search (including posts from friends and public posts)</li>
<li>Find Friend (including friend requests, search and suggestions)</li>
</ul>
Hope you like it.<br>
