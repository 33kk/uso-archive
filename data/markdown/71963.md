### Description
A retro header for the official Urban Terror site at http://www.urbanterror.info that mimicks the look of 2009 and earlier header graphics. (The version of Thingley used here is the one from 4.1 while the original header used a 4.0 screenshot.) The header only works correctly if the site is set to the default narrow view (“Proportional Width View”).
