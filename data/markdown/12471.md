### Description
Allows you to Customize the background image of the Search Bar.  

Once again, this style supports both Firefox2 and Firefox3.


Functions:
+ Background image will be a faded Firefox logo with shadowing - smaller
+ Text color of the textbox will be changed to black (from gray), making it easier to read
+ Font families provided to optimize the appearance of the search's text
= Optional - Change the color of the searchbar (instead of an image, you can make the searchbar a solid color)
= Optional - Text size of the update details will be larger (must uncomment)
= Optional - Bold text for easier readability (must un-comment)

Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font


Integrates with the following Styles:
+ Style 9362
+ Style 9359
+ Style 7956
+ Style 8930


Applies to:
+ Searchbar
+ Google, amazon, live, yahoo, Wikipedia, inquisitor, and answers toolbar (among many others)
+ Ctrl+K or Ctrl+E

Browser Versions:
- Works on Firefox 2
- Works on Firefox 3

Additional Information:
+ This style has been tested on Multiple Themes!  It has worked perfectly on each.
