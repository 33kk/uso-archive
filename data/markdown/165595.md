### Description
<b>This Style has kindly been updated by Tuxi to work with the shiny, updated FL UI of 2018.

The old 'Fallen London - Winking Isle Contraband Indicator' no longer works and will be retired.</b>

In order to progress the Seeking Mr. Eaten's Name (SMEN) storyline on Winking Isle, it's necessary to NOT possess a long and frustrating list of items. This Style flags up Winking Isle contraband items on the Bazaar tab, so you can easily sell them.
