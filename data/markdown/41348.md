### Description
Based on this mockup:

https://bug616018.bugzilla.mozilla.org/attachment.cgi?id=495136

Always intended to work with latest minefield nightly.

Note: if the addon bar border color doesn't match the contentarea's one, use this code:

 #main-window[sizemode="normal"] #browser {
    border-width: 0 1px 1px 1px !important;
    border-style: solid !important;
    border-color: rgba(26, 26, 26, 0.4) !important;
    background-clip: padding-box !important;
 }

 #main-window[sizemode="normal"] #browser-border-start,
 #main-window[sizemode="normal"] #browser-border-end{
    display: none !important;
 }
