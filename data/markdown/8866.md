### Description
Customizes ALL Stylish-Related windows.

** For a Dark Theme (green on black), please use this style - style 1676, by user 1609.  These two styles are similar.

Features:
+  Increased Font Size - Stylish Style Editor (Code)
+ Better (easier to read) Font Family - Stylish Code Editor
+ Custom Background Image on ALL Stylish Windows (Code Editor, Manage Styles, Updater...)
+ Personalized Text and Background colors on ALL Stylish Windows
+ Better (easier to read) Font Family - Stylish Manager
+ Background image will be a faded Firefox logo with shadowing
+ Supports a huge list of Fonts (discussed below)
+ Image supplied in default code is hard-coded - allowing you to view the background, even when offline!
+ Background image is static and is centered on the window.
= Optional - Font size of the popup will be larger (must uncomment)
= Optional - Bold text for easier readability (must un-comment)

Fonts Supplied (Supported) - Stylish Manager:
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista) wow!
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font

Fonts Supplied (Supported) - Stylish Code Editor:
- Currently, I supplied the following Fonts for the Code:
1. Consolas (Console Interface of Vista)
2. Courier New (XP's Code Font)
3. Courier (Windows XP, 9x... Code Font)
4. Monospace (Default code font on all operating systems)

Applies to:
+ Check for Updated Styles
+ Stylish Code Editor
+ Stylish Manager
+ Stylish Popups (from Userstyles)

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2

Additional Information:
+ Feel free to edit the CSS.  You can customize the fonts, enable bolding, and change the background image, even change the font size!  No Re-posts without asking though!
