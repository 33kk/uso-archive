### Description
Changes your mouse cursor into a pretty cute Greasemonkey icon when hovering over user script links.

p.s. if you are using the GreaseKit add-on for Safari, you may load this as a user script. if wrapped in script, most css styles (including this one) are fully compatible with GreaseKit.
