### Description
MBlue Nav-Style Sidebar Vista
--------------------------------------------------------------------
Description:
 1. Gives the Nav-Buttons style to the Sidebar.
 2. Skins the Sidebar header, buttons, splitter, grippy, toolbars, background, etc.
 3. Skins the BOOKMARK/HISTORY Panel.
 4. Supports "All-in-One Sidebar" extension.
 5. Supports "Clear Fields" extension.
 6. Hides AIOS useless menuitems.
 7. Highly Customizable.
 8. Integrates with the other "Nav-Style" styles.
 9. Integrates with the other "Nav-Style Sidebar Vista" styles for supporting Downloads-Panel, Addons-Panel, and more Extension Panels (TabSidebar, TagSifter, etc.).
--------------------------------------------------------------------
Notes:
-This style skins AIOS and the Bookmarks/History Panels ONLY, but it can integrate with other "Nav-Style Sidebar" styles for more extensions support (read below).

-Works with or without AIOS. Better results with AIOS installed.
 
-Supports both small and large icons (works better with small).
 
-Do not use other styles that changes the Sidebar settings.

-All-in-One Sidebar:
This Style is optimized to use only a few of the features of AIOS. To make it work well, open the AIOS Options and disable all the extra Toolbars features. You should also disable Multipanel, Error Console and Extra Panels (they are useless and looks bad inside the sidebar). Only the Bookmarks & History Panels are supported (read below how to use more panels).
--------------------------------------------------------------------
Downloads-panel:
If you have AIOS installed and want to use the Downloads-Panel in the Sidebar, you need the "Nav-Style Sidebar Vista (Downloads)" style.

Addons-panel:
If you have AIOS installed and want to use the Addons-Panel in the Sidebar, you need the "Nav-Style Sidebar Vista (Addons)" style.

More Extensions-panels:
If you want more sidebar Extensions skinned, you also need the "Nav-Style Sidebar Vista (EXT. NAME)" style that supports the Extension panel you want to skin.
