### Description
Don't like the new pastel DevTools colour scheme that Mozilla introduced with version 49+ onwards? 
<br><br>
This 'combo' style will change everything back to the DevTools colours used up to version 48, along with some tab UI and AutoComplete colour tweaks.
<br><br>
<b>Snippets of CSS used from:</b><br>
M0dz145's Firefox Developer Edition - Normal colors: https://userstyles.org/styles/129197/firefox-developer-edition-normal-colors<br>
BladeMight's Developer URL-Bar Autocomplete: https://userstyles.org/styles/130242/developer-url-bar-autocomplete
<br><br>
<b>Works best with the dark theme</b>.<br>
<b>If using the Stylus addon; copy the CSS code into your userChrome.css file instead</b>.
