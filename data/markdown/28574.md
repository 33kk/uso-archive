### Description
This style is based on style 319 by user 10.
I slimmed down and beautyfied the code and added support for Beta.

The preferred language links are now fixed to the top left corner while scrolling.
Customizing is very easy:
- change li.interwiki-xx to your preferred language version
- play with the other values as you like to change the positions and styles
