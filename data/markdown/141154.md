### Description
Fixes checkboxes and radio buttons to have a white background and black checkmark if using a high contrast theme in windows. Disabled boxes have a grey background. This is a global style.

I also have a fix for more general firefox + windows dark theme issues : documented here https://www.reddit.com/r/firefox/comments/643h6q/fix_for_using_firefox_with_a_high_contrast_theme/
