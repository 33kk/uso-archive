### Description
A glossy theme with full lightweight theme/Personas compatibility. Updated from style 39806 for Firefox 29 and above, now with color customization. Tested on Linux, and to a lesser extent Windows 7; untested on other Windows versions and macOS/OS X. Some Windows platform testing done under Wine 1.9.x.

### Update notes

* Windows 7 - When the system title bar is hidden, the window control buttons (close, minimize, maximize) are blacked out.
* Windows 7 - When the window is maximized, part of it may go offscreen.
