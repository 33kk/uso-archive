### Description
I don't like the Youtube's header size, so I decided to hide it and show just when I want to search or use some links. And that's what this code do. Hide top header and leave just a tab and when you hover it with the mouse the header is displayed again. Pretty simple. Compatible with all my Youtube Scripts [ http://userstyles.org/styles/25233 , http://userstyles.org/styles/23387 ].
