This was the old dark theme on Finobe, which existed from May - Late 2017. The theme was changed in Late 2017. This is a replica of it.

This is also the same dark theme on Nobelium. I imported it from there.

IMPORTANT: Don't use the real dark theme or the 2009 theme while using this theme. They don't work properly together.
