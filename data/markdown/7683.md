### Description
Need FaviconizeTab extension.

Normal: Only selected tab with title, other faviconized.

Hover other tab: Show text.

Configuration FaviconizeTab:
<-----------------
tab[faviconized="true"] {
   min-width: 36px;
   max-width: 36px;
}

tab[faviconized="true"] .tab-text,
tab[faviconized="true"] .tab-close-button {
   display: none;
}
------------->

Paste this in extensions/faviconizetab@espion.just-size.jp/content/faviconizetab.css

In FaviconizeTab's preferences:
-in auto faviconize tab sections: write '*'
