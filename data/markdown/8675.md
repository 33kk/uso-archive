### Description
Description:
 Transforms the sidebar to integrate better with the Simple-Gray Theme.
--------------------------------------------------------------------
What it does:
 - Gives the sidebar (better if using AIOS) the Simple-Gray look
 - Skins the Bookmarks & History when opened inside the sidebar
 - Makes the AIOS Top-Sidebar-Toolbar Tabs-like
 - Sidebar thin splitter (2 versions to choose)
--------------------------------------------------------------------
Supported Extensions:
 All-in-One Sidebar, Tab Sidebar, TagSifter, QuickNote.
--------------------------------------------------------------------
Notes:
All-in-One Sidebar: This Style is optimized to use only a few of
 the features of AIOS. To make it work well, open the AIOS Options
 and disable all the extra Toolbars features. You should also
 disable Multipanel, Error Console and extra panels cause they are
 useless and looks bad inside the sidebar. Only the download and
 Bookmarks/History Panels should be enabled. Enable the Extension
 Panel when using the "Simple-Gray Slim-Extension" for better
 results.

Downloads: If you want your downloads to be in the sidebar, you
 should use the "Simple-Gray Slim-Download" Style to integrate it
 better in the Sidebar.

Extension-list: If you want the Extensions List to be in the sidebar,
 you should use the "Simple-Gray Slim-Extension" Style to integrate
 it better in the Sidebar.

More Extensions: You can use this in combination with the
 Simple-Gray Sidebar Ext-Support to skin some extension that opens
 in the sidebar.
