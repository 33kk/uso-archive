### Description
Compresses the grabber/resizer between the URLBar and the Search Bar.  Greatly increasing the size of your Awesome Bar.
This resize panel allows you to shorten or lengthen the searchbar, but it is a bit big, so this code fixes the problem!

Integrates with the Following Styles:
+ Style 7173
+ Style 7723

Works only on:
-Firefox3 (AwesomeBar is lengthened)
