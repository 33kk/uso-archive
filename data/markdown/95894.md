### Description
<p>A port to Stylish of the <a href="https://userscripts-mirror.org/scripts/show/153894">FiMFiction.net Recolorizer</a> Greasemonkey script by yrfoxtaur (plus enhancements).</p>
<p>This style allows you to set a custom site background on FiMFiction.net.</p>
<p><B>IMPORTANT:</b> Click the "Advanced Style Settings" button to configure this.</p>

### Update notes
<b>Instructions:</b>
<ul>
<li>To set a solid background color, just use the same color for the top and bottom of the gradient.</li>
<li>To use both a gradient and a tiled image, edit the image to have transparency.</li>
<li>There are plenty of good textures at <a href="http://subtlepatterns.com/">Subtle Patterns</a>.</li>
<li>Let me know if there are any other background image positioning presets you'd like.</li>
</ul>

<b>Caveats:</b>
<ul>
<li>I don't actively use this style myself so please <b>inform me immediately if something breaks</b>.</li>
<li>Some browsers have cache bugs with <code>data:</code> URIs. If you middle-click "See It", also reload.</li>
<li>As of this writing, the configuration options don't work when loading as a userscript, so you'll have to edit the CSS in the userscript version by hand.</li>
</ul>
