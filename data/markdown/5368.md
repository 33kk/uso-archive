### Description
This style change to monospace the font used to display Discogs internal messages.

Monospace is a generic term used to characterize a font where "all glyphs have the same fixed width" [1].
It's commonly set as Courrier or MS Courrier New.
See your browser help documentation to learn how configure this font.

Note : the composing text area already uses the monospace font

[1] http://www.w3.org/TR/CSS21/fonts.html#monospace-def

Edit on March 8th 2010: adjusted font size to comply with new Discogs look.
