### Update notes
<b>*</b>On macOS, you need to set the <b>General</b> > <b>Show scroll bars setting</b> in <b>System Preferences</b> to "<b>Always</b>" for this property to have any effect.
<br>
<br>
<br>
<hr>
<b>For version 63 only<b>: 

this feature is behind the:

<ul><li><code>layout.css.scrollbar-colors.enabled</code> 
<li><code>layout.css.scrollbar-width.enabled</code> </ul>

preference (needs to be set to true). To change preferences in Firefox, visit <u>about:config</u>.
