### Description
Dark styles for Wikipedia. See "Advanced Style Settings" for different themes.
<br/>
Features:
<ul>
<li> Hides top/left banners (visible on-hover).</li>
<li> Improved readability by adding a slight shadow to all text.</li>
<li> Shadows on everything because depth is cool.</li>
<li> Syntax-highlights code snippets.</li>
<li> Works with mathematical formulas.</li>
</ul>
