### Description
This style needs a Tampermonkey script to work... as described in this blog post:
https://blog.workflowy.com/2016/05/19/different-fonts-for-different-workflowy-outlines/

A handful of tags that allow one to:

- Transform an entire outline to UPPERCASE – #uc
- Cause certain lists/ tasks to flash intermittently (fade in and out) – #blink
- Text-align: center – #center
- Hide the bullets for specific outlines – #nb 
- Create simple styled block quotes –#quote 
- And more...
