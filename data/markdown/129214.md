### Update notes
This is a quick and dirty first release so the css is a bit untidy and a little too general.  Depending whether/when the website is updated, I will try to tidy it up in future.

I have only tested it with Chrome and Firefox, though it should also work with Opera and Safari.

Not all pages have been changed yet - notable omissions in this initial release are:
Forum FAQs
Forum search
Settings

Let me know (via PM on a.net, or by posting to the thread in Site-Related) if you notice anything that this breaks, or that you would like updating.  Also let me know if anything stops working properly when DM deploy updates or fixes to the site - they often rename or move objects so some of these style improvements can occasionally get undone.

note that we cannot add or change functionality, only appearance.
