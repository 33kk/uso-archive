### Description
Inverts all colors except for images and videos.

Also makes the sidebar (with the contacts list) automatically resize when you hover over it, so that there's more space for the messages.

This style also removes the notifications above the contact list. If you prefer to keep those, remove the "display: none;" line in the code where it says "annoying message".
