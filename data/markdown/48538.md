### Description
For vanilla 4chan (Yotsuba / Yotsuba B / Futaba / Burichan) + 4chan X (2.9.0+) users.

Adjusts style of inline replies and hover preview to be clearer. 

E-mail and filename expansion options plucked from a github issue (https://github.com/aeosynth/4chan-x/issues/144).
Outline fading from github comment (https://github.com/aeosynth/4chan-x/issues/271#issuecomment-2044308).
