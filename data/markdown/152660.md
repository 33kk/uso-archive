### Description
<p>Various readability and usability improvements:</p>
<ul>
<li><b>Customizable fonts</b> — fonts can be set for various elements. By default, Segoe UI for UI, Verdana for text.</li>
<li><b>Customizable text width</b> — width of work and article text can be changed. By default, 45em.</li>
<li><b>Tags colored by category</b> — red warnings, cyanish relationships, yellowish characters, grey all others.</li>
<li><b>Expanded required tags' icons</b> — text next to required tags' icons explaining what they mean.</li>
<li>Columns in tag index</li>
<li><b>Compatible with <a href="https://userstyles.org/styles/97658/dark-night-archive-of-our-own">dark theme</a></b> — to enable dark theme, install <a href="https://userstyles.org/styles/153549/">Dark</a> style modifier.</li>
</ul>
