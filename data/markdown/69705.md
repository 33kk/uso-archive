### Description
A new style which changes the look as well as cleaning up many useless elements. Please note the font used in the screenshots is Quicksand and will need to be installed; instructions how to do this below.

### Update notes
<u><b>Instructions On Using 'Quicksand' Font</b></u>
1. Download the font from here: http://www.fontsquirrel.com/fonts/quicksand
2. Extract files and install 'Quicksand - Regular' to your computer. 
3. On this page select 'Quicksand' from drop down menu, then hit 'Update Style'.
