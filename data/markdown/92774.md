### Description
This userstyle hides back and forward buttons in a maximized Firefox window, meant to be used with extensions that provide alternative ways of using them, like <a href="https://addons.mozilla.org/en-US/firefox/addon/milewideback/">MileWideBack</a> or <a href="https://addons.mozilla.org/en-US/firefox/addon/edgewise/">EdgeWise</a>.
