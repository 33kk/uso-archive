### Description
Allows you to set a Specific Width for the FindBar.  
You can essentially make the Find Bar bigger than the default, or really small.... it's up to you!
The find bar will stay at the width you specify, and will not grow, shrink, or be able to be moved physically. 
Allows you to use the Most Possible area for your Searches.


The FindBar can be accessed by pressing the Key Combination (Ctrl+F   or   Command+F on MACs)  It can also be activated by typing on a page if you have the right setting (Tools / Options / Advanced Tab / ENABLE "Search When I Start Typing")

Instructions:
- Specify the Width in EM (Relative Size) - Customizable
- You can also shrink the find bar if you wish.

Code Provided:
- The code provided will greatly increase the find bar's size with a width of 40em.
