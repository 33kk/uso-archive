### Description
Темный контрастный стиль для Два.ч, основанный на сочетании оранжевого цвета и оттенков серого.

Для установки в <i>Opera</i> нажимаем "Load as plain CSS" и следуем <a href="http://my.opera.com/micahrayaills/blog/how-to-install-custom-user-style-sheets-in-opera-and-firefox">инструкции</a>.

Если у вас установлен <a href="http://www.freedollchan.org/scripts">Dollchan Extension Tools</a>, то часть его оформления тоже будет изменена.

Скриншоты актуальны для <i>Opera</i>, <i>Chrome</i> и <i>Firefox</i>. Работа с другими браузерами не проверялась.
