### Description
This style not for stylish/stylus.

First of all, set two options:
Menu > Customize:
 - Themes: Light

Then go to: about:profiles, click "Open Finder" on root folder. In your profile folder create folder named "chrome" and file "userChrome.css" inside it. Put css code into this file.

Restart Firefox.
