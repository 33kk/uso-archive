### Description
Only displays the "Apply" button when you make a change to Menu Editor (a Firefox extension that allows you to change the layout of your menus).

Why show a button that you can't click!?

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2

Integrates With Style 8297.
Disabled buttons are hidden and Enabled buttons are shown
