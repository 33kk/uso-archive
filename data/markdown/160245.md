### Description
A list that aims to make the websites of the Prisjakt group's websites (also incl. Pricespy, Hintaopas, Kompario, leDenicheur and Pagomeno) considerably wider and more befitting for widescreen monitors.

Tested on a 1440px-wide monitor. Some columns are of uneven length, since the sites re-use web code with the same names for multiple different scenarios, thus limiting how much I could even them out.

Note that the example screencap was taken in combination with unrelated and non-included filter lists.
