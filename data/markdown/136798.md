### Description
A style that removes all the distracting stuff when reading a Medium article (plus some miscellaneous improvements), based on style 135054.

Note: This only works with articles hosted on medium.com. To add off-site Medium blogs, please open Stylish settings, select "JustRead for Medium" and add the domain below the code window.
