### Description
Dark-themed page style for the "Install Extension" page in Firefox.
This style also applies to the "Add a new theme" page

Features:
+ Background image will be a Dark / Black faded theme Firefox logo with shadowing
+ Text size of the update details will be larger (14pt instead of 12)
+ Optional Bold text for easier readability
+ Supports a huge list of Fonts (discussed below)
+ Background image is static and is centered on the window.
+ Background is a dark gray themed color, perfect for those using Darker themes such as NASA Night Launch

Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font


Applies to:
+ Extensions List
+ Addons List, new extensions ("Software Installation - Only Install Addons from authors whom you trust")
+ Themes Install page

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2 as well

Integrates with Styles:
+ Style 8028
+ Style 8320
