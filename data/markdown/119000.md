### Description
Please view this prototype in Chrome using the Chrome Stylish addon.  This will allow you to view any page in adf.org using the prototype.
<li>Left pane navigation improvements on mouse, touchscreen, and screen reader accessibility</li>
<li>Modernized display of links and other page elements, such as author bio information</li>
<li>Fix for font sizing in small devices (needs changes to existing files to complete this)</li>
<li>Improved display of search, login, and logged-in areas</li>
<li>Fix for "stretched-out" appearance in large resolutions and wide screens</li>
