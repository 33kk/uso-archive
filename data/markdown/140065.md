<b><a href="https://addons.mozilla.org/fr/firefox/addon/uaswitcher/?src=search">"User Agent Switcher"</a></b>
AND
<a href="https://addons.mozilla.org/en-US/firefox/addon/user-agent-string-switcher/">User-Agent Switcher and Manager</a>
I use this one with this configuration in in custom mode as shown below and it worked immediately:
{
  "www.youtube.com": "Mozilla/5.0 (compatible; Googlebot/2.1"
}

With this trick i regain the OLD YouTube interface and keep my YouTube addons and userscripts working on it.
