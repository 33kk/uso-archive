### Description
This style changes the background color of the main horizontal bars in Firefox (except scroll bar).
The code uses RGB to define the color so you can very easily & quickly change it.  This style
can be used in combination with style 3728 to also change
the tab colors.
::
This Style was tested on Firefox 2.0 using its default theme "Firefox (default) 2.0".  Code credit to np.
