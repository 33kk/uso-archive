### Description
用必应每日一图作为知乎（zhihu.com）的背景。

是不是觉得知乎白亮白亮的背景色晚上看着太伤眼了？这个 CSS 样式给知乎设置了必应每日一图作为背景，它每天会自动换背景，页面色调也统一调成了米黄护眼色，看着心情好多了！

### Update notes
2020-09-11: version 0.28
知乎的新版界面，首页左中右三栏，左栏吸左，右栏吸右，在高分屏上看着很崩坏。所以仿照微博的三栏结构，对知乎的新首页进行了改造，改造成三栏抱团居中，还加了个半透明背景，先这样用着，等知乎的新界面稳定下来以后，再全面更新一下这个 Style 。

2020-09-09: version 0.27
知乎推出了新版界面，就是加了个左边栏，把这个左边栏给调成护眼色。

2020-08-31: version 0.26
搜索结果页的“杂志文章”条目背景是白色的，把它也改成护眼色。总之这个主题的设计思想就是“讨厌白色背景”，因为太伤眼了，所以只要看见有新的白色背景的块元素，我就会把它的背景修改成护眼色。

2020-07-17: version 0.25
知乎界面又有些更新，加了个侧边栏，把它也改成护眼色。

2019-11-03: version 0.24
把背景图片滤镜修改为上明下暗的渐变。

2019-07-25: version 0.23
把搜索页等一些新出现的白色部分搞成米黄护眼色。

2019-06-26: version 0.22
把知乎圆桌、书店、创作中心，特别是电子书的阅读界面，都搞成了米黄护眼色。
然后隐藏了时间线和边栏上的广告。

2019-05-27: version 0.21
专栏文章页文章宽度增加了80像素，之前偏窄，导致底部推荐阅读显示不全。

2019-05-18: version 0.20
增加了一个颜色值：hovercolor，用来取代一些链接的 hover 状态色。之前大部分是白色的，在黄色的大背景下，白色看着有点儿像撒了石灰，感觉不太好，现在把 hover 状态的颜色改为浅蓝色，清爽一些。

2019-05-18: version 0.19
修改了暗色的颜色值，让它更搭配一些。

2019-05-15: version 0.18
补充了一些漏掉的白色的背景：邀请页、回答问题时的全屏编辑界面、问题页的引用段落等等。

2019-05-10: version 0.17
补充了一些漏掉的白色的地方：消息、私信的浮动面板、个人设置界面等等。

2019-05-10: version 0.16
评论列表的颜色之前漏掉了，纯白色，太伤眼了，现在改为稍暗的米黄色。

2019-05-07: version 0.15
添加了明黄(lightcolor)、暗黄(darkcolor)的区别，主要应用在输入框上。

2019-05-06: version 0.14
小小的修改，把散布各处的背景颜色值提到全局变量 bgcolor 里，小小加深了背景颜色值： #f9f9e9 。

2019-05-06: version 0.13
纯白色的背景大晚上还是太亮了，即使开着 Win10 的夜间模式也觉得太亮了，伤眼睛。所以把多数页面的内容背景修改为养眼米黄色： #f9f9f3 。

2019-05-04: version 0.12
样式的适用域修改为： domain(zhihu.com)

2019-05-03: version 0.11
添加了知乎专栏里 .GreyBg-body, .LightGreyBg-body 这两个新类的必应每日一图背景支持。之前只有 .WhiteBg-body 。

最初创建。
