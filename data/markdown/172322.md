### Description
Reveals text hidden by spoiler bars on SpaceBattles and Sufficient Velocity. 

Spoiler bar formatting: 
<code>
[qs]text[/qs]
</code>
Without the userstyle, text inside the spoiler bar's [qs] tags is blacked out (black text on a black background). The text is revealed when the mouse moves over the spoiler bar and the text color changes to grey. 

With the userstyle installed, the color of the spoiler bar text is red so the text is always visible.
