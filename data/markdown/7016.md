### Description
N'affecte que les articles de l'actualité et les agrandissement d'images (aucun changement sur les sections Test, Dossier, Téléchargement, etc...). Ce style rend la largeur de la page dynamique (vive les grandes résolutions) et allège l'affichage. Les commentaires ne sont plus affichés directement.

Le code est commenté pour que l'on puisse modifier le comportement du style, il est possible de :
- Supprimer entièrement les commentaires (plus de titre, ni de bouton d'accès au forum, ni de nombre de message)
- Supprimer la barre "Mon Clubic" permettant de se connecter
- Modifier le ratio page/écran (par défaut 98%)

(Faire un clic droit puis "Afficher l'image" pour agrandir l'image ci-dessous)
