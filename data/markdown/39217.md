### Description
Style for the addon-bar to set it to the right edge of the browser window and to only have the width which is necessary, like in some the mockups of Mozilla (http://jboriss.wordpress.com/2010/06/16/removing-firefox%E2%80%99s-status-bar-and-rehousing-add-on-icons-part-3-of-2-wut/).
Only for Firefox 4.0!

### Update notes

Update 08.03.2011: Add option for full transparent and removed close-button.
Update 12.05.2011: Remove Personas background for full transparent version.
Update 13.05.2011: Add margin-bottom to vertical scrollbar and removed horizontal scrollbar for fixing the scrollbar problem.
