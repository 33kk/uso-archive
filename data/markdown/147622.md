### Description
This a remake of the connexus modern using the solarized palette.

<p>This theme is for anyone who's ever looked at a connexus lesson and thought, <i>when was the last time this style was updated? 2000?</i>. Changes:
<ul>
  <li>Lessons<ul>
      <li>-Solarized Colors</li>
      <li>More modern UI</li>
  </ul></li>

  <li>Pearson Textbooks<ul>
      <li>Better readability</li> 
      <li>Better utilization of screen space</li> 
      <li>More modern UI</li>
  </ul></li>
</ul>

If you are making changes to the theme, the source LESS files are easier to work with and are commented. In addition, if you find an issue or the theme lacks support for a course, make an issue or pull request on the gitlab page.<br /><br />Source: <a href="https://gitlab.com/tacticalpha/solarized-connexus">https://gitlab.com/tacticalpha/solarized-connexus</a>
