### Description
This style is designed to make your location bar dark transparent. The color of the URL bar text and border will change with your security level. Clear is normal, green is secure, and red is unsecured. 

Based on style 17312 by user 10984

Tested on Mozilla/5.0 (Windows NT 6.1; rv:2.0b10pre) Gecko/20110117 Firefox/4.0b10pre ID:20110117030328

Looks best when using a transparent navbar.
