### Description
Gives you Rounded Edges and Transparency on all tooltips.

* Thanks to ( http://www.darknet.org.uk/wp-contentithemes/tonus/plugins/fancytooltips/fancytooltips.css)  for helping me get some of the code (provided to me by user 8168)
* Also thanks to ( http://www.victr.lm85.com/fancytooltips/ )

** Request came from HERE: ( http://userstyles.org/forum/comments.php?DiscussionID=1591&page=2&amp%3Bpage=2 )

Applies to:
+ Navigation Toolbar (Hover over button)
+ Hovering on a tab (tooltip will have radius)
+ Also Applies to EVERY single item on any web page (Hover over a link)

Features:
+ Dark Gray Border
+ Dark Gray Text
+ Light Gray Background
+ Rounded Edges
+ Transparent (see-thru) tooltips


Browser Versions:
+ Works only on Firefox3

Integrates with the following Styles:
+ Style 8558
+ Style 8414
+ Style 8410
