### Description
The progress bar was removed in Firefox 4 beta8pre so to get it back you need to install "status-4-evar" add-on here: https://addons.mozilla.org/en-US/firefox/addon/235283/
This style makes the progress bar wide, looking like fission add-on was. It also adds a nice green (windows vista/7 like if you're using one of these two) style to it.

Credits to foxxyn8 for the help!
