### Description
A skin I've put together to add a light and fluffy feel to Azurilland. The code has been authored primarily by myself, but I've used and looked at code from Azurilland users Screem, shhmew, and Thundawave. Credit for the site background goes to <a href="https://aristanova.deviantart.com/art/Rainbow-Sky-Custom-Box-Background-410856303">Sarah--Elizabeth on DeviantArt</a>.
