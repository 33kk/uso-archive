### Description
Based on and extending Lou Cypher's Colorize Firefox 2 Active Tab, this script colors the active tab, the inactive tabs and whatever tab you're hovering over.  It also bolds and sets the font colors.  Everything can be easily modified by changing the color and attribute codes.
