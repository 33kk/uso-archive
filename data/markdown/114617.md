### Description
This style changes the appearance of Google Docs documents providing a beautiful and distraction free writing environment. It replaces the document and sheet backgrounds with a theme similar to ommwriter (edit freely to use other images), hides away the toolbar and the comments section (Thanks IanHead). Use it in fullscreen mode for best effect. Enjoy.
