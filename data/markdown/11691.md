### Description
This Style applies the default Firefox theme for dir listings on Apache dir listings. For example the page will be shown in a rounded box and there will be more nice links.

Note: Please disable this style while viewing other pages than dir listings, otherwise there will be a funny effect. Except you like to browse in a rounded box ;)
