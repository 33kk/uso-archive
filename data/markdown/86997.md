### Description
This style makes Thunderbird tabs look like those in <a href="http://gnome-look.org/content/show.php/?content=154296">the FlatStudio GTK+ theme</a> (without the rounded corner, as when tabs-2.css is selected in gtk.css).  This is a direct port of my <a href="http://userstyles.org/styles/86995/firefox-flatstudio-tabs-like-tabs-2-css">Firefox FlatStudio Tabs</a> style.
