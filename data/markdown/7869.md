### Description
Make the Tab Toolbar a custom height.  This allows you to bring up your favorite sites, while conserving more screen space.  This style ONLY works on Firefox 3.  Compresses the tabs list. 

** Code has been modified based on style 104 by user 2.  Thanks!

Features:
- Currently menu will be shortened, but you can make it larger if you want.

Browser support:
+ Works on Firefox3 (Fx3 Only)

Additional Information:
- Simply edit the CSS and change the value from 24px to your custom value.


Integrates with Style 7784.
