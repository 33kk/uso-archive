### Description
Replicon's Timesheet used by many companies, but with added Calibri Vista fonts, and some updated user interface.

Additional Information:
- Bolds the Calendar headings for better readability
- Calibri default font - Still supports XP machines with Arial
- Sets default font to 10px


Instructions:
- If needed, change the moz-document prefix to match your company's URL.
