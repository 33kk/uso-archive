### Description
All credit goes to the original authors.

The difference with this style as opposed to Streamlined Gmail 1, is that a widescreen monitor is no longer a requirement. Edit the text colours to suit your theme.

This style moves the account activity and usage to below the Search Mail button.

The style implements the following:
1. Hides Invites.
2. Hides Chat.
3. Hides Ads and relocates actions.
4. Hides the Buzz Icon.
5. Hides each component in the footer separately.
6. Includes the code if you want to switch off quick links.
7. Includes the code if you want to switch off the entire footer at once.
8. Moves the % usage and account activity below the Search Mail button.
9. Adds row highlights.
