### Description
<b>NOTE: As of 01.01.2013 this style will no longer be supported.</b>
Replaces right control panel icons with a color of your choice. This style is not automatic and requires a bit of work.

### Update notes
A graphics application is required to manipulate the resulting screenshot. The graphics application must be capable of removing the background color and saving the image as a transparent background PNG or GIF file. This image can be stored locally, on an image storage site, or converted to data URI.
The steps:
  1) Select <b>Icon Template</b> background color (black or white)
  2) Select new <b>Icon Color</b>
  3) Install the style or select the <i>Preview</i> button (FF only).
  4) Take a screenshot of your tumblr dashboard. Cut the mini-icon set from the top of the screen. Cut <i>inside</i> the green outline. Final image dimensions should be 120w X 50h. Remove the white or black background. Save edited image.
  5) Return to this style page. For <b>Icon Positioning</b>, select Install Style.
  6) Provide your new icon image URL. The style should take care of positioning.
