### Description
<p>Для <b>vk.com</b>:</p><ol><li>Выделяет цветом собеседников, у кого или от кого есть непрочитанные сообщения.</li><li>Уменьшает радиус скругления на аватарках.</li><li>Подкрашивает заголовки записей на стене.</li><li>Усекает по высоте панель для возвращения наверх/назад.</li><li>Увеличить игровую область для приложений может <a target="_blank" class="stl" href="/styles/135548">Vkontakte - Game</a>.</li><li>Скрыть рекламу может <a target="_blank" class="stl" href="/styles/135146">Advertisement - Hide</a>.</li></ol>
