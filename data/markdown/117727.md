### Description
Display a visual badge and remove the underline below tabs for <b>Private Tab</b> add-on for Firefox. Also optionally hide the private browsing indicator usually displayed in the title bar.

### Update notes
<b>IMPORTANT</b>
This userstyle require Private Tab (https://addons.mozilla.org/en-US/firefox/addon/private-tab/) and at least Firefox 35.

To avoid display issues with empty favicons Classic Theme Restorer (https://addons.mozilla.org/en-US/firefox/addon/classicthemerestorer/) or a similar add-on is recommended for restoring the generic dashed square icon (as in older releases before Firefox 28 and Australis UI landed). The latter doesn't apply for Palemoon.
