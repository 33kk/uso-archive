### Description
Only displays the enabled buttons on the Firefox user interface.

Features:
+ Disabled buttons (grayed out) will not show (hidden)

Works on all Extensions Options, advanced options of any type, settings pages...

Why show a button that you can't click!?

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2
