### Description
Changes your cursor, with costumization settings!

### Update notes
A cursor will need 2 images: the pointer and the highlighter.
The pointer will display at any times except when you hover over a link;
The highlighter will display when you hover over a link.
NOTE: You must install with the settings already changed, else you'll get error 400!
NOTE 2: Cursor images must be 32x32!
