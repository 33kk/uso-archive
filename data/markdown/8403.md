### Description
This style expands on Del.icio.us Chocolate by delarge.

The primary changes are:

1) Switch of font from Tahoma to Trebuchet MS for readability
2) Darkening the item bookmarked site link colors
3) Increasing the spacing between tag bundles and dulling down the bundle name 
4) Putting a gray background behind the active tag
5) lowering the visual intensity of less frequently used utility / control links (e.g. edit / delete tag links0
