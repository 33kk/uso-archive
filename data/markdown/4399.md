### Description
This style moves the GO Button to the left of the Address Bar.  It has an option to also move
the drop-down arrow to the left of the Address Bar (use option by uncommenting 1 line
in the code as described in the code itself).
::
Whenever you activate this style, open a new window or restart Firefox to see the change.
::
This Style was tested on Firefox 2.0 using its default theme "Firefox (default) 2.0".
Code credit to ChoGGi.
