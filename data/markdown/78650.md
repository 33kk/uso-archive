### Description
Amazon displays a list price along with the actual price, thus highlighting (irrelevant?) savings one is apparently making, which then dominates the value being actually spent on the item. 

This hides the list prices on amazon, so that you can decide for yourself, if you want to pay the actual price for the item on the page.
