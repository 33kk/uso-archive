### Description
<p><b>Attention: this theme no longer works with stylish because of changes to the stylish extension. See the installation procedure below to install this theme.</b></p>

<p>Make Firefox use Windows 10's accent color. Also correctly style inactive Firefox windows. This theme only works with Firefox 60 or newer.</p>

<p> Installation:
<ol>
  <li>Enter 'about:profiles' into the Firefox url bar.</li>
  <li>Click 'Open Folder' in the Root Directory field of the default profile.</li>
  <li>Create the folder 'chrome' if it does not already exist. Enter the folder.</li>
  <li>Create the file 'userChrome.css' in the chrome folder if it does not exist. Add the CSS from this theme to userChrome.css</li>
</ol></p>
