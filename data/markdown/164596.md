### Description
Reverts the worst of the changes in the new Gmail theme.

<ul>
<li>Forces Cozy spacing. This is between Compact and Comfortable.</li>
<li>Restores the font to Arial instead of Roboto.</li>
<li>Reduces the font size to approximately what it was before.</li>
<li>Fixes font kerning for Firefox on Linux. See Mozilla bug <a href="https://bugzilla.mozilla.org/show_bug.cgi?id=939132">939132.</a></li>
</ul>
