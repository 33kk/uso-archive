### Description
Animates the background of the Favicon Throbber of a Loading Tab.

Works only for Firefox 3

** Thanks to user 1819 for the style 2125, upon which this is based (the style noted does not work though!)


Features:
+ Animated background of a loading tab's favicon (throbber)
+ The animation will cycle through Red, then Yellow, then Orange.  This is done in a slow manner, not to be too distracting to the user.
+ More flashy to the eyes, notifying you that your browser is busy loading a page.
+ Animation and background color may be customized

Applies to:
+ Tabbar
+ favicon (another visual alert)
