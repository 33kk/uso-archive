### Description
Este estilo permite hacer aparecer y desaparecer la barra que contiene el nick, el avatar y las estadísticas en un foro de tipo vBulletin.

Para que NO se vea esa barra: Normalmente, no se verá.

Para que SI se vea esa barra: Pasa el puntero del ratón por DEBAJO de la barra de fecha del post y aparecerá el avatar, nick y estadísticas.
