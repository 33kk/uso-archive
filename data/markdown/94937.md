### Description
Changes the width of The Fanfiction Forum. Currently set to 50% min-width and 75% max-width, it can be adjusted to whichever percentage you want just below.

Credit to acebrain01 for his <a href=http://userstyles.org/styles/23356/dlp-wide>DLP Wide</a> which inspired me to make this.
