### Description
После обновления дизайна Вконтакте изменилось используемое семейство шрифтов. По какой-то причине новые шрифты на моём компьютере отображаются из рук вон плохо. Данный стиль возвращает старые шрифты (font-family: tahoma, arial, verdana, sans-serif, Lucida Sans) на место.
