### Description
Plain text, only a few graphics, all language support, easy color substitution.

Works fine with XingUserImagesMouseOver (http://userscripts.org/scripts/show/6173) for Greasemonkey (https://addons.mozilla.org/en-US/firefox/addon/748).

To show XING-Header remove the following lines:

/* header ausblenden  */
#header-navigation, #header-logo, #header-search, #header-status{
display: none !important;
}
