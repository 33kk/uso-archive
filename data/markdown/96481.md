### Description
A toggleable Dark mode for users of <i>Style 79374</i>

### Update notes
Make sure to install or (re-)enable this style <b>after</b> the main 'light' style to allow it to take priority.

Dark-mode overrides are more aggressive attempts at darkening everything, but will sometimes darken things that shouldn't be darkened (like the phylogeny timelines on a species' page).
