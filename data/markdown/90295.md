### Description
A visual enhancement for Twitter, focusing on the content. With additional featuresfor Persian users, e.g. using "Iranian Sans" as main font for tweets and "Droid Arabic Naskh" for links, buttons and other texts, plus realigning some objects to make a better feeling in Persian Twitter.
It also can be used in English Twitter for better Persian support.
