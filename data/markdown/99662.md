### Description
Reduces the font size for almost every text element on the new photo page. Also makes the full EXIF block a bit better readabe by dimming EXIF field names.
Do not use this style with default super-thin font unless you are going to break your eyes. Please, install <a href="http://userstyles.org/styles/99605/flickr-replace-the-proxima-nova-font">this style</a> to replace Proxima Nova font with default sans-serif one. Another option to consider for font improvement - making the photo title bold by <a href="http://userstyles.org/styles/99650/flickr-bold-photo-title-on-single-photo-page">this style</a>
