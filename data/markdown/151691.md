### Description
A mobile mode for browsing kissmanga with touchscreen devices.

### Update notes
Version 1.0:
* Mobile mode is working on following pages:
    - Home
    - Manga List
    - Manga Details
    - Chapter
    - Search Results
    - Advanced Search
    - Login
    - Register
* iframes cannot be modified with Stylish, so the comments will stay as they are :(
* Pages on other domains (Comics List, Shop etc.) won't be modified.
* Did you notice a bug? Do you like the mobile mode? Please let me know.
