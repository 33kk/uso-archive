### Description
This style is used in combination with my userscript to fix display inconsistence and incorrectness of CJK ideographs at lang-8.com journal pages.
For further description, please see the page of the userscript: https://greasyfork.org/en/scripts/25510-lang-8-add-lang-tag

The style simply forces the browser to use sans-serif for all the contents, so in order to make it display correct language-specific variants of a certain ideograph, you also need a "localized forms"-enabled font and make it default sans-serif. I recommend Adobe Sans Han Sans.
