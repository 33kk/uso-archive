### Description
In firefox, after using adblock, there should be some white block in the navigate pages. So this CSS can remove them.

In opera, just leave in css:
#Header #Top #Logo 
 { display: none !important; } 
#Header #Top #Menu #Mt1
 { display: none !important; }  
#Header #Top #Menu #Mt2
 { display: none !important; }  
#Header #Top 
 { display: none !important; } 
#ContentTop #CtCenter #CtCTop #CrctTitleb #right 
 { display: none !important; } 
#Ws #End #AdsE 
 { display: none !important; } 
#Ws #Top #AdsT 
 { display: none !important; }
