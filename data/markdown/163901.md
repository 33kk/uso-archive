### Description
※這個樣式表，光點安裝是會破圖的！
與 NGID、ID 查詢、Komica 通知 相容，Chrome, Firefox 可正常渲染。
<a href="https://i.imgur.com/LYI3d5o.gifv" target="_blank">示範影片</a>

1. 為什麼會破圖？因為，你還要自己加上面兩句語法。<a href="https://i.imgur.com/1o5lbWw.gifv" target="_blank">教學影片</a>
2. 為什麼要你加？因為，這個網站禁止引用第三方的樣式，所以只能讓大家自己編輯。
3. 為什麼做這個？因為，無聊看到有人在說前端醜，就稍微試做了一下。

<h3>好，到底要加什麼？</h3>
→ 新建空白樣式表，命名為 Komica-MD-Driver，並匯入以下語法:
@-moz-document regexp("(http://|https://).*komica.org.*"), regexp("(http://|https://).*komica2.net.*") {
  @import url("https://cdnjs.cloudflare.com/ajax/libs/material-design-iconic-font/2.2.0/css/material-design-iconic-font.min.css");
  @import url("https://fonts.googleapis.com/earlyaccess/notosanstc.css");
