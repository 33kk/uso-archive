### Description
<b>NOTE: As of 01.01.2013 this style will no longer be supported.</b>
Replaces your default tumblr Home icon with a country related landmark icon, but only when you return home!
NOTE: The intent of this style is to serve as a tutorial example.
<a href="http://ohnemitleid.tumblr.com/post/36776405599" title="Basic Dropdown Option Style Tutorial" title="Basic Dropdown Option Style Tutorial">The published tutorial</a> attempts to provide <a href="http://ohnemitleid.tumblr.com/post/36776405599" title="Basic Dropdown Option Style Tutorial">information about how to create a style with dropdown options</a>.

### Update notes
The images for the style are embedded data URI, so the style shouldn't "disappear" unless tumblr makes one of its many revisons that may affect the style.
