### Description
Kiosk Mode - Disables Right Click context menu from Firefox 3 and 2.  Also Prevents users from accessing the Options menu item from the Tools menu.


Features:
+ Users cannot right-click on a web page - no menu pops up
+ Users cannot right-click on the main toolbar to customize toolbar buttons
+ Users cannot access the "Options..." menu (Windows only, I believe).
+ Increased security on Firefox.
+ Prevents tinkering and editing, viewing webpage sources, and changing custom Network / UI configurations....
+ Great for Kiosk Mode, Schools, Internet Cafe's, Kids computer, Public computers.....

Browser Versions:
+ Works on Firefox3
+ Works on Firefox2

Integrates with Style 7910

Additional Information:
+ Users will still be able to press key combination Alt + O to get the options popup window, but if they are novice users, they shouldn't know its there.
