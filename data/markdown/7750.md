### Description
Allows you to Customize the menu highlighting color of ALL menus.  When your mouse is hovering over a menu, it will be highlighted in a custom color.  Increases menu navigation and readability.
Once again, this style supports both Firefox2 and Firefox3 (Only Text color works... Background is not changed on Firefox 3)

** Thanks user 5181 for the similar code of your style 5095

Functions:
- Text color of the currently hovered menu will be changed. (Fx3 And Fx2)
- Background Color (the menu selector) will be changed to your own personalized color (This only works on Fx2)

Applies to:
- File, Edit, View, History, Bookmarks, Tools and Help menus
- Navigation Toolbar (example: right clicking on back button)
- Main Context Menu (Right clicking on any web page)
- Right-Clicking on a tab
- Right-Clicking on the Location Bar (address bar)
- Also Applies to EVERY single menu on any window of a popup or pane



Browser Versions:
- Works on Firefox 2
- Works on Firefox 3 (Only Text color works... Background is not changed on Firefox 3)

Integrates with Style 6489
