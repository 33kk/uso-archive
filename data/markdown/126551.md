### Description
<strong>Tomorrow</strong> is a simple dark color theme built upon <strong>Oscar</strong> front-end for Searx.</br>
This theme works with the new and the old (from Searx 0.8 to 14.0) instances.</br>
</br>

<strong> Deprecated </strong></br>
As the version 13 of Sears bought a official dark option this theme is not being maintained anymore. It can still be useful for instances of Search using an old version.</br>
