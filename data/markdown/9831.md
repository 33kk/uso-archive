### Description
MBlue Nav-Style Bookmarks-Bar Vista
--------------------------------------------------------------------
Description:
 1. Transforms the BOOKMARKS-BAR and make it similar to Vista.
 2. Transforms the TOOLBARBUTTONS and make it like the NAV-BUTTONS.
 3. Can use Separator/Spacer to customize the buttons (read below).
 4. Skins TAGSIFTER icon.
 5. Transforms the BOOKMARKS POPUPS and make them Nav-Style.
 6. Supports SMALL and BIG ICONS.
 7. Highly Customizable.
 8. Customizable BOOKMARKS-FOLDER ICONS (40 icons to choose from).
 9. Integrates with the other Nav-Style styles.
--------------------------------------------------------------------
Notes:
-Using other Styles/Extensions/Themes may change some margin/border settings, making the buttons not aligned. If so, try not using other styles that changes margins in the Bookmarks-bar, and use the default firefox theme. 

-Using other Styles/Extensions/Themes may interfere with the popups settings. If so, try not using other styles that changes the Bookmarks popups, and use the default firefox theme.

-Some Ext. Icons (like Download Helper) will not work correctly.

-Works in the Bookmarks-Bar only.

-Even if this style supports big icons, it works better with small ones, because the default big icons are too big to fit inside the buttons, so this style resizes them.
--------------------------------------------------------------------
How to Customize the Buttons:
READ THE INSTRUCTIONS INCLUDED!!!
--------------------------------------------------------------------
CUSTOMIZING:

 1. BOOKMARKS-BAR BACKGROUND-COLOR.
 2. BOOKMARKS-BUTTONS BACKGROUND-COLOR (HOVER).
 3. BOOKMARKS-BUTTONS (OPEN) + POPUPS BACKGROUND-COLOR.
 4. TEXT COLOR/SIZE/WEIGHT/OPACITY/ALIGN.
 5. BOOKMARKS ICONS OPACITY.
 6. NAV-STYLE ITEMS OPACITY.
 7. BOOKMARKS POPUPS TEXT COLOR/SIZE/WEIGHT/OPACITY/ALIGN.
 8. BOOKMARKS POPUPS ICONS OPACITY.
 9. BOOKMARKS POPUPS MAX-HEIGHT.
10. CUSTOMIZE BOOKMARKS FOLDERS ICONS (40 ICONS TO CHOOSE FROM).

> GO TO BOTTOM OF THE PAGE TO CUSTOMIZE
