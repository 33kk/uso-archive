### Description
Make Supremacy1914 games fair by using CSS to remove features Goldmark provides.<br>
Removed:
<ul>
    <li>Increase and decrease morale buttons on provinces and troops</li>
    <li>Production and build speed-up buttons</li>
    <li>Instant spy actions</li>
    <li>Buttons to open the Goldmark shops</li>
    <li>Button to buy resources using Goldmark in the market and the resource window</li>
    <li>Goldmark price display in support our troops window</li>
    <li>Popup window displaying goldmark ad</li>
</ul>
