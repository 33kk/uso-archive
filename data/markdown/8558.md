### Description
Allows you to Customize the text and background image of global tooltips.  

** Thanks for style 85 by user 12.  My style is entirely based on this code!!

Tooltips are the small informational popups that appear when you hover over certain buttons, links, and areas of web pages and the Firefox UI.

Features:
+ Background image will be a faded white Firefox logo with shadowing
+ Supports a huge list of Fonts (discussed below)
+ Image supplied in default code is hard-coded - allowing you to view the background, even when offline!
+ Background image is static and is centred on the tooltip.
+ Background color of tooltip (not affected by picture) will be white
+ Text color of the tooltip will be Black
+ Border of tooltip will be black
= Optional - Font Size may be customized (must uncomment)
= Optional - Bold text for easier readability (must un-comment)

Fonts Supplied (Supported):
- Currently, I supplied the following Fonts in the Code:
1. Segoe UI (User Interface of Vista)
2. Calibri (Text of Vista)
3. Verdana (XP Font)
4. Helvetica (XP)
5. Tahoma (XP User Interface Default Font)
6. Arial Global Font

Applies to:
+ Tooltip popup
+ Tool tip text
+ Show all History
+ Side Bar

Browser Versions:
- Works on Firefox 3 (ALL Themes - Full support!)
- Works on Firefox 2 (most themes)

Additional Information:
+ Feel free to edit the CSS.  You can customize the fonts, enable bolding, and change the background image, even change the font size!  No Re-posts without asking though!


Integrates with the following Styles:
