### Description
This style enable to change from Mastodon dark skin to light skin, it's like TweetDeck.<br>
It can change UI color only, so the original layout will be kept, you can add other layout change style!<br>
<br>
Mastodonを黒スキンからDeck風の白スキンに変更するスタイルです。<br>
このスタイルはUIの色のみを変えるもので、元々のレイアウトは保たれ、他のレイアウト変更スタイルを入れることができます。<br>
