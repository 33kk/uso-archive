### Description
A companion style for the FindBar Tweak extension for Fx25+. The style also works about the same if you don't use the extension: reduce the Mozilla-extreme height of the new findbar, move the close button back to the left side (where I'm used to looking for it), and add some color highlighting to the bar/buttons (to increase visibility).
