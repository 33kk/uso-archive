### Description
<p>If a link have a <code>title</code> attribute (tooltip text), this is shown after the link (blue).<br/>
<b>Ex.</b> this is a [link](http://reddit.com "hi") -> this is a <u>link</u> (hi)</p>

<p>Also, if a link doesn't have content, shows the url (green).<br/>
<b>Ex.</b> nothing here [](/nope "hidden") -> nothing here <u>/nope</u> (hidden)</p>
