### Description
Forum skin for the Bioware Social Network http://social.bioware.com/forum/

Chrome and Firefox 3.5+ only!

This takes the base from http://userstyles.org/styles/22783/kinointerfacetweaks KinoInterfaceTweaks, and runs with it.
This theme removes: signatures in posts, list of moderators, sub descriptions for forums, background images, etc. It also condenses the header and aligns the page numbers and forum breadcrumbs. It also extends the text to fill more of the box in posts and in the forum home.

To show properly,  go to Profile then Edit Profile: Personal Information. Under Site Theme, select "Default (Bioware).
