### Description
Menampilkan logo Unhas dan avatar Anda di KRS.
<pre>
------------------------------Untuk foto sendiri:---------------------------------
1. Scroll halaman ini sampai ketemu "Advanced Style Settings"
2. Upload foto Anda
	ATAU
1. Seret foto Anda ke jendela browser
2. Copy url-nya
3. Paste ke kotak "Enter url" di Advanced Style Settings)

---------------------------------Untuk nge-print:----------------------------------
Firefox
	1. Open Menu > Print atau pencet tombol "Alt" > File > Print Preview
	2. Page Setup
	3. Print Background (colours and Images)
Chrome
	1. Klik kanan > print
	2. More options > Background graphics
</pre>
