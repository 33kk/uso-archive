### Description
A nice minimalistic theme for Firefox inspired by Microsoft Edge. You can change your homepage motto before installing the style.

### Update notes
<sup><b>to get the full experience:</b></sup>
● Install <b><a style="color:blue;" href="https://fonts.google.com/download?family=Lobster"> Lobster Font</a></b>
● Install <b><a style="color:blue;" href="https://drive.google.com/file/d/0B7FMn1Q81klyblJQdV9rUG9SdVU/view?usp=sharing">CTR</a></b>
● Import <b><a style="color:blue;" href="https://drive.google.com/file/d/0B7FMn1Q81klyblJQdV9rUG9SdVU/view?usp=sharing"> These settings</a></b> to CTR
  ● For <b>Bookmark Folder icons</b> you can just use <b><a style="color:blue;" href="http://emojipedia.org/">EmoJis</a></b> as bookmark folder names.
  ● If your <b>Homepage</b> is set to another page, go to "about:preferences" and change it back to "about:home"
  ● To rearrange the bookmarks and search bar's placement, simply use the  <b>Customize</b> option on the menu.
