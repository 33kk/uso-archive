### Description
This script changes the small images around InsaneJournal to a more aesthetically pleasing set of icons. This script customizes the front page, comment/post pages, userinfo/profile page, explore page, update/edit page, and many more places around InsaneJournal that have small icons.

All icons used are by Pinvoke.org and FamFamFam.com.
