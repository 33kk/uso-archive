### Description
<p>This style drastically changes the appearance of FIMFiction and makes some features inaccessible!  Also, I have only tested this as an anonymous reader (no user account). FIMFiction has a creeping featurism that distracts me from reading and finding stories.</p>
<p>Here is a summary of what it does:</p>
<ul>
<li>Hides front page popular stories.</li>
<li>Hides all background images.</li>
<li>Hides most story statistics (like the views and comments counters).</li>
<li>Hides avatars.</li>
<li>Removes text shadows.</li>
<li>Expands hidden chapters.</li>
<li>Hides chapter release dates and word counters.</li>
<li>Hides the sidebar on chapter list pages.</li>
<li>Removes the gaudy hover animation on list items.</li>
<li>Hides Facebook Like links.</li>
<li>Hides advertisements.</li>
<li>Tweaks the story summary box.</li>
<li>Hides comment thumbs-up, thumps-down, and permalink buttons.</li>
<li>Hides the footer.</li>
</ul>
