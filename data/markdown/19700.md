### Description
Converts MediaWiki's <source> element syntax highlighting from what I assume is some Linux colour scheme to Visual Studio's. <em>Should</em> work across all programming languages, but drop a review in if you spot something missing.

The only thing this style can't fix is lines starting with # characters being highlighted like comments.
