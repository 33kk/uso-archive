### Description
This is a Canta dark theme for Tweetdeck. It follows Vince Liuice's Canta Dark GTK theme color scheme.

· I fixed column width to 450px. You can revert to default size by adjusting, commenting out or deleting the first rule (html.dark .is-wide-columns .column).
· Scrollbar choice :
styled: keep as is
native: delete both .column-scroller rules
none: uncomment rule 2 and delete rule 3
