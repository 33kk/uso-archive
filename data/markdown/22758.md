### Description
Style changes the appearance of the following: 
- Bright, highly visible selection color (global). 
- highly visible selection color for the tags <pre> and <code>. 
- Visible active links (clicked) 
- Highlight active form field, (eg Google search) 
- Change default design of system buttons

This style working global (for all sites)

Installation:

Preferences → Advanced → Cotent → Style options → My style sheet - here place path to your .css file
