### Description
No borders, no circles! this css is intended for profiles that use transparent pngs, normally if someone uses a transparent png image, when entering the user's profile you see a background, this css fixes that problem by removing the background. works on mobile twitter, web client and tweetdeck (this is my first css sorry)
(web client based on: https://userstyles.org/styles/145712/twitter-back-to-square )
