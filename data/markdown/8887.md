### Description
When a page is loading, Tab Scope's Stop Button will be a Throbber instead of the stop button.
Thanks to LouCypher for supplying the code to do this to the regular firefox stop button

Features:
+ Stop button animates the loading for you!
+ Increases page loading visual notification

Integrates with the following Styles:
+ Style 8803
+ Style 8635
