### Description
A dark and comfy userstyle for 4chon.net. Has a fixed postbox and optional mascots!

For the full experience install 4chon X and reply popup script.

4chon X http://userscripts.org/scripts/show/101133

reply popup script http://userscripts.org/scripts/show/110735


Features
- Styled wiki
- Fixed and styled postbox
- Fixed page navigation
- Supports HTTPS
- Support 4chon X and reply popup script

Tested on Firefox 8 and Google Chrome.

Opera not yet supported, working on it.

**Known Bugs**
- Checked checkboxes do not appear to be checked in Google Chrome. (It still works though.)
