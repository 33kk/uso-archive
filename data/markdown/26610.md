### Description
===================================================

Makes advanced search visible on all pages.

===================================================

Optional:

If you want to make it only visible on browse pages, edit style and change "domain(freshon.tv)" to "url-prefix(http://freshon.tv/browse.php)"

To make it visible on some additional pages (but not all), you can add other url-prefixes separated by commas, eg "url-prefix(http://freshon.tv/browse.php), url-prefix(http://freshon.tv/details.php)". Any page that has a URL that starts with one of those prefixes will show the advanced search.
