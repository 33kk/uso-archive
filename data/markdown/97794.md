### Description
Keeps the Bugzilla header fixed while the bug description or bug list gets scrolled.
The footer is only needed for the [My Bugs] link missing from the header.
The footer only becomes visible when you leave the bug body.
To access [My Bugs] leave the browser window and enter the footer from the side or below.
