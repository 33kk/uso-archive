### Description
This CSS requires WebMail Notifier extension for Firefox: https://addons.mozilla.org/pl/firefox/addon/4490
It replaces default statusbar and toolbar icon with prettier one. Also the CSS customizes font size and text alignment of toolbar icon's mail counter. Updated for WMN version 2.*
