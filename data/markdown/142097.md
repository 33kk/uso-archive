### Description
A dark gray theme for <a href="https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne" style="color:#009999">Stylus</a>. Stylish is not supported. Intended for use with the default CodeMirror theme. There are advanced options for a compact layout with 2, 3 or 4 columns. The compact layout works with the new default UI, so the classic UI will remain the same if selected.
