### Description
Allows you to Customize the hover-color of the toolbar buttons.  When your mouse is hovering over your bookmarks toolbar icons, they will be highlighted in a different custom color.  Increases screen navigation and readability.  This style customizes the button color based on its function (if clicked, it will turn green...)
Once again, this style supports both Firefox2 and Firefox3.  Fully Customizable colors - simply edit the HEX codes.

Functions / Features:
- Default Button Text Color - Black
- Hover Button Color - Blue
- When Clicking a Button - Green (Dark)
- Hovering / Display of a Disabled Button - Grey (Dark)
- Makes your favorite browser very colorful and appealing... almost fun!

Applies to:
- Bookmarks Toolbar (text of your bookmarks)
- Navigation Toolbar (if your toolbar buttons have text - Right Click / Customize and Select "Icons & Text")
- Main Toolbar (Text of custom or Extension's buttons)
- Also Applies to EVERY single toolbar button on any window of a popup or pane (Stylish toolbar buttons in the Manage Styles window, for example)

Browser Versions:
- Works on Firefox 3
- Works on Firefox 2

Integrates with Style 7719
