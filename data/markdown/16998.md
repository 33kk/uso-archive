### Description
This user style changes the syntax highlighting of Heroku Garden to match that of Mozilla's new Bespin web-code-editor.
More styles (Vibrant Ink, etc...) are on the way!
Don't change the font size if you tweak this style, because Heroku's code editor seems to have the size of the fonts hard-coded and the cursor will not line up with the text you see on screen.
