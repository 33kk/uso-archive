### Description
For Firefox 4+ and latest Firefox Beta, Aurora, and Nightly builds.

This moves the Connection Status Display popup messages to the top of screen on top of web page content. Idea from (bug 629915 - Change status info overlay to display at top of screen on top of content)

Tested on Firefox 4+ and up to the latest Firefox Beta, Aurora, Nightly builds.
