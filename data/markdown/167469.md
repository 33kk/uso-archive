### Description
Dark mode/Night mode for the basic theme on snahp.it forums.

For this theme to work properly, you first need to change your theme on your snahp.it profile. To do this, go to 'User Control Panel' > 'Board Preferences' > 'Edit Global Settings' and set 'My board style' to 'Basic'.
