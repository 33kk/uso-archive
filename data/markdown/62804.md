### Description
Brings back the "classic style" in blue, to the new Google Calendar.
Only works ok if you are using one calendar and select the blue color to it.
If you have more calendars, in other colors, only the border color will change.
For best effect turn off "Event dimming" in the Google calendar settings.
See this as a "work in progress"

I'm using bits from Jyri Tuulos: Google Calendar - Better 2011 style
http://userstyles.org/styles/50283/google-calendar-better-2011-style
