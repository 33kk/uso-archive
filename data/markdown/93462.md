### Description
Firefox theme that could make your firefox way more personal than ever!
This theme is actually my updated Firefox Candybar 2 theme with more minimalist and metro-esque approach.

First thing first you need to install Stylish add-on for Firefox
Make sure you do have Segoe UI font as this theme using it

The instructions on how to change the user picture and username are inside the script.

Works best with:
Metro Panoramic - http://userstyles.org/styles/93463/metro-panoramic?r=1380354270 by Pullipo (Myself)
Square Corner Tab - http://userstyles.org/styles/78804/square-corner-tab by wasurenai
Windows 8/8.1 Theme for Windows 7 - http://mare-m.deviantart.com/art/Windows-8-8-1-Theme-for-Windows-7-318932206 by mare-m
