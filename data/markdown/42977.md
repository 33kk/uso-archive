### Description
I can be found on Tumblr @ http://kaleidostar.tumblr.com

I created this theme on Mozilla Firefox, so I have no idea how it looks on Chrome.

[[ To Install ]]
01. Download Stylish
     https://addons.mozilla.org/en-US/firefox/addon/2108/
02. Click 'Install with Stylish' on the box on the right.
03. Sit back and look at the pretty.

Keep in mind that this theme does some goofy things to things I haven't been able to change, yet.  Like the follow/dashboard links on profiles and the text boxes for posting.  If you want to take off the "small caps" edit the code and take out: font-variant: small-caps;.
