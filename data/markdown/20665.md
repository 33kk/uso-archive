### Description
This Style will remove the Wikipedia logo from the upper left corner and replace it with a reduced list of languages. 
To each language a country flags is added to ease switching b/w languages.


This style requires you to use the new Wikipedia beta user interface. Which can be enabled on the Wikipedia site.

For the traditional user interface use this style: http://userstyles.org/styles/19102
