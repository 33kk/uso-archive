### Description
Slack doesn't support narrow windows. This hides the sidebar, always or when needed. Thanks to <code>0atman</code> and Robin.

### Update notes
<strong>
    Latest update (21 March 2018):
</strong>
Always wrap words, not just when sidebar is collapsed.

<strong>
    Tips:
</strong>
By default, this stylesheet hides the sidebar when the window is narrower than 600 CSS pixels. You can also choose to leave it always hidden.

To navigate when the sidebar is invisible you can use keyboard shortcuts; type <code>/keys</code> for a list of those.
