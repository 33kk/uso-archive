### Description
<strong>Ecosia dark theme - verified and working 2020</strong>
A dark theme for the Ecosia search engine & services. Uses lightweight CSS to ensure a seamless dark theme experience.

<strong>Compatibility</strong>
 • Tested on browsers: Firefox, Chrome, Edge, Opera
 • Works with Ecosia mobile-mode
 • Tested on 'Stylish' and 'Stylus' extensions

<strong>Currently works with the following Ecosia sites:</strong>
 • www.ecosia.org
 • info.ecosia.org
 • blog.ecosia.org

<i>Planned development:</i>
 • ecosia.teemill.com

<strong>Any issues, let me know and the theme will be updated ASAP - just ensure you are using the latest version at all times</strong>

<i>Important note: To turn on/off Ecosia advertisements in the search engine, see the 'Customize Settings' button found next to the installation button</i>
