### Description
Hides ads in the Promotions tab.

### Update notes
The options allow you to display the ads when hovering over the conversation list's top border and control the border and background colors.

The default values for the border colors should match Gmail's defaults with the hover hint color using the Promotion tab's top border color. (At least for the default theme.)
