### Description
An all in one style for your web apps.
If you use a complete download solution for movies, tv, & music... This theme works for all of the support apps used to download and manage .nzb & torrents. All themed for a integrated look and feel.

I use ProxMox for 2 Linux servers (VM's) to run all of these apps to support my movie, tv & music libraries.
