### Description
Spin the button at mouse click.<br>
If you own FDM 3D printer, please try printing <a href="https://www.thingiverse.com/thing:2438892">my spinner</a>.
<p>
Caution<br>
Do not place the toolbar button on the left neighbor of the overflow button.<br>
When the toolbar button is placed on the left neighbor of the overflow button, it unintentionally moves to the overflow menu when the toolbar button spins.<br>
<p>
About the panel twitching<br>
In Firefox 57, if you spin when hovering the toolbar button, the behavior becomes unstable, so in this style, I set it to spin when clicking the toolbar button.<br>
When you click the toolbar button, the panel twitches while spinning, but if the problem of Firefox 57 is fixed, I will rewrite this style.<br>
