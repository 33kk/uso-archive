### Description
Sets the UI of Firefox to transparent. That way, they are the same color as the current Windows 8 / Windows 10 color theme (even for auto color). Adjusts the transparency of the active tab too. Should also work for Windows 7 (including Aero Glass).
Tags: Win8, Win8.1, Win7, Win10
