### Description
- Customize Font (read Additional Info).
- Resize header.
- Justify paragraph
- Add pre-wrap and background to code.
- Display blockquotes as blocks.
- Change list bullet type

### Update notes
- To use google web fonts, go to links like 'https://fonts.googleapis.com/css?family=Asar|Lato|Playfair+Display|Slabo+27px', then append to this style later. Note that Unicode range is not supported,
- The fonts must be installed on your device or specified in "Web Font Source", except sans-serif, serif, monospace, etc.
- The fonts name should be quoted in '', except sans-serif, serif, monospace, etc.
