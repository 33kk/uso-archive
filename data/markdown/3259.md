### Description
This is a slightly more radical reversion of the eBay search formatting to an earlier state.  The left-hand sidebar menu (category browser) is narrower with original font size and the annoying animated ad spaces on top and left sides have been vanished.  Grayed-out text (subtitles etc) is returned to solid black. Alternating item list rows are shaded and links are underlined.
------------
30 Aug 2007: To eliminate the bright yellow background on the sidebar and top searchbox, add these lines to your style:

#matchcats .matchcatecontent {background-color: #efefef !important;}
#basicsearch div {background-color: #fff !important;}
#buyingguides .buyingguidescontent {background-color: #efefef !important;}
#searchoptions #srchOpsDiv {background-color: #efefef !important;}
#productfinder {background-color: #efefef !important;}

------------
04 Sept 2007: To eliminate the yellow searchbar gradient, add these lines:

#searchtabs {background-image: none !important; list-style-image: none !important; background-color: #fff !important;}
#basicsearch div {border: 1px solid #ccc !important; background-color: #efefef !important;}
#basicsearch div input, #basicsearch div input div {background-color: #fff !important;}
#find label {text-transform: uppercase !important; font-weight: bold !important;}
.r2-nav {background-image: none !important; background-color: #fff !important; border-color: #fff !important;}
.r2-panelHeader {display: none !important;}
.srchBox, .srchBoxE, .srchBoxW {background-image: none !important;}
#expHigh h3 {background-color: #efefef !important;}

------------

2 Nov 2007: I've eliminated the Guides and Neighborhood boxes on the left-hand column that have inexplicably been placed above search refinement boxes.  To get those back, remove the final two CSS commands.
