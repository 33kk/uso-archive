### Description
Theme with background and different block colors support for <a href='//fb.com'>facebook.com</a> and <a href='//messenger.com'>messenger.com</a>.

Compatible with the latest versions of browsers <a href='//google.com/chrome/browser/'>Chrome</a> and <a href='//mozilla.org/firefox/new/'>Firefox</a>.

If you have any problems with the style or you have found an error, feel free to write me -  <a href="//m.me/489734454703065">link</a>.

Do not forget to update the style or use the extension with auto updates - <a href="//add0n.com/stylus.html">link</a>, <a href="//github.com/openstyles/stylus">src</a>.

If after install you don't see any changes check your in browser support the CSS var() feature - <a href='//fiddle.jshell.net/dv1kuo3h/show'>link</a>.
