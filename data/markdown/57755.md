### Description
An effort to get back the older look of Gmail.
<b>What does this UserStyle do?</b>
<ul>
<li>Changes new Gmail style to the older one</li>
<li>Reverts the top navigation in Gmail to the classic one</li>
<li>Implements classic Gray-Blue Gmail theme</li>
</ul>
<b>Version 3.2 features and Changelog</b>
<ul>
<li>Made things more tidier than before, fixed almost everything wrong in v2.0</li>
<li>Re-fixed Chat and Mail list scrolling</li>
<li>Re-styled buttons</li>
<li>Re-styled the Menu bar</li>
<li>Re-fixed no-scrolling of Gmail in Version 3.2.2</li>
</ul>
