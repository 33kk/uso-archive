### Description
主要说明：
1. 针对常用百度页面进行简化和修改，已完成页面可见下方“已完成的工作”；
2. 元素风格以圆角矩形为主、阴影为辅，方党退避；
3. 网页查询的结果为单栏居中，其它因页面结构原因放弃居中；
4. 若页面中屏蔽了你需要的元素，请告知；
5. 特殊样式难以及时适配，你可以加QQ群：912825034 进行反馈。

已完成的工作：
1. 百度首页｜百度知道首页 | 产品大全页
2. 百度网页结果页｜知道结果页｜文库结果页｜资讯结果页｜百度汉语结果页
3. 知道具体页｜文库具体页 ｜ 百科具体页
4. 部分页面的字体略放大
半完成的页面：
1. 图片结果页
2. 百度翻译页面

接下去会做的工作：
1. 百度网页结果更多特殊情况适配
2. 百度百科更多情况适配
3. 优化样式
4. 百度经验

“自用”样式，可能有不兼容，但你通过以下途径找到我提出bug或建议：
