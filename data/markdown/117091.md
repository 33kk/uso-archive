### Description
For folks who have imgur but not the preview link blocked at work OR for folks who use browsers which display the images at a fixed size but scale the text up. This will double (relatively) the size of the reddit thumbnails.

Good starting values:
Width (in px) - 140
Height (in px) - 140
Scale - 1.8
