### Update notes

2019-02-04:
- Added "Active in the last X hours" to the list of text that should be white
- Updated the default colors to match tumblr's new theme. For reference, the old theme's default colors were: Header: #263141 // Background: #2B384A // Message Bubble: #FFFFFF // Send Button: #529ECC
