### Description
This changes the lightbox background colour on Flickr to be pure black (#000000) rather than dark grey.

Currently (26/06/10) you can only use the lightbox function of Flickr if you are logged in. (see here: http://blog.flickr.net/en/2010/06/23/a-new-photo-experience-your-photos-happier/ )
