### Description
A Material Dark Theme for Facebook. Based on VSCode/Jetbrains Material theme.

<h3>IMPORTANT</h3>
<ul>
<li>- You must have "new" Facebook enabled. (<a href="https://www.pocket-lint.com/apps/news/facebook/150761-the-new-facebook-update-how-the-all-new-design-looks-and-works" target="_blank">See how</a>)</li>
<li>- You need to activate "Dark mode" (See link just above)</li>
<li>- Doesn't work on messenger currently</li>
</ul>


<strong><a href="https://github.com/CharlieEtienne/material-facebook" target="_blank">Github Repo</a></strong>

This theme is SUPER easy to customize ; you just need to change hue and/or saturation to get your own dark nuance.

Custom accent doesn't work everywhere for now. You can uncomment to test it but it won't apply everywhere.

If your eyes like it, I did the same on some other well-known websites with the same theme:
