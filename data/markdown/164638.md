### Description
A Minimal theme for LinkedIn.
If you use LinkedIn on a daily basis and want to read posts better this is a clean theme that makes Linkedin enjoyable for you!
You can use this theme for LinkedIn with Stylish or <a href="http://add0n.com/stylus.html">Stylus</a> extension in any browser.
