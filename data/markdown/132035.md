### Description
Reduces the navigation bar height and restyles the forward- and back button.

### Update notes
This userstyle is a result of a great combination between two existing userstyles.

Firefox 29+: Slim toolbar for Australis (https://userstyles.org/styles/111601/firefox-29-slim-toolbar-for-australis)
UI - Reduced navbar height + small back button (https://userstyles.org/styles/109643/ui-reduced-navbar-height-small-back-button)

I delegate all the credits to them user 287362 and user 167700

[30/08/2016] Added forward button visibility option
"Hidden" option will only show the forward button if forward is possible
"Visible" option will always show the forward button
