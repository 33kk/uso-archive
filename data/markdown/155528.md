### Description
A customizable theme for rbxtrading.com, allowing you to change the background color/image, header color, text color, button colors and many more! Works in conjunction with the built in RBXT themes, so choose both theme settings wisely. Screenshots are using the built in Orange theme.
Made by NeloBlivion, credit to Samtasm (https://userstyles.org/users/344952) for the original theme which I used as the base for this.
The full guide can be found here: https://www.rbxtrading.com/t/the-first-unofficial-rbxtrading-customizable-theme/161862
