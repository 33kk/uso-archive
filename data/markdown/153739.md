### Description
This theme is specifically designed to match the dark mode used by Firefox's inspector and browser toolbox (background colors, text colors in the editor, fonts etc.).  Additionally, it is meant to compliment [ShadowFox](https://github.com/overdodactyl/ShadowFox) - a universal dark theme for Firefox.
