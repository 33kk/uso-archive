### Description
For people having dificulties to digest the default Trello style in year 201x.

To use the theme, apply stylesheet from this repo in custom CSS management extension of your choice (tested on ControlFreak and Stylish for Chrome). Designed for <strong>4-column</strong> boards only (<em>might not be suitable if you're participating in multi-project/user environment</em>).

<a href="https://github.com/pentago/trello-air-theme">Contributions welcome</a>, requests are not.
