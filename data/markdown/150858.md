### Description
Dark theme for the eldora menu sites. Compatible with the "Eldora - Compact, Fullscreen, No clutter" style.
<br /><br />
By default this style applies to all eldora menu sites with the format www.eldora-[COMPANY].ch
<br />
Examples: eldora-suva.ch, eldora-csgastronomy.ch, eldora-roche.ch
