### Description
Compresses the Firefox 3 Awesome Bar entries.  Allows you to see more suggestions, in less of a screen area.
Works for both Firefox 3 and Firefox 2.
Allows you to use the Most Possible area for your Location Bar or the Awesome Bar in Fx3

Features:
+ Smaller Autocomplete Text
+ No Padding
+ Still Easy to read
+ Greatly improved screen space
+ More of a Blue color for links, than the original Awesome Bar's hyperlink text

Integrates with the following Styles:
+ Style 8027
+ Style 7723

Works on:
-Firefox2 (LocationBar is somewhat changed)
-Firefox3 (AwesomeBar is greatly altered)
