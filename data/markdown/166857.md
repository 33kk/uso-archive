### Description
Choose your own background color for Google Keep.

 - Only affects background color behind the notes; not the notes themselves.
 - Only applies to Google Keep.
 - Defaults to a "red wine" background
 - Sidebar text color automatically adjusts to contrast with your chosen background color

Click "Customize Settings" gear icon above to set your own color.

### Update notes
Update May 2020:
 - Removed option to set custom text color. Text color will now use CSS "invert" function to automatically contrast with your chosen background color

Update Feb 2019:
 - option to add a small border around the note labels (I find this easier to see)
 - colored and bordered the box for adding labels to notes
