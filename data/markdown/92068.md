### Description
This will replace Trihex's sub icon with the 4 eggs seen in the screenshot. The icon was made by Asbrandt so I just whipped up this quick script since it's better than a black star ;)
This can be used with the yoshi mod icon for MAXIMUM SWAG: http://userstyles.org/styles/92084/

NOTE: If you have the Twitch - Dark / Black / Customisable Theme installed, you must remove these lines in the dark twitch theme code for this icon to show:
"span.tag.subscriber.trihex > a,
span.tag.subscriber.ctrihex > a{
    background-image: url("http://herpderp.com/twitch/trisub.png")!important;
}"
