### Description
This playmoss theme features a damask lace-like patterned background in pink & grey. It features dashed borders, larger text, rounded icons and a pink, grey and white color palette. This theme is not recommended for those with contrast issues, but I think the visibility of the theme is otherwise good. Everything on the code is commented for complete customization if the contrast is not sufficient.
