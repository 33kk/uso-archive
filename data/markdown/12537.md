=====  Remarks for me (and maybe you also) ======
To have more compress style but keep preferment, I suggest:

- Install Compact Menu 2 (https://addons.mozilla.org/en-US/firefox/addon/4550) and always hide menu bar
- Follow style 13328 or my non published style which better for small icon (see it on the bottom)
- Install Custom Button 2 (https://addons.mozilla.org/en-US/firefox/addon/5066) 
- Install Show/Hide Toolbars Custom Button I made (http://custombuttons2.com/en/forum/buttons/buttons-database/showhide-toolbars-bookmark-tab-status-bar.html) and put it on both side of navigation bar
- Double right click or shift-right click to edit one of the buttons, in "Initialization code" tab, add 

/* ========== Initialization code ========== */

document.getElementById("personal-bookmarks").flex=0;

/* ========== Initialization code ========== */

On the top of script so that the Bookmarks Toolbar Items will not be flexible

- Put all useful buttons on bookmark bar
- Enjoy!
