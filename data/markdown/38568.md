### Description
Tries to improve slightly the usability...

Main changes :

- Bigger search areas
- Ads removed
- Content area larger
- Blocks more clearly separated
- Background a little bit lighter to facilitate reading
- Lines under some links removed to facilitate reading
- Various small tweaks...

The *Ergo collection aims at improving usability without touching what makes the « soul » of websites. So no radical changes, website is still recognizable at first sight. You can find more *Ergo styles by clicking on my name up there.
