### Description
Facebook shows the partial information that is not interested. For example, all the polls, or a friend is friends with someone.

What messages should not be displayed, you can adjust it yourself.

### Update notes
=== LEGEND ===

Add Friend: a friend is friends with someone
Add Experience: all changes in someones profile like job, educaion etc.
Generic Edge: all likes for fanpages etc.
Games: for example highscores in flash games
Discover Platform Application: for example spotify
Tagged In Checkin: someone marked its position for example restaurant
Vote Question Poll: all the pollings
