### Description
This style tries to recreate "Safari-styled" tabs.

Features:
1. "Safari"-like tabs
2. Better look of close button (it takes the one from installed Firefox theme replacing existing (under Linux ugly GTK) icon.)
3. Fix for Faviconize Tab extension where even with "Don't hide close button" feature turned on inactive tabs would lose close buttons (see before & after screenshots)

Compatibility: Windows/Linux, Firefox 3.X (tested).


Many thanks to user 15521 for the inspiration ( style 13360 , style 13386 , style 13387).
