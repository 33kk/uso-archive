### Update notes
Поддержка Куклоскрипта. В настройках:
- «Общее» → «Стиль скрипта» → «Solid grey»
- «Форма» → «Кнопки разметки текста» → «Стандарт.»
- «Картинки» → «Раскрывать картинки по клику» → «В посте»
- «Посты» → «Кнопки постов» → «Solid grey»
При включённой опции «Открывать текстовые спойлеры» (либо для активирования данной функции без Куклоскрипта) также включить перед установкой соответствующую опцию для корректного раскрытия спойлеров.
Применять со стандартным стилем Futaba.
