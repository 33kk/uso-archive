### Description
<b>You need the browser extension "Stylus" to use this theme!</b> 
<small>(Currently available on desktop Chrome, Opera, Firefox and Firefox Android)</small>

A dark version of the web UI (https://blockchaincuties.co) with additional indicators and some fixes

<b>Features:</b>
<ul>
<li>- cooldown indicator colors from red to green</li>
<li>- many hover effects added for better user experience</li>
<li>- changed (not all yet) icon pictures to font awesome icons for better scaling and coloring</li>
<li>- made some buttons more rectangle with smaller border radius</li>
<li>- adventure report results are more visible and centered</li>
<li>- adventure reports are a bit more dynamic with some animations ;)</li>
<li>- custom picture option for the owner pic on the cutie's page and leaderboards</li>
<li>- hidden discord / telegram icons</li>
<li>- hidden adventure cooldown on siring market</li>
<li>- removed the "Buy now" button from items under "My market"</li>
</ul>
