### Description
This style create space on tab bar when firefox maximized.
You can grab the window when it maximized, like Opera.
This space help you grab window, open context menu and more.

Operaみたいに最大化時にタブバーの上にスペースを作ります。
これがあるとウィンドウ最大化時にもタイトルバーを掴んでウィンドウを移動させたり、
タブバーの上を右クリックすることでウィンドウのコンテキストメニューが開けたりします。
英語はそのうち直します。
