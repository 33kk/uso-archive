### Description
- Optimiert für 1680*1050-Bildschirme.
- Enfernt das neue Layout.
- Entfernt den kompletten rechten Container.
- Verkleinert die Top-News und ordnet sie ins Layout ein.
- Stellt die News wieder mittig dar.

Anpassungen der Containerbreiten können per Hand im CSS vorgenommen werden:
Einfach die Einträge mit 1280 bzw. 1240 auf die gewünschte Breite setzen.
