### Description
Removes all pictures included in the logs from the "Printer Friendly" view of the cache description page.
So one can turn it on or off just as needed.

To remove ALL pictures from the cache description in the "Printer Friendly" view use style 12297 .

2008-11-24: Little update to exclude the effect of removing the picture from the "show all logs" view.
