### Description
According with this post on the Sophos weblog, a norwegian researcher have pointed out that a data URI could be used in phishing attach. 
http://nakedsecurity.sophos.com/2012/08/31/phishing-without-a-webpage-researcher-reveals-how-a-link-itself-can-be-malicious/
This style change the color of the Firefox's urlbar (default is red=#f00) on a data link to alert to be careful on these links.
This style could not be applied via userChrome.css, by default Firefox not allow to detect the url to use with styles.
Update: added code to show a red hand when mouse is over a data link in pages.
