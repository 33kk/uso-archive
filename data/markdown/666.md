### Description
Replacement for the WellRounded extension (https://addons.mozilla.org/firefox/933/ ) by Lee Netherton.

This is style 665 updated to work in Firefox 2
See style 1079 for one that works in Thunderbird

Improvements:
Find in page box doesn't screw up when search box removed
Rounds all textboxes and menulists (experimental, please report any errors)
Fixed for new url bar and search box

This is a premliminary implementation and I welcome comments and suggestions for how to deal with the new go button and search bar (especially notification of search plugins).

Recent fixes:
2006-11-19:
Added support for Microsummary Buddy Extension
Improved urlbar buttons code (feed, security and safe browsing icons)
