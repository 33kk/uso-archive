### Description
Displays tab name (website page title) upon tab hover. Designed for larger numbers of tabs and darker background themes. With style applied, Firefox system tooltips are disabled for tabs only. 

Screenshot shows Firefox system tooltip and the same hover with a Quicktip. NOTE: Additional information regarding some style variance and other notes available in the More Info section (below).

Design based upon previous similar style for Bookmark Quicktip. http://userstyles.org/styles/48331/ff4-bookmark-quicktip
