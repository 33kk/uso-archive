### Description
Makes About:Config Page display a background image of your choice.  
About Config allows you to change Custom configurations to your browser and Configure Arguments.  Another excellent way to implement a new UI feature for your favorite web browser.

* Thanks to user 11487 for style 8167

Integrates with Style 8932.

Features:
- Appears on about:Config
- Fully customizable style, for your own enjoyment.

Additional Information:
This style works on ALL builds of the browser, so whether you are in the Default update channel, or have ventured into the Betas or Nightly Build channels, this style will work for you.

Firefox Browser Versions:
+ This style works on Firefox 3
+ This style also works on Firefox 2

Instructions:
In the CSS, change the website image I provided to the image you wish to display
You may also change the background HEX color to match your image.  The background color of text in this style has not been changed.
