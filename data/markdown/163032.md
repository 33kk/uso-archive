### Description
This is my Tree Style Tabs theme to match the Noia grey gradient theme, although it works fine with a standard light theme.
Colours unread/unloaded/active/etc.

The style originally started off based on this <a href='https://www.reddit.com/r/FirefoxCSS/comments/7emhsq/my_compact_treestyletab_css_and_sidebar_hover/'>treestyletab css and sidebar hover</a> style which I customised to my own colour scheme and preferences (without sidebar hover)
