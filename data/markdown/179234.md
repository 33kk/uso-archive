### Description
Theme for the ConnextBox Cable Modem Router as provided by UPC / Ziggo (among others).

&bullet; Replaces the plain-text login-textfield with a password-textfield like one (i.e. non-visible passwords !)
&bullet; Enables the Hidden 'Check Router Status' link at the Login screen
&bullet; Some subtle styling of links, removing the default text-underline unless hovering

Note: If you don't use the router's default IP address of 192.168.178.1 you should replace this in the CSS-code to the correct IP address for your setup.
