### Description
This is an interface tweak for the forum at http://social.bioware.com/forum/

1) It removes unnecessary clutter : Rss button, Portrait, Menu, DA ad, moderators list, etc.
2) It increases available page width for content. (by ~20%)
3) It increases text size and changed formatting for forum titles, topic titles, and posts to increase readability.
4) It increases the number of forum titles that can be displayed vertically (by 100%+, doubles it)
5) It increases the number of topic titles that can be displayed vertically (by 50%+)
6) It adds alternating background colors for topic titles
7) It reduces the size of player portraits on the right side of topic titles

This forum tweak will be updated on a regular basis. If you install it using Firefox "Stylish" plugin, you will automatically be updated to the latest version automaticly.

feedback & suggestions are welcome.
