### Description
Colonne de droite trop chargée ? Pas besoin d'avoir le cadre des articles les plus populaires ? Ce thème masque le contenu de ces catégories et ne laisse que leur titre (ou une petite bande pour le menu) apparent : passez la souris dessus, et le contenu apparaitra sur demande. Le reste du temps, c'est caché, et la page parait plus épurée. Il est aussi possible de supprimer purement et simplement l'instagram (sélectionner l'option avant d'installer le thème).


NB : si vous êtes dans le cas où les titres de Soompy sont invisibles et le menu tout mélangé, ce thème ne peut rien pour vous, et n'est responsable de rien : le problème vient certainement d'un module de protection type Ghostery (c'est ce que j'ai pour Firefox). White-listez y Google Ajax search api (me demandez pas pourquoi) et l'affichage se rétablira.
