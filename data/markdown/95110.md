### Update notes
<strong>Description</strong>
* Brings back Google search tools to left side.
* Displays all action menu items ("Cached", "Similar", etc.).
* Adds underline to search result title.
* Adds highlight search keywords.

<strong>What's New</strong>
Latest Update: Jun 27, 2015
* [Fix] Minor bugs fixed.
* [Add] Added a option to display checkmarks on the sidebar.
* [Remove] Removed "Display all tabs" for horizontal tabs because Google changed the menu structure and it is now difficult to display tabs through pure CSS.
