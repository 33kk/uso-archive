### Description
Set the Markdown input area as monospace in some known sites, including <a href="https://sourceforge.net/">SourceForge</a>, <a href="https://forum.userstyles.org/">UserStyles Forum</a>, <a href="https://www.jetbrains.com/youtrack/">YouTrack</a>, and <a href="https://simplenote.com/">Simplenote</a>.

Also set input area on paste sites as monospace.

### Update notes
Uses browser default monospaced font. For better font, <a href="http://www.howtogeek.com/208552/how-to-change-the-default-fonts-in-your-web-browser/">change your browser default fonts</a>. For example, I set my Chrome to use Monaco by default.
