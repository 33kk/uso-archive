### Description
When you need to read texts on the web, and you hate white backgrounds, then this theme is for you!

This style overrides all colors and makes web pages easy to read.

Features:

* Dark background
* Lighter and good looking input elements
* Black borders

How to use?

Simply turn on this global style when you need it for reading ugly designed web pages and turn it off when you dont need it.
