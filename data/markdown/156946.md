### Description
Quick and dirty dark & modular theme for fast.com

Fast.com is a simple speed checking site, quicker than speedtest.net, and much more pretty.
Now you can customize it with ease. This style has several options for quick and simple customization.

### Update notes
Feel free to request themes or changes in the comments.

If you think of another color for either the logo or background that might go well, let me know. I'll review it and implement if I like it.
