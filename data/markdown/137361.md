### Description
A stripped down, minimal dark theme for Whatsapp, completely rewritten and actively updating.

Fully configurable.

For inputs with (RGBA) label, you can choose a color <b><a href="http://bit.ly/2FoTbTG" target="_blank">here</a></b> then paste on them.

If you don't want the doodles, set <code>Doodles Opacity</code> to <code>0</code> and update. <code>White Doodles</code> allows inverted doodles color. I saw this idea on Vednoc theme and it was great.

Currently working on a UserCSS version, will upload when agenda allows.

Use it with Stylus for <a href="http://bit.ly/2QlhtDc" target="_blank">Chrome</a>, <a href="https://mzl.la/2ZnBhde" target="_blank">Firefox</a>, <a href="http://bit.ly/2s8vYlX" target="_blank">Opera</a> or <a href="http://bit.ly/39alNOh" target="_blank">Edge</a>.

Try <a href="https://bit.ly/39a3w3b">Transparent Telegram</a> theme.
