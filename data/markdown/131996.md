### Description

<b>⚠️ Warning:</b> This userstyle will not work with Stylish on Firefox 57 and later because of new Firefox WebExtensions API. See <a href="https://bugzilla.mozilla.org/show_bug.cgi?id=1310082" target="_blank">Bug 1310082</a> for more information.
