### Description
Makes the "XDA 2013 Beta - 1024" XDA theme look more like "XDA 2010". For those who prefer the old fonts and colors, but still want all the new features of the 2013 theme (e.g. DevDB).

Make sure you're using the "XDA 2013 Beta - 1024" theme on XDA to use this userstyle correctly.
