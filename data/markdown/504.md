### Description
This introduces a small image to identify the type is each hyper-link.
Small icons for these types of files:
- compression: 7z, zip, bz2, tar, tgz, gz, rar.
- movie: swf, mpg, wmv, mov, avi, ppt, pps, divx.
- application: exe, xpi, msi.
- image: png, jng, jpg, jp2, gif, svg, svgz, mng, bmp.
- text: css, txt, sok, xsb, vbs, js, class, java, torrent, rtf, doc, pdf, docx, pptx, xlsx, xul.
- interactive: xml, rss.
- sound: mp3, wma, wav, asx, asf, ogg.
- e-mail hyper-links
