### Description
This script removes the "Featured" and "Get Connected" boxes in Facebook.
Tested on Chromium.

Add the line "// @run-at document-start" to the "script.js" in your chrome user directory after the installation from userstyles.org.
// ==UserScript==
// @name Distracting boxes removal for Facebook
// @description This script removes the "Featured" and "Get Connected" boxes in Facebook
// @match http://*.facebook.com/*
// @match http://facebook.com/*
// @match http://ilike.com/*
// @match http://*.ilike.com/*
// @run-at document-start
// ==/UserScript==
