### Description
A retro header for the official Urban Terror site at http://www.urbanterror.info that mimicks the look of 2009 and earlier header graphics. The header only works correctly if the site is set to the default narrow view (“Proportional Width View”).

Unlike the Thingley and Crossing headers this one does not replicate an older header (although an Ambush header existed it showed a different scene).
