### Description
This is dark theme for github.com which aims to reduce eye strain when reading code.

Syntax highlighting implements One Dark Pro Vivid partially. Syntax highlight is extracted into its own section, so you can change easily with stylus.

Container width is changed to 96vw but you can disable or delete first section, if you like.
