### Description
<b>Description</b>

This project is a dark version of <a href="https://dns-shop.ru/">DNS-shop</a> website.

<b>How to contribute</b>  

Go to <a href="https://github.com/Kenya-West/DNS-dark">GitHub</a>, clone this project and start your changes. If you want to use styles from source code, then create new style in Stylish extension and put source code in opened window.

<b>How to use this style?</b>

You need <a href="https://github.com/openstyles/stylus#releases">Stylus</a> extension installed on your browser to use this style.
