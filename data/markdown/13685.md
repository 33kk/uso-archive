### Description
Study 36 different kanji each day via Daily Page for Student of the Japanese Language <http://www.manythings.org/japanese/daily/>. I've been using this site for a while with this style that I constructed myself. It cuts out everything from the page except for the kanji, which are moved from one small row at the top of the page to full-screen size.

毎日、Daily Page for Student of the Japanese Languageで漢字を36個勉強しようではありませんか。日本語を勉強している僕が一年前に作成したこのスタイルシートは、ページに上に載っている36個の漢字を拡大し、漢字以外の内容を見えないようにします。では、漢字の勉強、頑張りましょう！
