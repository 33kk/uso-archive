### Description
此样式为初学 CSS 时的黑历史，已经停止维护。

此样式为百度系列产品萌化，优化了部分显示效果，屏蔽了一些无用元素及推广。

※样式应用情况：
　√ 百度搜索
　√ 百度网盘

※注意事项：
<u><b>请将百度云的皮肤设置为除了前三个默认皮肤之外的皮肤！</b></u>
<u><b>请将百度云的皮肤设置为除了前三个默认皮肤之外的皮肤！</b></u>
<u><b>请将百度云的皮肤设置为除了前三个默认皮肤之外的皮肤！</b></u>

否则就会出现背景图显示大小不正常的情况！如果是使用艾拉的背景的话，推荐“月半弯”，色调比较配。

※关于自定义背景：请勿使用 userstlyes 自带的图片上传！这个上传实质上是将图片转成 base64 然后加入 css 代码中，如果你的图片比较大，就会造成页内 style 过长而导致网页出现异常。如果出现了这种情况，删除样式之后重新添加即可。建议使用其他在线图床，将图片上传之后获取链接，然后填入自定义背景设置项中。
