### Description
Gives you Rounded Menu Highlighting for ANY menu item highlighted using the mouse or keyboard.

* Thanks to user 5424 for introducing me into the "Wellrounded" types of styling with style 8366

This makes your menu's currently selected row have a rounded border
This style supports only Firefox3, giving it even more of a WOW effect!.

Applies to:
+ Currently Selected Row / Item on Menus
+ Context Menus
+ Tab Menus
+ Main Menu Menus
+ Countless other menus



Additional Information:
+ Uses new CSS3 Circular Radius code!
+ Uncomment some of the code to change the Border Color of these rounded menus

Browser Versions:
+ Works on Firefox 3
+ Also works on Firefox 2

Integrates with the following Styles:
+ Style 8414
+ Style 8562

Works well with:
+ All of the Well Rounded styles
+ The Styles Displayed above
