### Description
A dark version of the LWN.net website.

I looked through userstyles.org but I found no dark themes that were pleasant to the eyes. So I decided to make a dark theme using Chris Kempson's dark Atelier Savanna colorscheme.

This stylesheet still has some quirks but, in general, it should work fine.
