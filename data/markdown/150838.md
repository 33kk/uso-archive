### Description
This theme requires the <b>Tab Center Redux</b> add-on developed by <i>Edouard Oger</i><br><br>

<span style="color: #006bbc;">addons.mozilla.org/en-US/firefox/addon/tab-center-redux</span>

A dark theme with azure accents with auto-shrinking sidebar showing just the favicons, hidden scrollbar. Mozilla Firefox 57.0b11 (Developer Edition)<br><br>This theme assumes that you use Firefox 57+ with Dark theme enabled.<br>In the Tab Center Redux's settings, Enable the Dark Theme, and set Compact to 'Always'.

<h2>Installation:</h2>
https://github.com/JZersche/mozilla-zdark-tcr/tree/modern

Follow the instructions on the link above. 

Create a userChrome.css file in the your main Firefox profile folder. Type about:profiles in to the address bar of Firefox if you don't know where that folder is. The userChrome.css file goes into it's own folder named 'chrome' so create that as well.

Do not put userChrome.css inside the Add-On settings. It does not go there.
