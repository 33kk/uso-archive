### Description
Enable the night mode setting for the theme to work. This theme has much more contrast than the default night mode. Edited version of <a href="https://userstyles.org/styles/159073/hulu-night">Hulu Night</a> by learsi. I made the text brighter and more readable, fixed some bugs/unreadable text, and darkened a lot of areas that still had a white background. Some areas are not themed, such as the account settings page.
