### Description
Makes the threads on Aeria Games forums compact by doing the following:
<ul>
<li>Shows only as much of user info that fits for the post, hover it to show all.</li>
<li>Resizes and moves avatars to show more of them.</li>
<li>Removes signatures.</li>
<li>Limits text width to make it more readable.</li>
<li>Only shows post action when hovering over the post.</li>
<li>Removes some of the useless padding and margins.</li>
</ul>
