### Description
<b>The Australis minimalist and customizable style for Firefox with Chromium theme. For Firefox 20 or more, and requies <a href="https://addons.mozilla.org/firefox/addon/fxchrome/">FxChrome</a> add-on.</b> Classic Style Edition.

### Update notes
This is classic style, we are working on an improved interface in Second Edition.

Would you love to have a nice user interface, customizable and easy to use? If Chrome can do it, so can Firefox. With some practical steps you can make it happen, and complement to other styles. How can you pick a color? How does it work? This system uses practically a basic components (<abbr title="Hue, Saturation, Lightness">HSL</abbr> and <abbr title="Red, Green, Blue">RGB</abbr>) in tabs or toolbar. You can improve your own design, with high quality images, installing <a href="http://userstyles.org/styles/86501/australion-my-plus">MyPlus</a>.

Everything and more is in the comfortable design of Australion, free lifetime, and you can make a suggestion to your want.
