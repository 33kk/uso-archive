### Description
・タブの読み込みボタンをフォクすけにします。
・サイト指定のアイコンがない場合フォクすけにします。
・アイコンすべてフォクすけにする機能はコメントアウト(無効化)されてます。
　詳しい方で使いたい方は書き換えてみてください。
*Change loading icon on tabs to Foxkeh.
*Unless site-specific icons, change to Foxkeh.
*Some of the functions are disabled by comment-out,that changes all the icons into Foxkeh.
　Please edit if you want to use.
