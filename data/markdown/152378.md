* The main text is now in the center of the page.
* Two theme color options!
* The sidebar is now at the bottom of the page.
* Selectable Font-size.

### Update notes
HowTo - Choosing Theme Color and Font Size:
* Click on the "Advanced Style Settings"-Button when you are on the install page of the style.
* Choose a theme color via drop-down menu.
* Choose a font size via drop-down menu.
* Click on the install button of the style.
(If the style is already installed, click on the update button of the style)
