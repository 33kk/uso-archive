### Description
This makes the top bar with messages, the friend list, favourites, etc. fixed. 

I am aware that others exist like this, but this one simply has more to it. Others simply make the main overhead element fixed so many things do not work (i.e. friends menu doesn't scroll, badges appear over the bar, etc.).

I'm no longer maintaining these past a simple bug fix release (2014). Feel free to modify this and redistribute. I no longer use dA.
