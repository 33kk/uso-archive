One theme to rule them all ! 

An alternative theme for Muropaketti's MuroBBS bulletin board (also knowns as Muro). 

<b>NOTE: Also install the Web 2.0 Icon-package</b> : http://www.userstyles.org/styles/84069/

MuroBBS - Web 2.0 Light theme for Firefox & Chrome

* clean & mean ! 
* light ( in the both sense; luminosity & weight ! )
