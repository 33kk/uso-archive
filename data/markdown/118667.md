### Description
This theme allows you to change several colors of facebook, there's new gradients and options to change them with color pickers. The default theme is a simple clean dark theme with a solid color background, nevertheless it can be replaced by an image which will look very nice because the content of the page is contained in transparent boxes. 

To create your very own color scheme please just pick colors in the palettes, and maybe a background image to customize it even more.
