### Description
MBlue Nav-Style Menubar Vista
--------------------------------------------------------------------
Description:
 1. Transforms the Menubar.
 2. Makes the ToolbarButtons Nav-Style.
 3. Can use Separator/Spacer to customize the buttons (read below).
 4. Supports SMALL and BIG ICONS.
 5. Fixes Download-Helper icon.
 6. Skins FoxClocks & ForecastFox.
 7. Highly Customizable.
 8. Integrates with the other Nav-Style styles.
--------------------------------------------------------------------
Notes:
-Using other Styles/Extensions/Themes may change some margin/border settings, making the buttons not aligned. If so, try not using other styles that changes margins in the Menubar, and use the default firefox theme. 

-Works in the Menubar only.

-Some Ext. Icons may not work correctly.

-Some Extension Icons doesn't work well with separators, place them between other icons or spacers.

-Even if this style supports big icons, it works better with small ones, because the default big icons are too big to fit inside the buttons, so this style resizes them.

-If you want to use the Nav-Buttons (Back-Forward, Stop, Reload) inside the Menubar, you also need:
 1) Nav-Buttons Vista on XP (fix back/forward/dropmarker) [*]
 2) Combine Nav-Buttons Vista (fix & combine stop/reload) [*]
