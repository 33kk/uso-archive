### Description
This restyles the Google Hangouts interface to be dark with grey highlights similar to the DeepDark style. This is for the hangouts web page or the gmail hangouts sidebar. This is NOT for the chrome webextension (add-on).

The Google Bar - Dark Style theme by Devo (https://userstyles.org/styles/120351/google-bar-dark-style) optionally can be applied alongside in order to style the top Google bar interface.
