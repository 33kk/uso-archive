### Description
WordPress の管理画面のボタンのうち、集中執筆モードのボタンが文字化けを起こしている場合、この CSS を追加すると文字化けが修正されるかもしれません。

やっていることは該当する部分にフォント「dashicons」を指定しているだけです。他に文字化けしているボタン等あれば、その CSS クラスを追加すれば、その文字化けも直るかもしれません。
