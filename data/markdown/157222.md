### Description
Testrail custom CSS more suited to my preferences of usability.

Style includes:
* case title word wrap
* case title black font
* row colours alternate
* row colour highlighted - blue
* big checkboxes
* test run title fixed in view at bottom of page

Note: If you have admin rights on your Testrail instance you can add this UI script in the administration area under Administration > Customization

https://discuss.gurock.com/t/word-wrap-case-title/2639
