### Description
Adds "…1, …2, ... …9" for convenient control+number (alt+number on linux) switching between tabs.

Also decreases min-width (= more tabs visible while many) and removes max-width (= auto-wide tabs while few), increases font-size; set browser.tabs.animate = false in about:config.
