### Description
Control new underline html elements on websites.
For chromium 64+ and Firefox 70+.

Without <code>!important</code>.

<a href="https://developer.mozilla.org/en-US/docs/Web/CSS/text-decoration-skip-ink" title="https://developer.mozilla.org/en-US/docs/Web/CSS/text-decoration-skip-ink">https://developer.mozilla.org/en-US/docs/Web/CSS/text-decoration-skip-ink</a>

### Update notes

<pre><u>JjGgYyPpQq</u>, <u><b>JjGgYyPpQq</b></u>, <u><i>JjGgYyPpQq</i></u></pre>

<h3>Syntax:</h3>
<pre><code>/* Single keyword */
text-decoration-skip-ink: none;
text-decoration-skip-ink: auto;</code></pre>
