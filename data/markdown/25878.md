### Description
This a theme for the Ratchet & Clank wiki which makes the wiki look (in my opinion) much, much better. It styles userpages, pages, the main page, and even the new message alert. I have yet to style the toolbox at the bottom of the page.

UPDATE 3/4/2010: minor fixes concerning the page bar on userpages
UPDATE 3/5/2010: hover effect on userpage tab bar, rounding and -moz-box-shadow on sub-menus (and box shadow on widgets that are being dragged), header, new logo, background (taken from ratchet-galaxy.com), gradients on the main page, rounded avatar border, centered search box, rounded search suggestions box (still no styled toolbox)
UPDATE 3/6/2010: hover effects on page controls (edit, move, watch, etc.), support for webkit browsers (Google Chrome, Safari), box shadow fixes on widgets
UPDATE 3/18/2010: major fixes for new wikia stuff: save page/preview bar when editing styled
UPDATE 3/27/2010: page tabs hover effects, button styles, gallery hover effects, cool looking x's on the bookmarks widget, small movement of the sub-menus
