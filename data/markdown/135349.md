### Description
Стили для обновленного overclockers.ua
- все скруглено
- без градиентов
- большой заголовок
- хлебные крошки смещены вверх
- фон списка тем при наведении
- уменьшен отступ в списке тем
- цитаты и спойлеры - изменен цвет
- выделение "форума" в списке тем
- увеличины шрифты
- адекватный (маленький) размер картинок в постах (задолбали своими картинками на пол экрана!)
- и др.

для телефонов все остается неизменным, эти стили только для ПК.
