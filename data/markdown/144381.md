### Description
A flat, almost borderless theme for Firefox.<br>
Note: does not work on Firefox 57+<br>
<br>
In addition to this stylish theme, the browser in the screenshot also uses:<br>
- Firefox "Lollipop Road" theme (Win/Mac/Linux/BSD)<br>
https://addons.mozilla.org/en-US/firefox/addon/lollipop-road/<br>
- XFWM "Chrome window title bar" (Linux/BSD XFCE only)<br>
https://www.xfce-look.org/p/1165642/<br>
<br>
Optional: uncomment a line in the css code to also remove the light border above the navbar.
