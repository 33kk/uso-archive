### Description
This style does these enhancements:

■ It changes default monospace font to Segoe UI (intended for Windows), Droid Sans (intended for Linux) or sans-serif for Streamsquid.com web site.

■ It also adjust font size to make text a bit bigger.

■ It makes smaller boxes of music tag elements.
