### Description
你一定会喜欢的简约且好看的百度轻萌化！
在pan_cao大大的“Baidu Lite 百度轻”上进行了一定的轻萌化，<s>百度轻魔改版</s>。

作者博客：<a href="https://kira.cool" target="_blank"><b>kira.cool</b></a>

<b>实现功能：</b>
1. 修复原版“Baidu Lite 百度轻”的搜索框不能正常使用的问题
2. 新闻搜索可以正常使用
3. 卡片半透明化
4. 高分屏适配
5. 支持图片背景

<b>注意：</b>
1. 在未登录百度账号时，百度会额外引用一个未登录的 css 文件，在 v1.0.6a 版本中简单修复了这个 css 文件额外带来的部分问题。但仍有一些问题未处理，强烈建议您登录百度账号以避免出现不必要的问题。
2. 同样的，在使用百度主题后也有可能会出现不必要的问题，建议您在使用本主题时不要使用百度自带的皮肤功能。
