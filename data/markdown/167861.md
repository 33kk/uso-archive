### Description
Coffee 5.0

Since Twitter now is forcing the "New Layout" on pretty much everyone, I've decided to update 
this style to work with the new layout.
Not everything works yet, if you have problems or requests send me a dm at @twi_styles.

Features:
-- 3 color themes for you to choose from ( you can use the regular Twitter theme selector at More(in the left sidebar) - > Display or in Settings -> Display. )
-- hide promoted tweets.
-- wide top bar resembling legacy twitter style.
-- add breaks between tweets for easier timeline reading.


Old Coffee style was moved to https://userstyles.org/styles/175461/coffee-legacy.
