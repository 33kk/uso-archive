### Description
A redesign to make YouTube more flat and close to Material Design.

ATTENTION!
To achieve the desired interface, do the following steps:

1) Go to https: www.youtube.com/?gl=US
2) Open the developer tools (Ctrl + Shift + i)
3) Go to the ‘Resources’ tab and delete the VISITOR_INFO1_LIVE cookie for the YouTube domain
4) Go to the console and define the VISITOR_INFO1_LIVE cookie using the following command:

document.cookie="VISITOR_INFO1_LIVE=Qa1hUZu3gtk;path=/;domain=.youtube.com";


Based on: "Clean YouTube 2016 - by UNDERDEV" - https://userstyles.org/styles/121329/clean-youtube-2016-by-underdev

### Update notes
Important note:
Google no longer supporting the steps necessary for this theme in some regions. 
Therefore, the desired interface may not work correctly.

If you want to apply the Material Design on YouTube, try this other Style:
https://userstyles.org/styles/133852/youtube-material-design-2016
