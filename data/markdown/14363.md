### Description
If you use dual monitors, you'll know fullscreen flash can't work while you use another program on the other monitor.
This will emulate fullscreen, just make sure you are logged into tvcatchup, enable this style and fullscreen firefox (F11)

This only works in 1280x1024
If you use another resolution, change the height and width accordingly.
