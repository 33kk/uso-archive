### Description
This style restores the green (verifiedIdentity), blue (verifiedDomain) and grey identity-box colors from Firefox 13 and replaces page-proxy-favicons with different images.
Page-proxy-favicon images are part of Firefox default theme and can be found inside 'omni.ja' file, where this style sheet links to.

Icon appearance can be tested by visiting the pages linked on Mozillas identity-box overview: https://wiki.mozilla.org/AMO:Editors/EditorGuide/ThemeReviews#Identity_Boxes
