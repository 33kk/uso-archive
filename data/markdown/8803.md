### Description
Automatically hides Tab Scope buttons based on your current navigation of the page.   

Features:
+ If you cannot move back, the back button is removed.  
+ If you can't move forward, the foreward button is hidden.
+ If you can't stop navigation , the stop button is hidden.
+ If you are on a new tab or window, BOTH buttons are removed!

Gives you a larger area, and more screenspace for your browser...They only display when required!  
Works for both Firefox 3 and Firefox 2.

Unified Back/Forward/Stop buttons that only appear when you need them (nothing to go to).  Visually notifies you if you have a page to navigate to by enabling the button!

**Must have the Tab Scope Extension installed**

Integrates with Style 7577.
Integrates with Style 8887.
