### Description
This style replaces the menubar labels with icons. Cross-platform, using the <a href="http://tango.freedesktop.org/Tango_Icon_Library">Tango Icon Library</a>.

For a Linux-specific version that uses system native icons, see style 27922.

Currently this does not support the extra menus added by icons, video chats, etc.
