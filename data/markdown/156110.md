<b>PLEASE ADD THE CODE IN USERCHROME.CSS AND USE IT WITH THE DEFAULT FIREFOX THEME!</b>

For all the Firefox users who would like to properly enjoy my <a href="https://dpcdpc11.deviantart.com/art/Simplify-10-Dark-Purple-Windows-10-Theme-731839388" target="_blank">Simplify 10 Dark Purple - Windows 10 Theme</a>, I've created this matching style.

<b>INSTALLATION INSTRUCTIONS:</b>

- go to your profile folder. More info here: http://kb.mozillazine.org/Profile_folder
- then go to the "chrome" folder there, creating it if necessary
- in the chrome folder copy the userChrome.css file included here
- switch to the Default Firefox theme and then restart Firefox!

Enjoy!
