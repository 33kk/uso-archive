### Description
AMBOSS Lernkarten leichter ausdrucken. Nicht benötigte Elemente werden für den Druck ausgeblendet, die Darstellung beim Browsen wird nicht beeinflusst.
In dieser Version werden noch mehr zusätzliche neue Objekte für ein besseres Schriftbild ausgeblendet, sowie das Facharztwissen nicht gedruckt. Außerdem kann das Schlüssel-Wissen gelb hervorgehoben werden (Option Hintergrund-Drucken im Druckdialog von Firefox oder Chrome aktivieren)

Eine Version ohne das Facharztwissen findet ihr hier:
https://userstyles.org/styles/144158/amboss-easyprint-v3

Zur Verwendung: 
- Browser-Addon Stylish installieren
- den Style anwenden
- Vor dem Druck alle Reiter ausklappen (Leertaste) und dann drucken (zB als PDF).
