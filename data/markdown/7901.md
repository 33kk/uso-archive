### Description
Allows you to Compress the button size of the "Organize Bookmarks" toolbar buttons.  The text of each button will also be removed.
Increases screen size, allowing you to view and organize more bookmarks.
This style supports Only Firefox2.


Applies to:
- Organize Bookmarks Toolbar (Main Menu / Bookmarks / Organize Bookmarks)
- Compressed Buttons
- Text Removed on Buttons (toolbar buttons will just be buttons!)

Browser Versions:
- Works only on Firefox 2

Integrates with Style 7719
