### Description
INSTRUCTIONS!!!!

If you want to download just one theme:
1. Select your theme from the list and click Download.

If you want multiple themes or the theme is already installed:
1. Select your desired theme from the list, then click on the "Show Code" button below.
2. Open the Stylish menu and select Write new theme > Blank theme.
3. Copy and paste the code from the this site into the window.
4. Give the theme a name then click Save.

~~~~~~~~~~~~~~~~~~~~

Pokécheck is a site where users can upload, download, and clone Pokémon.

These themes skin the entire site in the colors of your choosing. Version 3.0 represents a merger of four earlier developmental forks into one friendly UserStyles.org theme selector.

Adding new themes to this site is very easy, so I am open to suggestions and comments.
