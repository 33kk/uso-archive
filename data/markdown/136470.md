A dark theme for ZeroNet, should works on all official zites and their clones. Some unofficial zites are also supported, like NullPaste, ZeroWiki, Kaffiene Search. Nevertheless, some other zites may have side effects - that some bright elements are still not changed.

Code theme is Vim Desert.

Also check this one for some misc. zites: https://userstyles.org/styles/138414/zeronet-dark-miscellaneous-website

Wish you enjoy it :-)
