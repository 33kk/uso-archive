### Description
Takes the Facebook Timeline, makes it wider so that the posts take up the full width of the screen.

Edit the code and use a % sign for a better result or edit it to fit your screen resolution.

Hides profile pictures and timeline about me area/photos/etc. Shows only shared posts and posted statuses.
