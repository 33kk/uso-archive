### Description
You need BetterDiscord to inject transparent theme and transparency patched Discord.

Rough instructions (You need Node installed):<br>
<code>npm install -g asar</code><br>
Extract <b>%appdata%\discord\(latest version)\modules\discord_desktop_core\core.asar</b> with <br>
<code>asar extract core.asar core</code><br>
<strong>core/app/mainScreen.js</strong><br>Set <b>transparent</b> -> <b>true</b> around line 290, remove <b>backgroundColor</b> line and save <br><br>
Remove/Backup&Remove <strong>core.asar</strong> and then:<br>
<code>asar pack core core.asar</code><br>
Relaunch discord.</br>
In case betterdiscord options are now gone, you need to reinstall BD (so it patches on top of transparency trick): <a href="https://github.com/Jiiks/BetterDiscordApp/releases">BD</a><br>
Set Custom CSS in betterdiscord with this. (without <i>@-moz-document</i> brackets)
