### Description
This will make the sidebar act like a popup menu. Bookmark/history sidebars will be dark transparent. Other sidebars will require additional styling. Hover at the far left side of the screen to show the sidebar.

Install notes: Deactivate sidebar - install style - reactivate sidebar.
Tested in Firefox 4 & Minefield 4.2a1pre on Windows 7
