### Description
A complete makeover for Reddit.

Version: 1.3.1

At this moment, avaliable for Chrome only.

<strong>SOME FEATURES:</strong>

<ul>
<li>RES-compatible</li>
<li>Includes an RES Nightmode and Dashboard</li>
<li>Fixed Header: a fully transformed header fixed as a column on the left</li>
<li>100% responsive</li>
<li>Better comment readability with a narrower column</li>
<li>Permalinked comments optimized for even greater readability</li>
<li>Warns about not being able to vote when not logged in</li>
<li>Link timespan "links from: today, this week, this month" no longer has to be accessed through dropdown</li>
</ul>
