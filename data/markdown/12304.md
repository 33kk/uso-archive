### Description
This style highlights the rows and menus in Gmail when you hover over them.

Colors are customized for the "Tree" Theme of GMail.

If you are using a different theme, you may want to check the following styles:
- "Classic" Theme: http://userstyles.org/styles/6897
- "Default" Theme: http://userstyles.org/styles/12303

I would like to credit this theme (http://userstyles.org/styles/678) for the inspiration.
