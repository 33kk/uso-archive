### Description
This style modifies the find bar in Firefox 4+ so it looks like a little pop-up in the bottom right corner of the window. The add-on bar auto-hides when not used and pops up with a fade out effect when you hover the bottom right area of your FF window. It fades out of screen when not used occupying no space on the window.
The find bar is styled to fit the same look and also packs a dynamic info label effect so it wastes no extra space when  the info is not needed.
Customized code so the background covers add-on buttons pinned in add-on also, not only status bar add-on icons.
