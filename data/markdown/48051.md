### Description
<strong>Hey 2019, Looking for updates?</strong> <a href="https://github.com/zvuc/firefox-sliding-bookmarks-bar">This userstyle now lives in github.</a>
<br>
Hides the bookmark bar by default, only appears when hovering over the tabs and the navigation toolbar. (slides down)
<br><br>
<strong>If you're using Firefox 57 and above, installing this style through Stylish will not work.</strong> Please paste the code into userChrome.css file in your profiles folder (<a href="https://www.userchrome.org/how-create-userchrome-css.html">See how</a>) to apply. (Make sure you set the correct pixel value for <code>--toolbar-offset-height</code> variable depending on your theme configuration.
