### Description
Makes the background color for viewing direct Imgur images (or webm!) a darker color so it is easier on the eyes. Or you can specify your own color for a personalized viewing experience.

### Update notes
<b>NEW!:</b>
<ul>

<li>Hard to see some transparent images? Hover over the image to make the background of it a lighter color to see better!</li>
<li>You can now choose your own custom color or simply use the default setting.</li>

</ul>
