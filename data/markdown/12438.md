### Description
This is a user style for the ISO50 Blog, http://blog.iso50.com/ 
The style simply removes the side bars, header graphics, and other distracting elements.  So all you are left with is the content, centered horizontally on the page.

Updated Version:
Added "@-moz-document domain" property to the CSS code for Firefox, so the style sheet will only affect the ISO50 blog.
