### Description
This style hides the Dashboard tips in Wordpress.com blogs, be sure to change the URL for your blog in :

@-moz-document url(http://yourblogurl.wordpress.com/wp-admin/) 

If you have more than one blog in Wordpress.com just add another url() :

@-moz-document url(http://yourblogurl.wordpress.com/wp-admin/), url(http://otherblog.wordpress.com/wp-admin/) 

* Jan 28th - Updated for new Wordpress.com Dashboard
