### Description
A modification of my old themes, gives a much cleaner layout and easier to read text, hides useless stuff like tripcodes and some other things. 

Warning:  If your resolution is different than 1280x800, then you will most likely need to change the offset for the post form by changing some code.

.postarea form {
 margin-left: 60%;
 position: fixed !important;
 z-index: 40 !important;
}

.postarea form:hover {
 margin-left: 60%;
 position: fixed !important;
 z-index: 40 !important;
}

Change "margin-left" to fit your display, then save it.
