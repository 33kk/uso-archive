### Description
Additional changes to "YouTube "Earth Hour 2012" Dark Theme Redux" by BRYN

### Update notes
Main function for this style is to widen Youtube's "wide" player to 1280px width.
It is meant to be used in addition to "YouTube "Earth Hour 2012" Dark Theme Redux" by BRYN
You can get the style from here: http://userstyles.org/styles/63482/youtube-earth-hour-2012-dark-theme-redux

You also may select default 720p fixed player size or use Youtube Center scripts, just select "No resize" option of the style. Also added 1024 pixels wide edition that fixes playlist positioning and video size at 1024x768 resolution.

Changes:
