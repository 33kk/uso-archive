### Description
Dark stackexchange chat with less empty spots, margins and roundy corners, with also stuff like hiding user list and room description (leaving stuff for stuff that matters, AKA stars) and hover-to-see images.

Based on https://github.com/fredley/Porkchat/blob/master/src/dark.css which is GPLv3 licensed.
