### Description
Fixes the Bug with the URLBar when you have the Personas Extension installed.
+ FIXES BUG 19397 ( https://www.mozdev.org/bugs/show_bug.cgi?id=19397 )
The Personas Addon cuts off half of the last character on the Location Bar.
This extension fixes the bug by increasing the URL Bar's Font Size.

Features:
+ Increases Font Size of the URLBar (customizable to your monitor resolution)

Integrates with the following Styles:
Personas Fix

Additional Information:
This style works only on the new Firefox 3 browser.
