### Description

### Update notes
<p>This stylesheet changes styles on launchpad to use monospace, in hope of making the page more straightforward in terms of WYSIWYG for CLI translaions and bug comments. It also enhances clarity by showing all spaces in translations, an important feature for CLI alignment. Actually from the screenshot, you can see the aftermath of not using `white-space: pre` by default â€” the old translator didn't see the spaces.</p>

<p>Shameless plug: If you deal with CJK characters and want better alignment, try <a href="https://be5invis.github.io/Iosevka/inziu.html">Inziu Iosevka</a>, a nicely-hinted blend of <a href="https://be5invis.github.io/Iosevka/">Iosevka</a> (a strictly half-width monospace typeface) and Source Han Sans.</p>
