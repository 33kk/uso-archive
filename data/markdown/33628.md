### Description
Портированный для баша стиль с  широко известного в определённых кругах форума. Кроме изменения внешнего вида "под фотон", также удаляет ненужные элементы (новости, контакты, и т.д.) частично убирает рекламу (скрыть ВСЁ не удалось, в некоторых местах остаются серые полоски на её месте). А вообще, для скрытия рекламы советую использовать AdBlock+.
* 8 октября: Исправлено ошибочное блокирование комиксов вместе с рекламой, куча мелких фиксов.

(Тёмная версия стиля находится здесь: style 33547)

Установка:
• Firefox — проверьте наличие установленного плагина Stylish (https://addons.mozilla.org/ru/firefox/addon/2108/), нажмите на кнопку справа - "Install with Stylish"
• Chrome — проверьте наличие установленного плагина Stylish (https://chrome.google.com/extensions/detail/fjnbnpbmkenffdnngjfgmeleoegfcffe), нажмите на кнопку справа - "Install with Stylish"
• Opera — Нажмите на кнопку "Install as plain CSS" сохраните открывшийся файл с расширением .css, далее зайдите на bash.org.ru, нажмите F12 выберите "Настройки для сайта" ("Edit site preferences"), перейдите на вкладку "Вид" ("Display"), в поле "Моя таблица стилей" ("My Stylesheet") укажите местоположение сохранённого css-файла. Нажмите "Ок".
• UserJS - во все поддерживаемые браузеры стиль можно установить как юзерскрипт, нажатием на кнопку "Install as user script". Для FF и Chrome требуется аддон Greasemonkey, установка userjs в Оперу подробно описана на офф. сайте
