### Description
<p><b>**this is no longer going to be updated**</b></p>
<p>This code makes the header have a gradient and has optional features such as: changing the new post icon colors on the dash, and the follow text color.</p>

<p>If you like this you should check out my <a href="https://userstyles.org/styles/149949/rainbow-tumblr-dashboard-theme">Tumblr Rainbow Theme</a>! It has a rainbow gradient header, that matches the rest of the theme completely. </p>

The theme used in main photo is <a href="https://userstyles.org/styles/106629/tumblr-nighttime-bakathemes">Nighttime</a>, with just a few tweaks.

### Update notes
<b>Instructions on how to get the same effect as on image #4</b>
<br>
<ol>
<li><p> Use this <a href="https://assets.tumblr.com/images/quick_compose_sprite.png?v=a9092a0967f882ab67dd7e721f555044">image </a>(or just use this <a href="https://www.dropbox.com/s/t7nhy78obgz7iq9/quick_compose_sprite.psd?dl=0">psd</a>) and change the colors</p></li>
<li> <p> Change the url()&nbsp;of this code to the url of the image you just changed:</p><blockquote><p>.tab-post-type .post-type-icon {</p><p> &nbsp; &nbsp;background-image: url(change this!);</p></blockquote><p>ex:&nbsp;</p><blockquote><p>.tab-post-type .post-type-icon {</p><p> &nbsp; &nbsp;background-image: url(http://i.imgur.com/PkEvqfj.png);</p> </li>
</ol>
