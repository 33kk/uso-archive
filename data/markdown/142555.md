### Description
<ul>
  <li>Hides lock icons for secure (HTTPS) pages</li>
  <li>Autohides info "i" icons for all pages except those that use permissions</li>
  <li>EV SSL site name and "Firefox" text is still displayed</li>
</ul>

<b>Please use <a href="https://addons.mozilla.org/en-US/firefox/addon/snap-http-padlocks/">this extension</a> to get broken padlocks for all HTTP sites and therefore easily differ between insecure and secure connections. 

If you can, also vote for <a href="https://bugzilla.mozilla.org/show_bug.cgi?id=1310447">this config preference</a>!</b>
