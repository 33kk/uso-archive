### Description
this style applies the site <a href="http://php.net/">php.net</a> correctly if you are using the new beta design.
and there only if the browser inner-width is smaller then 976 pixels.
in this case it reduces some elements to get a better view on the current article.
in detail the header, menu, uvTab ( feedback & ideas ) & footer will be shrinked to a minimum. if you hover any of these shrinked elements they will appear.
