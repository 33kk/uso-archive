### Description
Removes search bar, user's name, invisible warning, invite toggle bar, and whitespace from Gmail's chat bar.  If your chat window is toggled off, you will need to disable this userstyle in order to access the toggle button.  This has not been tested with the new Hangouts app, now available in Gmail.
