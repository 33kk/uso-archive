### Description
HTML 5+ hyperlinks (&lt;a&gt;) can include a ping attribute with URLs that the browser calls when the user clicks on the link. This stylesheet highlights them.
<p/>
See <a href="https://html.spec.whatwg.org/#ping">The HTML5 living standard</a> for details on the ping attribute.
