### Description
This is a dark theme for the admin area of your own installations of WordPress (tested with version 4.7.5).
Currently, the wordpress.com admin area is not supported, but it is on the way.
There are also code sections to add the dark theme to the following plugin/theme admin areas: Akismet (version 3.3.2), Jetpack (version 4.9), Contact Form 7 (version 4.7), and Elegant Themes - Divi (version 3.0.46).
